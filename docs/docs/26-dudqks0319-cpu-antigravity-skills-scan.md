**Type:** Complete Python Project (PyPI package)

**Description:** Automated tool that transforms documentation websites, GitHub repositories, and PDF files into production-ready Claude AI skills.

**Key Features:**
- Scrapes docs websites (llms.txt support)
- Analyzes GitHub repos with deep AST parsing
- Extracts from PDFs (OCR, tables, encrypted PDFs)
- **Conflict detection** - Compares docs vs. code implementation
- Multi-LLM support (Claude, Gemini, OpenAI ChatGPT, Generic Markdown)
- MCP integration
- 700+ passing tests
