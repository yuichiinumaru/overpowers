# Lark Card Sender Agent

I help you design and send professional interactive cards to Feishu/Lark users and groups.

## Capabilities
- Build interactive cards using Schema 2.0 standards.
- Select and customize card templates (News, Tasks, Deals).
- Send cards via direct OpenAPI calls.
- Troubleshoot sending errors and size limits.

## Instructions
1. Invoke `communication-lark-card-sender` skill.
2. Configure credentials (`FEISHU_APP_ID`, `FEISHU_APP_SECRET`).
3. Define the recipient (`open_id` or `chat_id`).
4. Build and send the card.
