# Agents Filename Convention Report

## New Convention
The new filename convention is `category--sub_category_or_name.md`.
This allows for easy grouping while keeping all files flat, preventing subdirectory clutter and making alphabetical sorting inherently grouped by domain.

Categories include:
- `team-*`: Specific agent groupings/frameworks (e.g., team-claude, team-mcp)
- `role-`: Orchestration, Planning, and Leadership (e.g., manager, architect)
- `test-`: QA and Testing
- `sec-`: Security and Audit
- `db-`: Databases
- `infra-`: DevOps and Infrastructure
- `ai-`: Data Science, Machine Learning, and AI
- `ops-`: Monitoring and Operations
- `msg-`: Messaging queues
- `front-`: Frontend
- `back-`: Backend
- `mobile-`: Mobile
- `lang-`: Programming languages
- `mkt-`: Marketing, Social Media, Growth
- `design-`: UI/UX, Visuals
- `game-`: Game Development
- `research-`: Academic and Content Research
- `doc-`: Documentation writing
- `finance-`: Payments and Fintech
- `cms-`: CMS platforms
- `tooling-`: Build tools and bundlers
- `vcs-`: Version control systems
- `review-`: Code review and linting
- `misc-`: Miscellaneous agents
