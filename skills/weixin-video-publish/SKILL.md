# 腾讯视频号发布

使用浏览器自动化技术，通过微信视频号助手发布视频作品。

## 功能特性

- 📹 视频上传（支持MP4格式，最大2GB）
- 📝 短标题（最多50字，建议10-15字，**最少6字**）
- 📖 视频描述（最多1000字）
- 🏷️ 话题标签（通过#话题格式添加）
- 📍 地理位置（支持添加位置）
- ✅ 原创声明（必须勾选）
- 👀 可见范围（朋友可见/公开）

## 快速开始

### 前置条件

1. **浏览器环境**：OpenClaw 浏览器已启动
2. **账号登录**：微信扫码登录视频号助手
3. **视频文件**：视频文件已准备好

### 发布命令

```
帮我发布视频到视频号
视频：/path/to/video.mp4
标题：视频标题
描述：视频描述内容
话题：#人工智能#科技
```

### 视频文件要求

- 格式：MP4
- 大小：最大2GB
- 时长：建议1分钟-10分钟
- 分辨率：建议1080P及以上

## 详细发布流程

### 步骤1：打开发布页面

浏览器访问：
```
https://channels.weixin.qq.com/platform/post/create
```

或从视频号助手首页点击「内容管理」→「视频」→「发表视频」

### 步骤2：扫码登录

1. 首次访问需要微信扫码登录
2. 使用微信扫描页面上的二维码
3. 在手机上确认登录

### 步骤3：上传视频

1. 点击上传区域或「+」按钮
2. 选择本地视频文件
3. 等待视频上传完成

### 步骤4：填写短标题

- 点击短标题输入框
- 输入视频标题（**最少6字，最多50字**）
- **注意**：少于6字会发布失败！
- 示例：凡人修仙传故事 = 7字 ✅

### 步骤5：填写视频描述

- 点击描述输入区域
- 输入视频描述内容
- 可以在描述中添加#话题

### 步骤6：添加话题

在描述中使用 #话题 格式：
```
#人工智能 #AI #视频生成 #原创
```

### 步骤7：设置地理位置（可选）

1. 点击「添加位置」
2. 搜索位置（如「苏州中心」）
3. 选择具体位置

### 步骤8：声明原创

1. 勾选「声明原创」
2. 阅读并同意相关协议
3. 点击确认

### 步骤9：选择可见范围

- 朋友可见：仅微信好友可见
- 公开：所有人可见（推荐）

### 步骤10：发布

1. 确认视频信息无误
2. 点击「发表」按钮
3. 等待发布成功

### 步骤11：验证发布

1. 进入「内容管理」→「视频」
2. 确认视频存在且状态正常

## 平台规则

### 标题字数限制

| 限制类型 | 字数 |
|----------|------|
| 最少 | **6字**（必须） |
| 最多 | **50字** |
| 建议 | 10-15字 |

**重要**：标题少于6字会发布失败！

### 描述字数限制

| 限制类型 | 字数 |
|----------|------|
| 最多 | **1000字** |
| 建议 | 50-200字 |

### 视频要求

| 项目 | 要求 |
|------|------|
| 格式 | MP4 |
| 大小 | 2GB以内 |
| 时长 | 无明确限制 |
| 分辨率 | 建议1080P+ |

### 推荐设置

- 标题字数：7-15字（必须大于6字）
- 描述字数：50-200字
- 地理位置：添加有助于曝光
- 可见范围：公开

## 常见问题

### Q1: 标题提示少于6字？
**A**: 视频号标题必须至少6个字，增加标题字数后再试

### Q2: 上传失败？
**A**: 检查视频格式是否为MP4，或尝试压缩视频

### Q3: 微信扫码失败？
**A**: 确保微信已绑定手机号，在手机上确认授权

### Q4: 发布后找不到视频？
**A**: 进入「内容管理」→「视频」查看发布状态

### Q5: 原创声明失败？
**A**: 确保视频为自己原创，勾选声明后按确认

## 注意事项

1. **标题必填6字**：视频号标题必须至少6个字，少一字都不行
2. **必须原创声明**：不声明原创无法发布
3. **微信登录**：需要微信扫码登录
4. **视频格式**：仅支持MP4格式
5. **发布后要确认**：务必在内容管理中确认发布成功
