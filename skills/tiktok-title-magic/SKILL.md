# 抖音标题魔法 - TikTok 爆款标题生成器

## 概述
基于抖音/TikTok 算法和爆款内容数据，生成高点击率视频标题。内置流量密码词库和热门话题标签。

## 定价
- **按次收费：** ¥5/次
- **包月订阅：** ¥99/月（无限次）

## 功能特性
- 🎯 AI 分析热门视频标题模式
- 🔥 内置 1000+ 流量密码关键词
- 📊 标题吸引力评分（0-100）
- 🏷️ 自动生成热门标签组合
- 📈 A/B 测试建议（3-5 个变体）

## 使用方式
```bash
# 基础使用
/tiktok-title "视频内容描述"

# 指定风格
/tiktok-title "产品测评" --style 悬念
/tiktok-title "教程分享" --style 干货
/tiktok-title "生活 Vlog" --style 共鸣

# 批量生成
/tiktok-title "美妆教程" --count 5
```
