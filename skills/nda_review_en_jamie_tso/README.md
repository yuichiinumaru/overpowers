# NDA Review Skill

A Claude skill for analyzing and reviewing Non-Disclosure Agreements (NDAs).

## Overview

This skill helps review NDA documents by analyzing key clauses, party obligations, remedies, liability provisions, and standard exceptions.

## Contents

- `SKILL.md` - Main skill definition and instructions
- `LICENSE.txt` - License information
- `references/` - Reference materials for NDA analysis
  - `KEY_CLAUSES.md` - Common NDA clauses and their implications
  - `PARTY_OBLIGATIONS.md` - Analysis of party obligations
  - `REMEDIES_LIABILITY.md` - Remedies and liability provisions
  - `STANDARD_EXCEPTIONS.md` - Standard exceptions in NDAs
  - `DURATION_SCOPE.md` - Duration and scope considerations

## Usage

This skill can be used with Claude to analyze NDA documents and provide insights on:
- Key terms and clauses
- Party obligations and responsibilities
- Liability and remedy provisions
- Standard exceptions
- Duration and scope of the agreement

**Remember:** All outputs from this skill must be reviewed by a qualified legal professional before being used for any legal purposes.
