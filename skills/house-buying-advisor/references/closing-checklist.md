# 签约与贷款方法论

## 签约前核验清单

### 产权核验（最重要！）
- [ ] **房产证原件**：核对面积、地址、产权人信息
- [ ] **产权人身份证**：与房产证一致
- [ ] **婚姻状况**：已婚需配偶书面同意（不是口头！）
- [ ] **共有产权**：如有其他共有人，全部到场或出具公证委托
- [ ] **抵押查询**：不动产登记中心查有无抵押（必查！）
- [ ] **查封查询**：法院有无财产保全/查封
- [ ] **欠费查询**：物业费、水电燃气有无拖欠

### 房屋状态
- [ ] **户口**：是否有他人户口未迁出（学区房尤其重要！）
- [ ] **租赁**：在租中？租客是否知情？（"买卖不破租赁"原则）
- [ ] **违建**：有无未报批的改建/加建
- [ ] **维修基金**：余额多少？是否够用？

## 合同关键条款

### 定金
| 要点 | 说明 |
|------|------|
| 金额 | 通常总价 1%-5%（法定上限 20%） |
| 形式 | 银行转账（保留凭证！），不给现金 |
| 法律效力 | "定金"：买方违约不退，卖方违约双倍返还 |
| 注意 | "订金"≠"定金"——"订金"是预付款性质，可退 |

### 付款时间节点
```
定金 → 首付款 → 贷款审批 → 过户 → 尾款放款 → 交房
```
每个节点约定明确日期 + 违约责任。关键是留足缓冲：
- 贷款审批可能拖延 → 约定的时间要宽裕
- 过户/交房日期要明确 → 避免"无限拖延"

### 违约条款（必须双向对等）
| 情形 | 处理 |
|------|------|
| 买方不买 | 定金不退 + 违约金（总价 X%） |
| 卖方不卖 | 双倍返还定金 + 违约金 |
| 逾期付款 | 每天万分之 X |
| 逾期交房 | 每天万分之 X |
| **贷款批不下来** | **提前约定！**解除合同退定金？还是补差价全款？ |

### 税费承担
必须写入合同，口头约定不算数：
- 卖方通常承担：个人所得税、增值税（如有）
- 买方通常承担：契税
- 中介费：各付各的 or 协商一方承担
- **"阴阳合同"绝不做**——风险全在买方

### 物业交割清单
- [ ] 水表读数 + 结清
- [ ] 电表读数 + 结清
- [ ] 燃气表读数 + 结清
- [ ] 暖气费结清
- [ ] 物业费结清到交房月
- [ ] 有线电视/网络过户或注销
- [ ] 钥匙/门禁卡/车位卡交接
- [ ] 家电家具清单（合同约定"包含"的逐项确认）

## 贷款决策框架

### 贷款方式选择
| 方式 | 利率 | 额度 | 适用 |
|------|------|------|------|
| 公积金 | 最低 | 有上限（各地不同） | 首选 |
| 商业贷款 | 中（LPR±基点） | 按首付比例倒推 | 公积金不够时 |
| 组合贷 | 公积金+商贷各一笔 | 公积金顶额+商贷补差 | 最常用 |

### 还款方式选择
| 方式 | 特点 | 适用 |
|------|------|------|
| 等额本息 | 月供固定，前期利息占比高 | 收入稳定、追求确定性 |
| 等额本金 | 月供递减，前期还款压力大 | 高收入、想少付总利息 |

### 贷款策略原则
1. **首付越少越好**：现金在手比提前锁在房子里灵活（通胀会稀释债务）
2. **贷款年限越长越好**：月供压力小，提前还款的权利在你手里
3. **提前还款条款**：签约前必须问清违约金（有些银行前 X 年提前还有罚金）
4. **利率类型**：固定利率 vs 浮动利率（LPR 调整会影响月供），各有利弊
5. **银行选择**：不一定选中介推荐的，自己比较利率和放款速度
6. **月供安全线**：建议不超过家庭月收入的 40%

### 月供速算
每 100 万贷款 30 年：
- 利率 3.0% → 月供约 4,216 元
- 利率 3.5% → 月供约 4,490 元
- 利率 4.0% → 月供约 4,774 元
- 利率 4.5% → 月供约 5,067 元

## 过户流程（一般步骤）
1. **网签**：住建委系统登记买卖合同
2. **缴税**：买卖双方各自到税务局缴纳
3. **过户**：不动产登记中心办理产权转移
4. **领证**：通常 5-15 个工作日出新房产证
5. **放款**：银行见新房产证后放款给卖方
6. **交房**：物业交割、钥匙交接

全流程通常 1-3 个月，主要取决于银行放款速度。

## 特殊场景

### 买来打通（相邻户型合并）
- **承重墙鉴定**：请有资质的机构 → 能不能开门洞
- **住建委审批**：不能私自砸墙！需要报批
- **加固施工**：钢结构加固，费用视开洞大小而定
- **物业同意**：需要物业审批 + 施工期间监管
- **产权不合并**：两套各自独立产权
- **决策顺序**：先确认技术可行性 → 再决定买不买（别买完发现打不通）

### 全款 vs 贷款
| 维度 | 全款 | 贷款 |
|------|------|------|
| 谈判优势 | 大（卖方怕贷款不批） | 一般 |
| 交易速度 | 快 | 慢 1-2 月 |
| 资金灵活性 | 差 | 好 |
| 适用 | 不差钱 + 想快速拿下 | 大多数情况 |

### 经纪人选择原则
- 同一套房可以有多个经纪人 → 选最配合的
- **从业经验 > 专业证书 > 门店大小**
- 好的经纪人特征：回复快、不背话术、会主动告诉你房子的缺点
- 换经纪人是正常的商业行为，不需要内疚
