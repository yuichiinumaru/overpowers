# 砍价弹药生成方法论

## 弹药生成框架

### 第一步：扫描房源硬伤
对目标房源逐项检查，找出不可改变的客观缺陷：

**房龄类**：
- 不能用公积金（房龄超限）→ "纯商贷月供更高，愿意接受的买家少"
- 贷款年限短（房龄+贷款年限≤N 年）→ "月供压力大，受众窄"
- 无电梯（老人不便、搬家成本高）→ "老楼没电梯是硬伤，溢价要打折"
- 管线老化（需全部重排）→ "装修预算要额外加 X 万"
- 隔音差 / 保温差 / 能耗高

**户型类**：
- 面积小/无法改造 → "改不了两室，家庭客户不会考虑"
- 北向/西向 → "采光不好，晾晒不便"
- 暗厨暗卫 → "通风差，长期居住体验不好"
- 无阳台 → "晾晒只能在室内"

**位置类**：
- 靠马路/铁路 → "噪音大，窗户常年关着"
- 低楼层被遮挡 → "采光受限，白天也要开灯"
- 顶楼 → "漏水隐患 + 夏天闷热"
- 紧邻垃圾站/变电站 → "生活品质硬伤"

### 第二步：构建市场对标
从 Phase 2 数据中提取对标弹药：

**同楼竞品对标**（杀伤力最强的弹药类型）：
- 方法：找到同栋楼或同小区的同户型、不同楼层在售房源
- 公式："X 层才挂 XX 万，没电梯的楼 X 层是黄金楼层，你这层凭什么更贵？"
- 为什么有效：同楼同户型是最精准的可比物，卖方很难反驳

**挂牌时长对标**：
- 挂了 3 个月+ → "这个价位市场不认可，挂了 X 个月没卖出去"
- 降过价 → "趋势很明显，降了 X 次还没卖"
- 核心逻辑：时间站在买方这边，每多挂一天卖方的心理防线就弱一分

**历史出价对标**：
- 有人出价没成交 → "上次有人出 XX 万都没成，市场更冷了"
- 核心逻辑：建立"你的报价不离谱"的事实依据

**大盘趋势对标**：
- 近 3-6 个月同区域成交均价走势
- 全市成交量变化（量跌价跌）
- 核心逻辑：个体房源不能脱离市场大环境定价

### 第三步：挖掘政策弹药
从 Phase 1 数据中提取：
- 贷款限制 → 受众群缩小 → 定价要匹配
- 学区政策变动（多校划片）→ 学区溢价缩水
- 税费承担 → 可作为谈判变量（"税费由谁出"可以换价格让步）

### 第四步：利用心理弹药
从看房和沟通中捕捉的软信息：
- **空置成本**："房子空着每个月就在亏钱（物业费+贷款利息+机会成本）"
- **共有产权分歧**：多人共有时，总有人更想卖 → 这是突破口
- **卖方紧迫度信号**：急用钱/换房/移民 → 底线更低
- **季节性窗口**：年底（冲业绩）、春节后（市场冷清）→ 砍价空间大

## 弹药使用原则

### 1. 结构化输出
每条弹药包含三要素：
```
[弹药类型] + [具体事实] + [数据来源]
```
示例：
- [市场对标] 同户型 X 层挂 XX 万，比你低 X 万 [来源：贝壳 XX 日数据]
- [硬伤] 房龄 XX 年，不能公积金，商贷只能贷 X 年 [来源：当地公积金中心政策]

### 2. 分批投放
- 第一轮谈判：只用 2-3 条最硬的（同楼竞品 + 最大硬伤）
- 对方不松口 → 第二轮补充市场和政策弹药
- 再不松口 → 第三轮上心理弹药
- 让对方感觉"你这边信息太多了，跟你耗不起"

### 3. 重复核心论点
谈判中不怕重复。核心弹药（通常是同楼竞品对标）可以在不同语境下反复强调。重复 = 强化锚点。

### 4. 弹药也有保质期
市场数据会过时，新房源会上架，政策会变。如果谈判拉长到数周，弹药库需要更新。

## 事后复盘方法论

每次交易完成后，回顾：
1. 哪些弹药在谈判中实际被使用了？
2. 哪些弹药产生了可观察的效果？（对方态度变化、降价幅度）
3. 有没有当时没准备但事后发现应该准备的弹药？
4. 策略层面有没有认知偏差？（比如"该不该藏底牌"这类框架性判断）

复盘的目的不是追悔，是为下次积累——**Skill 的价值在于每次使用都让它变得更锋利**。
