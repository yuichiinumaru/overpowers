# Zhouyi · Bazi 典籍工作手册

## 1. Skill 概览
- **用途**：当用户咨询八字、周易、奇门遁甲、传统命理经典或想追问“原典怎么说”时，快速调取整理好的提要并输出结构化答复。
- **范围**：不含即时排盘功能（需要另行说明），专注于阐释原理、典籍脉络、用神思维、经典与当代语境的桥接。

## 2. 触发条件（When to Use）
- 提问直接点名：八字、子平、四柱、奇门、滴天髓、渊海子平等。
- 需要辨析“理法 vs. 格局”、“调候 vs. 扶抑”、“气机流通”等概念。
- 用户要求引用、比较某部命理书，或需要推荐学习顺序。
- 需要从传统观点中提炼现代表达（职业规划、健康、关系等）。

## 3. 工作流程
1. **确认问题类型**：
   - 典籍溯源 / 书目推荐
   - 理论讲解（用神、格局、调候、病药）
   - 方法论/实践礼仪（命师素养、答疑边界）
2. **定位参考**：加载 `references/classic_texts.md`，按书名找到关键信息（时代、要义、场景、注意点）。
3. **组织答案**：
   - 先给出简明结论或结构化摘要
   - 根据需要展开：背景 → 核心观点 → 应用/示例 → 注意事项
   - 在段末或最后列出引用（书名 + 来源链接）。
4. **翻译成现代语境**：
   - 用“发展、资源、行为策略”等词取代直接的“富贵贫贱”断语
   - 提醒用户命理属于思路参考，不可替代医学/法律等专业建议。

## 4. Reference Map
- `references/classic_texts.md`：四大经典 + 补充典籍表，含定位、适用场景、来源。优先加载该文件。
- `references/qimen_texts.md`：奇门遁甲代表典籍速查，用于回答奇门歌诀、排局、案例时引用。
- 后续如新增拆解（调候笔记、奇门专章等），需在本节补充索引与使用指引。

## 5. 答复指引
- **结构**：
  1. 直接回应问题要点
  2. 若涉及不同学派，列出差异点与典籍出处
  3. 给出操作或思考建议（如如何练习、如何结合现代工具）
- **语气**：专业、克制，避免神乎其神；强调“参考价值”而非“必然命运”。
- **引用**：至少列出书名 + 链接或“Baidu Baike — 书名”。多书并用时按段或列表标注。
- **边界提醒**：遇到医学、财务、法律风险，说明命理无法替代专业服务。

## 6. 典型用例
- “帮我总结三命通会里对官杀的看法” → 引用三命通会条目，结合子平真诠/滴天髓对比。
- “如何理解滴天髓讲的寒暖燥湿” → 先述理气框架，再提穷通宝鉴的调候实际操作。
- “想系统学八字，先看哪几本？” → 引导四大经典的学习顺序 + 现代补充（命理约言、命理探源）。
- “命师应该具备什么素质？” → 调用《命理探源》的十条职业守则。

## 7. 限制与后续扩展
- 当前仅包含经典速查。若需要更细的调候表、奇门局式等，应增补 references/ 子文件后更新本节。
- 没有脚本/排盘工具，若用户需要推盘，请说明现状并视需求调用其他技能或外部服务。
