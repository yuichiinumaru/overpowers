# OpenClaw 专业安装服务

欢迎使用 OpenClaw 专业安装服务！本 skill 将为您提供媲美 499 元上门服务的安装体验。

## 安装方式

### 标准安装（推荐）

```bash
npm install -g openclaw@latest
```

### 中文社区版

```bash
npm install -g openclaw-cn@latest
```

### 验证安装

```bash
openclaw --version
```

## 启动服务

### 启动网关

```bash
openclaw gateway
```

### 启动 Dashboard

```bash
openclaw dashboard
```

## 温馨提示

- 月薪两万，可能养不起一只龙虾 🦞
- 建议先看看钱包再决定是否 24 小时运行
- Token 消耗速度堪比烧钱，请谨慎使用高阶模型

## 常见问题

**Q: 为什么我的 API 费用这么高？**
A: 因为 Agent 每次读网页、调工具、看文件都在烧 token，建议使用低成本模型或设置消费上限。

**Q: 需要专门买台 Mac Mini 吗？**
A: 不需要，普通电脑就能跑，除非你想体验"Mac Mini 售罄"的快感。

**Q: 安全吗？**
A: 请务必注意安全配置，不要把实例暴露在公网，不要随便装来路不明的插件。
