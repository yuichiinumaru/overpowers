# 八字排盘 🔮

根据出生时间排出八字（年柱、月柱、日柱、时柱）及大运信息。

## 使用方式

### 基本命令

```
排出八字 2020-01-01 12:00 男
排出八字 1990-06-15 08:30 女
```

### 参数说明

| 参数 | 说明 |
|------|------|
| 出生日期 | 格式：YYYY-MM-DD |
| 出生时间 | 格式：HH:mm（24小时制） |
| 性别 | 男/女 |

## 输出格式

返回结果示例：

```
出生：2020-01-01 12:00 男
乾造：庚子 丙戌 壬午 丙午
大运：庚子 辛丑 壬寅 癸卯 甲辰 乙巳 丙午 丁未 戊申 己酉

https://yoebao.com/bazi/detail.html?sex=0&timestamp=1577841600000
```

或

```
出生：1990-06-15 08:30 女
坤造：庚午 壬午 辛亥 癸巳
大运：辛巳 庚辰 己卯 戊寅 丁丑 丙子 乙亥 甲戌 癸酉 壬申

https://yoebao.com/bazi/detail.html?sex=1&timestamp=645409800000
```

## 输出说明

| 行 | 内容 |
|---|------|
| 第1行 | 出生信息（日期、时间、性别） |
| 第2行 | 八字四柱（年柱 月柱 日柱 时柱） |
| 第3行 | 大运（10步大运） |
| 第5行 | 详细排盘链接 |

## 术语说明

- **乾造**：男性八字
- **坤造**：女性八字
- **大运**：每步大运约管 10 年，共 10 步

## 注意事项

- 需要提供完整的年月日时分
- 性别影响输出（乾造=男性，坤造=女性）
- 八字按年、月、日、时四柱排列
- 大运从天干地支序数计算得出
- 返回链接可直接查看详细排盘
- 时间戳为 JavaScript 毫秒级格式

## 常见问题

**Q: 如何计算八字？**
A: 八字基于中国传统的干支历法，年柱、月柱、日柱、时柱各取两字，共八个字。

**Q: 子时、丑时如何处理？**
A: API 会根据实际时间自动计算对应的时辰。

**Q: 大运如何计算？**
A: 大运从天干地支序数计算，天干 sn%10，地支 sn%12。
