## 意识丧失与 CPR

### 有人晕倒 / 意识丧失

```text
1️⃣ 确认环境安全
2️⃣ 拍肩呼唤，判断是否有反应
3️⃣ 无反应时立即呼叫 120 / 当地急救电话
4️⃣ 检查呼吸约 10 秒
5️⃣ 无呼吸或异常呼吸，立即开始 CPR

⛔ 不要做：掐人中、灌水、随意搬动
```

### 心肺复苏（成人）

```text
1️⃣ 呼叫急救并打开免提
2️⃣ 患者仰卧硬地面
3️⃣ 胸外按压：胸骨中下段，深度 5-6cm，频率 100-120 次/分钟
4️⃣ 会人工呼吸就 30:2，不会就持续按压
5️⃣ 持续到急救人员到达或患者恢复
```

## 烫伤与噎住

### 烫伤 / 烧伤

```text
1️⃣ 立即用流动冷水冲 15-20 分钟
2️⃣ 小心脱去不粘连衣物
3️⃣ 用干净纱布或保鲜膜覆盖
4️⃣ 面积大、位置特殊或疼痛明显时尽快就医

⛔ 不要做：牙膏、酱油、酒精、挑破水泡、直接冰敷
```

### 噎住 / 气道异物

```text
1️⃣ 能咳嗽说话：鼓励继续咳嗽
2️⃣ 不能咳嗽 / 发声 / 发绀：立即海姆立克
3️⃣ 异物未出且失去意识：放平，呼叫急救，开始 CPR
```

婴儿 `<1岁`：背部拍击 5 次与胸部按压 5 次交替。

## 出血与中暑

### 出血

```text
1️⃣ 直接压迫止血
2️⃣ 抬高伤肢
3️⃣ 持续按压 10-15 分钟，不反复掀开看
4️⃣ 仍大量出血时立即急救
```

### 中暑

```text
1️⃣ 转移到阴凉通风处
2️⃣ 解开衣物并物理降温
3️⃣ 清醒者少量多次补液
4️⃣ 昏迷、抽搐或体温 >40°C：立即急救
```

## 骨折、蛇咬伤、触电

### 骨折

```text
1️⃣ 保持原位，不乱动
2️⃣ 固定上下两个关节
3️⃣ 冷敷消肿
4️⃣ 开放骨折覆盖后送医
```

### 蛇咬伤

```text
1️⃣ 远离蛇，不追打
2️⃣ 减少活动，保持冷静
3️⃣ 清水冲洗伤口
4️⃣ 伤肢低于心脏
5️⃣ 尽快送医
```

⛔ 不要做：吸毒、切开、止血带、草药偏方。

### 触电

```text
1️⃣ 先断电或用绝缘物使其脱离电源
2️⃣ 再检查意识和呼吸
3️⃣ 无呼吸立即 CPR 并急救
4️⃣ 电烧伤即使外观轻微也要就医
```

## 过敏反应与抽搐

### 严重过敏反应

```text
1️⃣ 立即呼叫急救
2️⃣ 有肾上腺素笔就尽快使用
3️⃣ 平躺抬腿；呼吸困难可半坐位
4️⃣ 无呼吸心跳时开始 CPR
```

### 抽搐 / 癫痫发作

```text
1️⃣ 清除周围危险物
2️⃣ 不按住患者四肢
3️⃣ 头下垫软物
4️⃣ 记录发作时间
5️⃣ 发作后侧卧位
6️⃣ 首次发作、超过 5 分钟、连续发作：立即急救
```

⛔ 不要做：往嘴里塞东西、掐人中、强按四肢。
