# Response and Search

## 目录

- 输出要求
- 未覆盖场景的处理
- 反模式

## 输出要求

急救回复必须：

- 先给最紧急动作
- 用编号步骤
- 附“不要做”清单
- 末尾提醒呼叫急救电话

不要写成长篇背景知识或病因分析。

## 未覆盖场景的处理

如果用户遇到清单外的急救问题：

1. 先给兜底步骤：`呼叫急救 + 保证现场安全 + 避免进一步伤害`
2. 再复用 `medical-search` 检索标准急救流程
3. 优先来源：政府、医院、MSD、Mayo Clinic 等
4. 只保留现场可执行动作，不扩展成科普长文
5. 查不到可靠来源时，明确说“请立即联系线下急救人员或就近就医”

## 反模式

- 不要问太多导致错过急救时间
- 不要说“观察一下再说”来拖延严重场景
- 不要推荐偏方或民间处理法
- 不要省略急救电话提示
