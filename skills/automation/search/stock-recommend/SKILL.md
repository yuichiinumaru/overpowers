# 基于国际新闻的股票异动分析 skill

## 描述
根据国际新闻分析全球市场动态，预测指定股票市场（A股、港股、美股）中最可能发生异动（大幅涨跌）的股票，并提供原因和相关新闻链接。

## 参数
- market: 股票市场，必须是 "A股"、"港股" 或 "美股" 之一（必填）

## 使用示例
```
/stock-news-picker A股
/stock-news-picker 港股
/stock-news-picker 美股
```

## 执行步骤

### 1. 获取国际新闻
使用 WebSearch 搜索以下关键词：
- 国际财经新闻
- 全球股市动态
- 地缘政治新闻
- 货币政策变化
- 科技行业重大新闻
- 能源和大宗商品新闻

### 2. 分析市场影响
根据不同市场的特点进行分析：

**A股关注点：**
- 美联储政策对北向资金的影响
- 中美贸易关系
- 中国宏观经济数据
- 科技监管政策变化
- 房地产和基建政策

**港股关注点：**
- 美股科技股走势
- 中国经济政策
- 港币汇率
- 互联互通政策
- 阿里巴巴、腾讯等权重股消息

**美股关注点：**
- 美联储利率决策
- 科技股财报和行业动态
- 地缘政治风险
- 经济数据（就业、CPI、GDP等）
- 人工智能和科技行业趋势

### 3. 推荐异动股票
使用大模型能力分析新闻，结合以下输出格式返回结果：

## 注意事项
- 确保搜索的新闻是最新的（最近24-48小时）
- 优先选择有明确新闻催化剂的股票
- 对于A股，注意节假日休市安排
- 提供具体的新闻链接供用户验证
- 保持客观中立，不提供具体买卖建议
