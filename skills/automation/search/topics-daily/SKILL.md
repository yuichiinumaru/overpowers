# X Hot Topics Daily

## 核心目标

每天按固定话题抓取 X 热帖，产出"可快速决策"的中文简报：
- 只保留信息密度高、互动高、可行动的信息
- 去掉重复话题和低质量灌水
- 强制过滤常见营销诈骗与谣言型内容
- 默认给出"最低成本 know-what"结论，减少 FOMO

## 固定工作流

### 1) 使用 browser 打开 X

优先使用 `browser` + `profile=chrome` 打开 X 探索页面。

### 2) 按主题抓取（默认 5 个）

默认主题：`AI`、`LLM`、`中国社会热点`、`新加坡社会热点`、`美国社会热点`

### 3) 过滤与去重（强制）

1. **营销诈骗与低质过滤**
   - 丢弃包含 giveaway、抽奖、评论 done 等模式
   - 丢弃夸张收益承诺、无来源爆料

2. **去重**
   - 相同新闻事件只保留一次主条目

3. **新鲜度**
   - 优先过去 24 小时内容

4. **区域平衡**
   - 中国/新加坡/美国三个区域尽量各保留 1 条

### 4) 内容提炼（强制）

每条入选热点必须补齐：
- 发生了什么（1 句）
- 为什么值得现在关注（1 句）
- 谁最该关心
- 是否可执行

### 5) 输出格式（默认：反 FOMO 速览）

```text
【X 热点追踪｜最低成本 know-what】
日期：YYYY-MM-DD

A) 今天真正要知道的（2 条必知）
- [事件 1]：发生了什么；为什么现在重要
- [事件 2]：发生了什么；为什么现在重要

B) 你可以直接忽略的（3 条）
- [噪音类型 1]：忽略原因
- [噪音类型 2]：忽略原因
- [噪音类型 3]：忽略原因

C) 10 分钟最低成本行动
- 1) 先看：...
- 2) 再看：...
- 3) 最后判断：...

D) 风险提醒
- 今日过滤掉的疑似诈骗/低质信息类型：...
```

## 质量标准

- 全文中文，简洁但信息完整
- 每日总条目建议 5-10 条
- 不能输出空话，要给明确结论
- 重点是"可读、可判断、可行动"
