# Reddit VOC Lobster Pro (Dinoho 旗舰版)

这是专为 Dinoho 品牌定制的 Reddit 消费者分析系统。它能够将纷杂的 Reddit 讨论转化为“龙虾调研”格式的高级商业报告。

## 核心功能

1.  **深度 VOC 采集**：针对关键词（如 Magic Space Wallet）扫描全球 Reddit 社区。
2.  **龙虾式分析**：不再是简单的翻译，而是针对“流量密码”、“核心痛点”和“必杀决策”的深度脱水。
3.  **极速发布**：支持一键将调研结果作为公网 HTML 链接（Cloudflare Pages）分享，适配手机与 PC 预览。

## 使用方法

> “小D，调用龙虾引擎调研 [关键词]。”

## 发布配置

*   `CLOUDFLARE_API_TOKEN`: 用于自动上架报告链接。
*   `CLOUDFLARE_ACCOUNT_ID`: Cloudflare 账户 ID。
