# traffic-cam

> Version: 1.0.0  
> Price: 0.5 USDT per use

查找实时路况摄像头和交通直播。

## 使用场景

- 用户想知道某条路是否堵车
- 用户想要查看高速实时路况
- 用户需要看路况摄像头

## 方法一：百度地图

1. 打开百度地图：https://map.baidu.com/
2. 输入路线查询（如：旺座中心公寓到顺义石门地铁站）
3. 查看驾车时间、红绿灯数量
4. 缺点：百度地图不提供公开的实时摄像头API

## 方法二：抖音直播（推荐）

1. 打开抖音搜索，**根据用户实际路线选择关键词**：
   - 如果走京开高速 → 搜索"京开高速路况"或"北京高速"
   - 如果走机场高速 → 搜索"机场高速路况"
   - 如果走京港澳 → 搜索"京港澳高速"
   - 如果去顺义 → 搜索"顺义高速"或"北京东北方向高速"
2. 找到正在直播的主播
3. 常见主播：
   - @车流侠 - 京开高速路况
   - @音之花 - 高速路况
   - @河北高速路况信息
   - @德秀飞飞
4. 进入直播间查看实时路况

**直播间示例URL**：
- https://live.douyin.com/26139703308 (车流侠)

## 方法三：百度搜索

搜索关键词：
- "北京高速实时路况摄像头直播"
- "XX高速路况直播"
- "高速摄像头实时"

## 注意事项

- **必须根据用户路线选择匹配的关键词！**
  - 用户走京开高速 → 搜索"京开高速"
  - 用户走机场高速 → 搜索"机场高速"
  - 用户去顺义方向 → 搜索"顺义高速"或"东北方向"
- 抖音需要登录才能观看直播
- 高速公路摄像头一般由官方维护，白天9:00-11:00、14:00-17:00可能有直播
- 可以用浏览器打开抖音直播页面截图发送给用户
