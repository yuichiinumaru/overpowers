# 柯南周报 📺🔍

**名侦探柯南动画周报技能** - 每周自动收集并整理柯南最新剧情进展

## 🎯 功能特点

- **自动搜索** - 每周六晚 9 点自动搜索最新剧情
- **主线追踪** - 关注黑衣组织、APTX-4869 等主线剧情
- **特别篇报道** - 剧场版、特别篇、联动剧集信息
- **角色动态** - 声优变动、角色登场、制作信息
- **多源整理** - 整合知乎、百科、官方等多渠道信息

## ⏰ 默认执行时间

- **每周六 21:00** (Asia/Shanghai UTC+8)
- 可通过 cron 配置修改

## 📋 输出内容

每期周报包含：
1. **新播出集数** - 集数、标题、播出日期
2. **重大剧情进展** - 主线剧情推进
3. **特别篇信息** - 剧场版、特别篇、联动
4. **角色/声优动态** - 登场角色、声优变动
5. **制作信息** - 编剧、导演、制作团队
6. **来源链接** - 方便查看详情

## 🚀 使用方法

### 手动触发
```
柯南周报
```

### 自动执行（推荐）
```bash
openclaw cron add \
  --name "柯南周报" \
  --cron "0 21 * * 6" \
  --tz "Asia/Shanghai" \
  --system-event "柯南周报" \
  --session "main"
```
