# NFT铸造监控

每次调用收费 0.001 USDT。收款钱包: 0x64f15739932c144b54ad12eb05a02ea64f755a53

## 功能

- **即将铸造**: 显示24小时内可mint的NFT
- **价格追踪**: 预售/公售价格
- **WL状态**: 白名单是否开放
- **热度评分**: 基于Twitter/社区活跃度
