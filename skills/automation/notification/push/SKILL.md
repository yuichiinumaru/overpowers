# Daily Hot Push - 每日热榜推送

智能推送中国热榜 TOP10，让你不错过任何重要新闻。

## 功能特点

✅ **多源聚合** - 微博、知乎、百度、36 氪四大平台
✅ **智能筛选** - 排除明星八卦、广告推广、主观评论
✅ **简洁输出** - 只推送 10 条最重要的新闻
✅ **定时推送** - 配合 OpenClaw Cron 每天早上 8 点自动推送
✅ **飞书集成** - 直接推送到飞书账号

## 数据源

1. **微博热搜榜** - 社会热点、舆论风向
2. **知乎热榜** - 深度讨论、专业观点
3. **百度实时热点** - 大众关注、搜索趋势
4. **36 氪 24 小时热榜** - 商业科技、创业动态

## 筛选标准

只推送符合以下标准的内容：

1. ✅ **重大事件** - 政策、国际关系、社会事件
2. ✅ **热议话题** - 引发广泛讨论的重要话题
3. ✅ **事实新闻** - 基于事实的报道，非主观评论
4. ❌ **排除** - 明星八卦、娱乐营销、广告推广

## 输出格式

```
🔥 今日热榜（03 月 16 日）

1. [新闻标题 1]
2. [新闻标题 2]
3. [新闻标题 3]
...
10. [新闻标题 10]
```

## 设置定时推送

使用 OpenClaw Cron 设置每天早上 8 点自动推送：

```bash
openclaw cron add \
  --name "每日热榜推送" \
  --cron "0 8 * * *" \
  --tz "Asia/Shanghai" \
  --channel "feishu" \
  --message "使用 daily-hot-push 技能获取今日热榜 TOP10 并推送"
```

## 使用方式

- "推送今日热榜"
- "查看今天的热搜"
- "给我 10 条最重要的新闻"

## 技术实现

1. 使用 web_fetch 从 tophub.today 获取各平台热榜
2. 使用 AI 智能筛选和去重
3. 按重要性和热度排序
4. 格式化输出并推送
