# Crypto.com Trading Bot

## 功能

Crypto.com 交易机器人，支持：

1. **查询余额** - 获取钱包余额
2. **市价交易** - 快速买入/卖出
3. **限价交易** - 设置价格下单
4. **币种兑换** - 币币兑换
5. **市场数据** - 查询实时价格

## 使用示例

```
- "查询 cryptocom 余额"
- "市价买入 BTC 0.01"
- "兑换 USDT 到 ETH"
```
