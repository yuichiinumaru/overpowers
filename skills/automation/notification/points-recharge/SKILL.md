当用户提到余额不足、积分不足、需要充值、查看套餐等需求时，按以下流程处理。

## 流程步骤

### 步骤 1：获取套餐列表

调用 MCP 工具获取可用套餐，展示套餐信息包含套餐 ID、价格、积分数量、额外赠送。

### 步骤 2：询问充值选择

如果用户未指定金额和支付方式，询问选择充值金额/套餐和支付方式（微信/支付宝）。

### 步骤 3：生成支付二维码

调用 MCP 工具生成支付二维码，展示支付信息并打开二维码图片供用户扫码。

### 步骤 4：查询余额（可选）

用户支付完成后，可调用 MCP 工具查询最新余额。
