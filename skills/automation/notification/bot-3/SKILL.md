# weibo-trending-bot - 微博热搜机器人

## 描述
实时监控微博热搜榜，追踪热点话题、明星八卦、社会新闻。自动生成蹭热点文案。

## 定价
- **包月订阅**: ¥19/月
- 实时热搜推送
- 包含蹭热点文案生成

## 用法
```bash
# 查看热搜榜
/weibo-trending-bot --hot

# 监控特定话题
/weibo-trending-bot --track "科技"

# 生成蹭热点文案
/weibo-trending-bot --generate --topic "#热门话题#"

# 设置提醒
/weibo-trending-bot --alert --keyword AI
```
