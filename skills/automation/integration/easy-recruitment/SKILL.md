# 招聘助手

自动化简历筛选与面试准备工具。

## 工作流程

### Step 1: 确认职位文件夹

用户以文件夹为单位维护职位，结构：
```
jobs/
├── 后端开发 -3 年经验/
│   ├── JD.txt
│   ├── 候选人 A.pdf
│   └── ...
└── personalprefer.txt (可选)
```

### Step 2: 读取并解析 JD

提取关键要求：硬性要求、核心技能、软性素质、加分项。

### Step 3: 批量解析简历

支持格式：PDF、DOC、DOCX、HTML、TXT

### Step 4: 匹配度评分

评分算法：
- 硬性要求（一票否决）
- 核心技能匹配（40 分）
- 经验相关性（30 分）
- 项目质量（20 分）
- 加分项（10 分）

### Step 5: 生成面试报告

对每个候选人输出：
1. 匹配点总结
2. 风险点/待确认
3. 定制化面试问题清单

### Step 6: 输出格式

- 选项 A：本地 HTML 报告（推荐）
- 选项 B：Markdown 报告

## 输出分级

- 🟢 强烈推荐（80-100 分）
- 🟡 建议考虑（60-79 分）
- 🔴 不太匹配（<60 分）
