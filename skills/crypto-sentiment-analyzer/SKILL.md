# crypto-sentiment-analyzer - 加密市场情绪智能分析系统

**痛点**：市场情绪难以量化，容易被 FOMO/FUD 影响，错过最佳买卖点
**价值**：AI 分析全网舆情，量化市场情绪，识别反向交易信号

## 使用方式

```bash
crypto-sentiment-analyzer --help
```

## 功能特点

- ⚡ **社交媒体监控** - 实时分析 Twitter/Reddit/Telegram 等全网舆情
- 📊 **情绪指数计算** - 基于 NLP 的多维度情绪评分，量化市场热度
- 💰 **KOL 观点追踪** - 监控行业大 V 观点，识别共识和分歧
- 🔒 **极端情绪预警** - 贪婪/恐惧指数预警，识别顶部和底部信号
- 🎯 **交易信号生成** - 基于情绪背离生成反向交易信号

## 适用场景

- 市场情绪量化分析
- 加密货币交易决策辅助
- 舆情风险预警
- 反向指标识别

## 配置

在 `~/.openclaw/workspace/config/crypto-sentiment-analyzer.json` 配置 SkillPay API Key。
