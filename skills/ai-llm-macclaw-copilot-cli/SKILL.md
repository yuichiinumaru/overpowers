# MacClaw Copilot CLI

使用 GitHub Copilot CLI 分析代码。

## 安装 (Installation)

```bash
brew install copilot-cli
```

## 使用 (Usage)

```bash
COPILOT_GITHUB_TOKEN=$(cat ~/.copilot/github_token.txt) copilot -p "你的问题"
```
