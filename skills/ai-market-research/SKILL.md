# ai-market-research - AI 智能市场调研分析系统

**痛点**：市场调研耗时，数据分散难整合，竞品情报难获取，决策缺乏依据
**价值**：AI 自动收集分析市场数据，快速生成洞察，支持科学决策

## 使用方式

```bash
ai-market-research --help
```

## 功能特点

- ⚡ **竞品情报收集** - 自动追踪竞品动态、价格、营销、产品更新
- 📊 **行业趋势预测** - 基于大数据预测行业发展趋势和机会
- 💰 **消费者洞察分析** - 分析消费者行为、偏好、痛点
- 🔒 **市场规模估算** - 多维度估算市场规模和增长潜力
- 🎯 **进入策略建议** - 基于市场分析给出市场进入策略建议

## 适用场景

- 新产品市场调研
- 竞品分析和对标
- 行业机会评估
- 商业计划书数据支持

## 配置

在 `~/.openclaw/workspace/config/ai-market-research.json` 配置 SkillPay API Key。
