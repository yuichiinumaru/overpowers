# 航空货运政策分析维度详解

## 一、补贴力度维度

### 1.1 财政补贴金额
- 一次性补贴：航线开通补贴、货量达标奖励
- 持续补贴：运营补贴、箱量补贴
- 专项补贴：跨境电商、国际货运

### 1.2 补贴计算方式
- 按货量：元/吨、元/TEU
- 按航班：元/班次
- 按增长：增量奖励
- 比例补贴：地方留成返还

### 1.3 补贴上限
- 年度最高限额
- 单航线最高限额
- 单企业最高限额

### 2.1 航线覆盖

### 2.2 货量要求
- 基础货量门槛
- 增量要求
- 特殊货类（生鲜、跨境电商）

### 2.3 企业类型
- 货运航空公司
- 航空货运代理
- 跨境电商企业
- 物流企业

### 5.2 续期情况
- 政策延续性好
- 政策调整频繁
