# Clipzy 在线剪贴板 分类接口

提供 Clipzy 在线剪贴板相关能力，包括加密内容存储、按 ID 获取密文，以及在受控场景下按密钥读取原文。

这个标签保留接口用途说明，但移除了不适合代码生成器处理的长篇 HTML 和脚本示例。

## 什么时候先读这个页面

- 当你已经确定要用这个 skill，但还没决定具体选哪个接口。
- 当同一类能力下有多个接口，需要先快速对比用途、路径和操作方式。

## 选型提醒

- 先让接口摘要和用户目标一一对应，再继续看参数页，不要只看名字就直接调用。
- 如果用户同时给了多个候选需求，优先选最贴近最终结果的接口，而不是最底层的原子能力。
- 进入具体接口页以后，再确认必填参数、默认值、互斥关系和报错处理。
