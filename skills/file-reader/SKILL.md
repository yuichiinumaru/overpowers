# file-reader

## 概述
文件读取技能。当用户需要读取文件内容、查看文件、cat 文件时激活。支持读取文本文件、配置文件、代码文件等。


## 基本使用方法
```bash
python  read.sh "<file>"
```



### 命令行参数

| 参数 | 说明 |
| --- | --- |
| `<file>` | 要读的文本（必填） |
