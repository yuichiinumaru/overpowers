# WeChat Moments Copywriter Lite

A Moments copywriting generator based on the **McKinsey Trust Formula**:

```
Trust = (Professionalism × Reliability × Intimacy) / Self-Interest
```

## Supported Copywriting Types

| Command | Description | Formula |
|------|------|------|
| `/moments professional` | Professional type: Build authority | Story case + Details + Beautiful results |
| `/moments reliable` | Reliable type: Build trust | Failure story + Persistent process + Success results |
| `/moments warm` | Warm type: Build intimacy | Life scenario + Real interaction + Emotional connection |
| `/moments altruistic` | Altruistic type: Reduce defense | Event + Details + Explanation + Values/Golden sentence |
| `/moments counter` | Counter-cognitive breakthrough | Break cognition + Plant concept |
| `/moments target` | Circle users | Filter target audience + Build boundaries |
| `/moments intro_100` | Self-introduction 100 chars | Deep cultivation field + Pitfall experience + Value hook |

## Formatting Standards

- First paragraph: 20-25 characters or less, scenario-based entry
- Paragraphs: No more than 3 lines per paragraph, one blank line between paragraphs
- Images: Use 1, 4, or 9 images
- Comments section: Marketing information goes in comments

## Usage Examples

```
/moments professional Outdoor consulting scenario
/moments warm Child says "mom you're the best"
/moments counter "Tutoring guarantees score improvement" is a lie
/moments intro_100 My core value proposition
```
