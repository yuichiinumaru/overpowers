# SDD 추론 규칙 상세

## 네비게이션 추론 패턴

### 버튼/링크 텍스트 → 기대 동작 매핑

| 텍스트 패턴 | 기대 동작 | 신뢰도 |
|------------|----------|--------|
| 편집, 수정, edit, modify, update | 해당 항목 편집 화면/모달 | HIGH |
| 상세, 보기, detail, view, info | 해당 항목 상세 화면 | HIGH |
| 삭제, 제거, delete, remove | 확인 다이얼로그 → 삭제 실행 | HIGH |
| 목록, 돌아가기, back, list | 상위 목록 화면 | HIGH |
| 생성, 추가, 새로만들기, new, create, add | 생성 폼/모달 | HIGH |
| 저장, save, submit, 확인, ok | 데이터 저장 + 성공 피드백 | HIGH |
| 취소, cancel, close, 닫기 | 이전 상태 복귀, 변경 폐기 | HIGH |
| 검색, search, 찾기 | 검색 결과 표시 | HIGH |
| 다운로드, download, export | 파일 다운로드 시작 | MEDIUM |
| 공유, share | 공유 옵션 표시 | MEDIUM |
| 설정, settings, config | 설정 화면 이동 | MEDIUM |
| 로그아웃, logout, sign out | 로그인 화면 이동 | HIGH |

### 행/카드 클릭 → 기대 동작

| 컨텍스트 | 기대 동작 | 신뢰도 |
|---------|----------|--------|
| 테이블 행 클릭 | 해당 항목 상세 화면 | HIGH |
| 카드 컴포넌트 클릭 | 해당 항목 상세 화면 | HIGH |
| 리스트 아이템 클릭 | 해당 항목 상세 화면 | HIGH |
| 탭 클릭 | 해당 탭 콘텐츠 표시 | HIGH |
| 아코디언 클릭 | 펼치기/접기 토글 | HIGH |

## 상태 변경 추론 패턴

### CRUD 후 기대 상태

| 동작 | 기대 상태 | 검증 방법 |
|-----|----------|----------|
| Create 성공 | 목록에 새 항목 존재 | 목록 재조회 후 항목 검색 |
| Read | 요청 ID와 일치하는 데이터 표시 | 표시 데이터의 ID 일치 확인 |
| Update 성공 | 상세에서 변경값 반영 | 상세 재조회 후 값 비교 |
| Delete 성공 | 목록에서 해당 항목 소멸 | 목록 재조회 후 항목 부재 확인 |

### 폼 상호작용

| 동작 | 기대 상태 |
|-----|----------|
| 필수 필드 비우고 제출 | 유효성 에러 메시지 표시 |
| 유효하지 않은 이메일 입력 | 이메일 형식 에러 |
| 비밀번호 불일치 | 불일치 에러 |
| 성공적 제출 | 성공 피드백 + 다음 화면 이동 |

## 접근성/UX 추론 패턴

| 상황 | 기대 UI | 비정상 |
|-----|---------|--------|
| 데이터 0건 | empty state 일러스트/텍스트 | 빈 화면 또는 깨진 레이아웃 |
| API 호출 중 | 로딩 스피너/스켈레톤 | 아무 표시 없음 |
| API 에러 | 에러 메시지 + 재시도 옵션 | 무반응 또는 빈 화면 |
| 권한 없는 페이지 접근 | 403 안내 또는 리다이렉트 | 빈 화면 또는 깨진 UI |
| 긴 텍스트 | 말줄임 또는 스크롤 | 레이아웃 깨짐/오버플로우 |

## API 추론 패턴

| 메서드 + 경로 패턴 | 기대 응답 |
|-------------------|----------|
| GET /items | 목록 배열 반환 |
| GET /items/:id | 단일 항목 객체 반환 |
| POST /items | 생성된 항목 반환 + 201 |
| PUT /items/:id | 수정된 항목 반환 + 200 |
| DELETE /items/:id | 204 또는 200 + 이후 GET 시 404 |
| POST /auth/login | 토큰/세션 반환 |
| POST /auth/logout | 세션 무효화 + 이후 인증 API 401 |
