测试和校验模型的定价信息展示与实际扣费是否正确。

## 价格换算规则

```
1 美元 = 400 积分
```

## 测试流程

### 第一步：获取官方定价

访问模型的 fal.ai 定价页面，记录价格信息并换算为积分。

### 第二步：检查 Executor 配置

检查 translate_api/app/api/v3/executors/ 下对应 Executor 的配置。

### 第三步：测试定价信息展示

检查 price_type、price_description、examples 价格计算是否正确。

### 第四步：测试实际扣费

创建测试任务，验证实际扣费是否与预期一致。

### 第五步：批量测试

运行测试脚本进行批量测试。
