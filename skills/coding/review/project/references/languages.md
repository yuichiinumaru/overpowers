# GitHub Trending - 编程语言参数 (l)

用于 `?l=` 参数，值需小写。以下为常用语言及对应 URL 值。

| 显示名称 | 参数值 (l) |
|---------|------------|

| JavaScript | javascript |

| C | c |
| C++ | cpp |

| Kotlin | kotlin |
| Swift | swift |
| Shell | shell |
| HTML | html |
| CSS | css |
| Vue | vue |
| Scala | scala |
| Lua | lua |
| Dart | dart |

| Cobol | cobol |
| Erlang | erlang |
| F# | fsharp |
| PowerShell | powershell |
| TeX | tex |
| Vim script | vim-script |
| Makefile | makefile |
| Dockerfile | dockerfile |
| Jupyter Notebook | jupyter-notebook |

> 说明：GitHub 支持更多语言，完整列表可在 https://github.com/trending 页面点击 "Language: Any" 下拉菜单查看。参数值需与 GitHub 内部标识一致。
