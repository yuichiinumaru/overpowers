# 06 — 工具推荐

> ⚠️ 以下信息截至 2026-02，工具能力和 API 可能已更新，请以官方文档为准。

## AI IDE 选择

| 场景 | 推荐 |
|------|------|
| 终端极客，要极致控制 | **Claude Code** |
| 多 Agent 协作 + 互相 review | **Claude Code Agent Teams** |
| 大规模并行（5+ Agent） | **Kimi K2.5 Agent Swarm** |
| 跨编辑器+终端+浏览器 | **Google Antigravity** |
| Agent 长期记忆/知识复用 | **Google Antigravity (Knowledge Base)** |
| 混合不同模型 | **OpenClaw sessions_spawn** |
| 快速原型验证 | **Bolt.new / Lovable** |

## 三大工具定位

```
Claude Code  — 最强单兵 + 团队协作
               终端原生，Agent Teams 互相 challenge
               适合: 深度控制和代码审查

Kimi K2.5    — 最强并行编排
               Agent Swarm 100 并行，Critical Path 调度
               开源模型，自部署成本低

Antigravity  — 最强自主性 + 知识积累
               跨编辑器/终端/浏览器自主操作
               Knowledge Base 让 Agent 越用越聪明
               Kit 2.0: 16+ 预设 Agent + 11 Workflow
```

## 辅助工具

| 工具 | 用途 |
|------|------|
| **rulebook-ai** | 统一规则文件生成（10+ Agent） |
| **vibe-rules** | npm 级规则分发 |
| **ruler** | 一套规则部署到所有 Agent |
| **Sentry MCP** | 生产错误直通 Agent 上下文 |
