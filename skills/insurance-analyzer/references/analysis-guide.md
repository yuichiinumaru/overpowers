# 保险保单分析指南

本指南提供详细的保单分析方法论。

---

## 一、分析维度详解

### 1.1 覆盖全面性评估

#### 基础保障检查清单

| 保障类型 | 必备程度 | 说明 |
|----------|----------|------|
| 医保 | ⭐⭐⭐⭐⭐ | 社会医疗保险，必备 |
| 百万医疗险 | ⭐⭐⭐⭐⭐ | 报销大额医疗费用 |
| 重疾险 | ⭐⭐⭐⭐ | 确诊即赔，覆盖康复费用 |
| 意外险 | ⭐⭐⭐⭐ | 覆盖意外身故/伤残/医疗 |
| 寿险 | ⭐⭐⭐ | 家庭经济支柱必备 |
| 年金险 | ⭐⭐ | 养老储备，可选 |

#### 家庭角色配置建议

| 角色 | 必备保障 |
|------|----------|
| 家庭经济支柱 | 医疗+重疾+意外+寿险 |
| 配偶 | 医疗+重疾+意外 |
| 子女 | 医疗+重疾+意外 |
| 老人 | 医疗+意外+防癌险 |

### 1.2 保额充足性评估

#### 重疾险保额参考

| 费用类型 | 建议保额 |
|----------|----------|
| 基础治疗费 | 30-50万 |
| 康复费用 | 10-20万 |
| 收入损失（3-5年） | 年收入 × 3-5 |
| **合计建议** | **50-100万** |

#### 医疗险保额参考

| 类型 | 建议保额 |
|------|----------|
| 百万医疗险 | 200-400万 |
| 中端医疗险 | 100-300万 |
| 高端医疗险 | 无限额/1000万+ |

#### 寿险保额参考

| 覆盖风险 | 计算方式 |
|----------|----------|
| 房贷余额 | 剩余贷款本金 |
| 子女教育 | 教育费用预估 |
| 父母赡养 | 赡养费用 × 年限 |
| 家庭支出 | 月支出 × 12 × 年限 |
| **合计建议** | **年收入 × 5-10年** |

### 1.3 费用合理性评估

#### 保费占比建议

| 指标 | 建议比例 | 说明 |
|------|----------|------|
| 保费/收入 | 5-10% | 家庭总保费占家庭年收入 |
| 保费/储蓄 | 10-20% | 保费占可支配收入 |

#### 不同险种保费参考（30岁成年人）

| 保障类型 | 主流产品年保费 |
|----------|----------------|
| 百万医疗险 | 300-500元 |
| 重疾险（50万保额） | 3000-6000元 |
| 意外险（50万保额） | 100-300元 |
| 定期寿险（50万保额） | 500-1000元 |

### 1.4 家庭适配度评估

#### 人生阶段与保障重点

| 阶段 | 保障重点 |
|------|----------|
| 单身期 | 医疗+意外+重疾（保额可较低） |
| 结婚后 | 补充寿险，关注配偶保障 |
| 有子女 | 加保寿险，确保子女教育 |
| 子女独立 | 可降低寿险，关注养老 |
| 退休期 | 医疗+意外，储蓄型保险 |

## 二、常见问题诊断

### 2.1 保额不足

**表现**：
- 重疾险保额 < 30万
- 医疗险保额 < 100万
- 寿险保额 < 年收入 × 5年

**建议**：
- 优先加保消费型产品提升保额
- 预算充足可考虑储蓄型产品

### 2.2 保障缺失

**常见缺失组合**：
- 有重疾无医疗 → 补充百万医疗
- 有医疗无意外 → 补充意外险
- 有社保无商保 → 逐步补充

### 2.3 重复保障

**表现**：
- 多份医疗险（报销型，不能重复赔付）
- 保障内容重叠

**建议**：
- 医疗险保留一份即可
- 重疾险可叠加赔付

### 2.4 费用过高

**表现**：
- 保费占比超过15%
- 储蓄型保险占比过高

**建议**：
- 优先配置消费型保障
- 减少储蓄型保险投入

## 三、分析输出规范

### 3.1 必须包含要素

1. **保单梳理表** — 提取的保单关键信息
2. **综合评分** — 覆盖/保额/费用/适配度
3. **问题诊断** — 发现的具体问题（按严重程度排序）
4. **优化建议** — 可执行的改进建议（按优先级排序）

### 3.2 评分标准

| 评分 | 含义 |
|------|------|
| ⭐⭐⭐⭐⭐ | 优秀，无需改进 |
| ⭐⭐⭐⭐☆ | 良好，可优化 |
| ⭐⭐⭐☆☆ | 一般，建议改进 |
| ⭐⭐☆☆☆ | 较差，需要改进 |
| ⭐☆☆☆☆ | 严重不足，强烈建议改进 |

---

## 四、注意事项

1. 以保单合同条款为准
2. 分析仅供参考，不构成购买建议
3. 如有疑问建议咨询专业人士
4. 注意区分消费型和储蓄型产品
5. 关注等待期、犹豫期、宽限期等关键时间节点
