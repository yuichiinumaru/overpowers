# Crypto Learning - 加密货币自学系统

## 目标

给用户提供"每天可直接学习"的加密货币课程推送：
- 解释清楚（小白能看懂）
- 内容完整（不是只有题目）
- 有最新资料整合（web_search 检索）
- 有可执行学习动作（自测/作业）

## 学习大纲

学习大纲在 `content.json`：
- 小白向（30 天）
- 投资向（60 天）
- 进阶投资（90 天）

进度在 `progress.json`。

## 输出质量规则（硬约束）

每次输出学习报告都必须满足：

1. **禁止只输出题目或一句话摘要**
2. **最少包含 6 个模块**（见下方固定模板）
3. **正文长度不少于 450 字**
4. **术语必须解释**：首次出现术语时给白话解释
5. **至少给 2 个生活类比或场景例子**
6. **至少给 3 条要点总结**
7. **必须含 3 道自测题 + 标准答案**
8. **最后给出次日预告**，并更新学习进度

## 固定输出模板

1. `📚 第 X 天学习内容`（阶段 / 主题 / 今日重点）
2. `📖 先讲人话版`（3-6 句白话解释）
3. `🧱 系统化知识`（定义、原理、关键组成、常见误区）
4. `🌐 最新资料整合`（基于 web_search 的信息融合）
5. `🧠 例子与类比`（至少 2 个）
6. `✅ 今日小结`（至少 3 条可记忆要点）
7. `📝 3 题自测（含答案）`
8. `🚀 明日预告 + 进度更新`

## 工作流程

1. 读取进度
2. 获取今日主题
3. web_search 检索并整合
4. 生成详细学习报告
5. 更新进度

## 用户命令

- **开始学习计划**：按当前进度继续推送
- **查看进度**：显示已完成天数、当前阶段/主题
- **跳过今天**：记录跳过日期，不推进学习内容
- **重置计划**：清空进度并从第一个主题开始
