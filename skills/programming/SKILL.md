# Programming Skill

这是一个专门用于编程任务的技能，提供以下功能：

## 功能描述

1. **Claude CLI 编程指南** - 演示如何使用 Claude CLI 工具进行编程任务
2. **编程工作流程** - 提供标准化的编程开发流程指导
3. **代码审查和优化** - 提供代码质量检查和性能优化建议

## 使用场景

- 当 OpenClaw 需要执行编程任务时
- 代码开发、调试和优化
- 项目结构设计和实现
- 技术选型和架构决策支持

## 输入参数

- `task`: 要执行的编程任务描述
- `context`: 相关上下文信息（现有代码、需求文档等）
- `requirements`: 特定的技术要求或约束

## 执行流程

1. 分析编程任务需求
2. 选择合适的编程方法和工具
3. 实施编码或优化建议
4. 提供结果反馈和进一步建议
