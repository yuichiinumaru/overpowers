# 大乐透开奖查询技能 (多数据源版)

从中国体彩网获取官方大乐透开奖数据，支持多个数据源自动切换，提供开奖结果查询和中奖规则说明。

## 何时使用

✅ **使用此技能：**
- "查询最新大乐透开奖结果"
- "大乐透 2026025 期中奖号码"
- "帮我看看这期大乐透开了什么"
- "大乐透中奖规则是什么"
- 彩票对奖、号码核对

❌ **不使用此技能：**
- 其他彩种查询（双色球、福彩 3D 等）
- 彩票预测、选号建议
- 历史数据统计分析
- 海外彩票查询

## 快速开始

### 方式一：使用 web_fetch 获取（推荐）

```bash
# 数据源 1: 中国体彩网（官方，优先）
web_fetch --url "https://www.lottery.gov.cn/kj/kjlb.html?dlt" --maxChars 8000

# 数据源 2: 体彩网手机版（备用）
web_fetch --url "https://m.lottery.gov.cn/dlt/" --maxChars 8000

# 数据源 3: 500 彩票网（第三方，备用）
web_fetch --url "https://www.500.com/dlt/" --maxChars 8000
```

### 方式二：使用 Python 脚本（自动多源切换）

```bash
# 查询最新一期（自动尝试多个数据源）
python scripts/dlt_lottery.py

# 查询指定期号
python scripts/dlt_lottery.py 2026025
```

## 数据源策略

### 优先级顺序

```
1. https://www.lottery.gov.cn/kj/kjlb.html?dlt  (中国体彩网 - 官方)
   ↓ 失败
2. https://m.lottery.gov.cn/dlt/  (体彩网手机版 - 官方备用)
   ↓ 失败
3. https://www.500.com/dlt/  (500 彩票网 - 第三方)
   ↓ 失败
4. https://lottery.sina.com.cn/dlt/  (新浪彩票 - 第三方)
```

### 自动切换逻辑

1. **首选官方渠道**：中国体彩网 (lottery.gov.cn)
2. **官方渠道失败**：尝试手机版 (m.lottery.gov.cn)
3. **官方全部失败**：使用第三方数据源
4. **数据校验**：交叉验证多个数据源确保准确性

## 大乐透规则

### 基本玩法

| 类型 | 选择范围 | 数量 |
|------|---------|------|
| 前区 | 1-35 | 5 个 |
| 后区 | 1-12 | 2 个 |

### 中奖规则

| 奖级 | 中奖条件 | 说明 |
|------|---------|------|
| 一等奖 | 5 前 + 2 后 | 全部命中 |
| 二等奖 | 5 前 + 1 后 | 前区全中 |
| 三等奖 | 5 前 + 0 后 | 前区全中 |
| 四等奖 | 4 前 + 2 后 | - |
| 五等奖 | 4 前 + 1 后 | - |
| 六等奖 | 3 前 + 2 后 | - |
| 七等奖 | 4 前 + 0 后 | - |
| 八等奖 | 3 前 + 1 后 或 2 前 + 2 后 | - |
| 九等奖 | 3 前 + 0 后 或 2 前 + 1 后 或 1 前 + 2 后 或 0 前 + 2 后 | 只中后区也有奖 |

### 追加投注

- 追加投注每注多加 1 元
- 中得一、二等奖时，可多得 80% 奖金

### 开奖时间

- **开奖日**：每周一、三、六
- **开奖时间**：约 21:10
- **销售截止**：开奖日 20:00

## 完整工作流程

### 步骤 1：按优先级尝试数据源

```
尝试数据源 1 (体彩网) → 成功？→ 解析数据
                          ↓ 失败
尝试数据源 2 (手机版) → 成功？→ 解析数据
                          ↓ 失败
尝试数据源 3 (500 网) → 成功？→ 解析数据
                          ↓ 失败
尝试数据源 4 (新浪)   → 成功？→ 解析数据
                          ↓ 失败
返回错误提示
```

### 步骤 2：解析数据

从返回的 HTML 中提取：
- 期号（如 2026025）
- 开奖日期
- 前区号码（5 个，范围 1-35）
- 后区号码（2 个，范围 1-12）
- 奖池金额（如有）

### 步骤 3：格式化输出

使用标准格式：
```
🎱 大乐透第 XXXXXX 期
📅 开奖日期：YYYY-MM-DD
🔴 前区：XX XX XX XX XX
🔵 后区：XX XX
💰 奖池：X.XX 亿元
📊 数据来源：中国体彩网

📋 中奖规则：
   一等奖：5 前 + 2 后｜二等奖：5 前 + 1 后
   ...
```

## 期号格式

大乐透期号格式为 `YYYYNNN`：
- `YYYY`：年份
- `NNN`：当年第几期

例如：
- `2026025` = 2026 年第 25 期
- `2025150` = 2025 年第 150 期

## 错误处理

### 常见问题

**问题 1：所有数据源都无法连接**
- 解决：检查网络连接，稍后重试

**问题 2：期号不存在**
- 解决：验证期号格式是否正确，或该期尚未开奖

**问题 3：数据解析失败**
- 解决：网站结构可能变化，建议访问官网查看

**问题 4：多个数据源数据不一致**
- 解决：以官方数据源 (lottery.gov.cn) 为准

## 注意事项

1. **官方数据**：优先使用中国体彩网数据，以官网为准
2. **网络依赖**：需要访问外部网站，确保网络畅通
3. **开奖时间**：周一、三、六约 21:10 开奖，其他时间查询为历史数据
4. **动态加载**：部分网站使用 JavaScript 动态加载，web_fetch 可能获取不完整
5. **理性购彩**：本技能仅提供信息查询，不构成购彩建议
