# Affiliate Marketing

AIアフィリエイトマーケティング。商品選定、コンテンツ最適化、リンク管理、コンバージョン追跡。

## 収益モデル

```
アフィリエイト収益 = トラフィック × クリック率 × コンバージョン率 × 報酬額

例:
├── 月間PV: 10,000
├── CTR: 5%
├── CVR: 3%
├── 報酬: $10/件
└── 月収: $150
```

## コンテンツ戦略

### レビュー記事

```yaml
Structure:
  1. hook: 問題提起/共感
  2. intro: 商品紹介
  3. features: 主要機能（5-7点）
  4. pros_cons: メリット/デメリット
  5. comparison: 競合比較
  6. use_cases: 使用シーン
  7. pricing: 価格情報
  8. verdict: 総評
  9. cta: 購入リンク

SEO Keywords:
  - "[商品名] レビュー"
  - "[商品名] 口コミ"
  - "[商品名] vs [競合]"
  - "[商品名] メリット デメリット"
```

### 比較記事

```yaml
Structure:
  1. intro: 比較の目的
  2. criteria: 評価基準
  3. overview: 各商品概要
  4. comparison_table: 比較表
  5. detailed_analysis: 詳細分析
  6. recommendations: おすすめ
  7. cta: 各商品リンク

High-Converting Formats:
  - "おすすめ[商品カテゴリ]10選"
  - "[A] vs [B] 徹底比較"
  - "[カテゴリ]の選び方"
```

### ハウツー記事

```yaml
Structure:
  1. problem: 解決したい問題
  2. solution_overview: 解決策概要
  3. steps: ステップバイステップ
  4. tools_needed: 必要なツール
  5. tips: コツ・注意点
  6. product_recommendation: 関連商品
  7. cta: 購入リンク

Natural Integration:
  - 手順の中で自然に紹介
  - 「私が使っているのは〜」形式
  - 具体的な使用例
```

## SEO最適化

### キーワード戦略

```yaml
Buyer Intent Keywords:
  high_intent:
    - "[商品名] 購入"
    - "[商品名] 最安値"
    - "[商品名] クーポン"
  medium_intent:
    - "[商品名] レビュー"
    - "[商品名] 比較"
    - "おすすめ [カテゴリ]"
  low_intent:
    - "[カテゴリ] 選び方"
    - "[カテゴリ] とは"

Long-tail Strategy:
  - 競合が少ないキーワード狙い
  - 具体的なニーズに対応
  - 複合キーワード活用
```

### オンページSEO

```yaml
Title Optimization:
  - キーワードを前半に
  - 数字を含む
  - 年号を含む（2026年版）
  - 60文字以内

Meta Description:
  - CTAを含む
  - 利点を強調
  - 155文字以内

Content Structure:
  - H2/H3で構造化
  - FAQ section追加
  - 比較表の活用
  - 画像にalt属性
```

## リンク管理

### リンク配置戦略

```yaml
Placement:
  above_fold: 1-2リンク（早期CTAテスト）
  in_content: 3-5リンク（文脈内）
  comparison_table: 各商品にリンク
  sidebar: 関連商品
  end_of_article: まとめCTA

Link Types:
  text_link: 自然な文章内リンク
  button: 目立つCTAボタン
  image_link: 商品画像リンク
  table_link: 比較表内リンク

Best Practices:
  - nofollow属性を付ける（規約遵守）
  - 短縮URLまたはトラッキングURL
  - リンク切れチェック自動化
```

### トラッキング

```yaml
UTM Parameters:
  utm_source: サイト名
  utm_medium: affiliate
  utm_campaign: 記事ID
  utm_content: リンク位置

Link Format:
  https://affiliate.com/product?
  utm_source=mysite&
  utm_medium=affiliate&
  utm_campaign=review-001&
  utm_content=cta-button
```

## コンプライアンス

### 必須事項

```
日本:
├── アフィリエイト広告の明示（PR、広告）
├── 景品表示法遵守
├── 薬機法遵守（健康関連）
└── 特商法表記

米国:
├── FTC開示（Affiliate Disclosure）
├── 正直なレビュー
└── 誇大広告禁止

共通:
├── 各ASPの規約遵守
├── リンクにrel="nofollow sponsored"
├── 虚偽の体験談禁止
└── ステマ禁止
```

### 開示テンプレート

```markdown
【PR】この記事にはアフィリエイトリンクが含まれています。
購入いただくと、私に報酬が支払われる場合があります。
これにより、読者の皆様に追加コストは発生しません。
```

## 失敗分析・学習

```yaml
Track Failures:
  - 低CTR記事
  - 低CVR商品
  - リンク切れ
  - 規約違反警告
  - 売上0の商品

Analysis:
  low_ctr:
    check: CTAの文言、配置、デザイン
    action: A/Bテスト実施

  low_cvr:
    check: 商品選定、ターゲット一致度
    action: 商品変更または記事リライト

  no_sales:
    check: トラフィック、商品需要
    action: SEO強化または商品変更

Learning Storage:
  path: /moltworker/data/affiliate-learning.json
  metrics:
    - product_performance
    - content_patterns
    - cta_effectiveness
```
