# 各平台文案风格适配指南

## 公众号风格
✅ 特点：正式、有深度、逻辑性强、篇幅较长
✅ 结构：
- 标题：有悬念/有痛点/有干货，长度20-30字
- 开头：钩子引入（痛点/热点/故事），引导继续阅读
- 正文：分点论述，逻辑清晰，每段3-5行，适当加小标题
- 结尾：总结核心观点，引导点赞/在看/关注/转发
✅ 语气：专业、客观、有深度，避免太口语化
✅ 字数：1500-3000字最佳

## 小红书风格
✅ 特点：活泼、利他、有画面感、实用性强
✅ 结构：
- 标题：有数字/有痛点/有利益点，长度15-25字，包含核心关键词
- 开头：前3行直接点出痛点/利益点，吸引继续看
- 正文：短段落，每段不超过3行，用emoji分隔，分点清晰
- 结尾：引导互动（点赞/收藏/评论/关注）
✅ 语气：像朋友分享，亲切自然，多用"宝子""家人们""谁懂啊"这类词
✅ 标签：5-7个标签，覆盖大中小流量词
✅ 字数：300-800字最佳

## 抖音/短视频脚本风格
✅ 特点：口语化、有节奏感、开头3秒抓注意力
✅ 结构：
- 开头（0-3s）：钩子直接炸场，比如"别划走！这个秘密我只说一次！"
- 中间（3-20s）：核心内容，节奏快，每1-2秒一个信息点，多用问句、感叹句
- 结尾（20-30s）：引导互动（点赞/关注/评论/转发/进直播间）
✅ 语气：口语化，有情绪起伏，避免书面语，多用"家人们""老铁""你知道吗"这类词
✅ 字数：300字以内，语速每分钟180-220字

## 知乎风格
✅ 特点：专业、干货足、逻辑严谨、有说服力
✅ 结构：
- 开头："谢邀/人在美国刚下飞机/利益相关匿了"（可选），直接给出结论
- 正文：分点论述，论据充分，有数据/案例支撑，逻辑链完整
- 结尾：总结核心观点，引导互动
✅ 语气：专业、客观、理性，避免情绪化表达
✅ 字数：1000-5000字最佳，越长越有流量

## 正式公文/商务风格
✅ 特点：严谨、正式、简洁、无歧义
✅ 结构：
- 开头：说明目的/背景
- 正文：分点陈述，条理清晰，用词精准，没有模糊表述
- 结尾：明确要求/下一步动作
✅ 语气：正式、客观、中立，避免口语化、情绪化表达
✅ 禁用词：网络热词、模糊表述（大概/可能/差不多）、感叹句、反问句
✅ 字数：根据需求，简洁明了

## 朋友圈风格
✅ 特点：轻松、真实、有温度、接地气
✅ 结构：
- 开头：生活化表达，比如"今天终于搞定了这件事！"
- 正文：真实分享感受/生活/工作，适当加表情
- 结尾：可以引导互动，也可以自然结束
✅ 语气：自然、真实、像和朋友聊天，避免太正式
✅ 字数：100-300字最佳，配图1/3/6/9张
