# 脑洞工厂 - 最强平行宇宙生成器

**一句话把任何场景炸成平行宇宙！**  
把普通问题瞬间变成荒诞新闻、搞笑后果、实用建议、AI画图词和朋友圈金句。

### 什么时候触发这个 Skill？
- 用户说：“what if 我辞职去日本当和尚？”
- “假如恐龙没灭绝现在会怎样？”
- “给我脑洞一个猫咪统治世界的世界”
- “平行宇宙里的我今天在干嘛”
- 任何包含“假如”“如果”“what if”“脑洞”“平行宇宙”“假如我”“如果世界”“alternate reality”的请求

### 使用方法（Agent 必须严格按以下步骤执行）
1. **确认主题**  
   提取用户核心“假如”场景。如果用户描述模糊，先问一句澄清。

2. **必须生成以下 5 个维度内容**（全部输出，不能省略）：
   - **新闻头条版**：像真实新闻网站一样写 150–250 字文章，包含标题、发布时间、来源
   - **3 个荒诞后果**：用 bullet points，极致搞笑或惊悚
   - **人生建议版**：如果你真的活在那个宇宙，该怎么活？实用 + 幽默
   - **图片提示词**：给 Grok Imagine / Flux / Midjourney 用的超详细英文 prompt，提供 3 个不同风格
   - **Meme 台词包**：5 条可直接发朋友圈/小红书/X 的金句，带 emoji

3. **额外彩蛋**（强烈推荐添加）：
   - 结尾加一句“本宇宙发生概率：XX%”并随机生成 1–99 的数字
   - 如果主题涉及东京/日本，自动加入真实地名（如涩谷、秋叶原）、文化梗（如社畜、推活、拉面）

### 示例输出格式（严格参考）
**What If：我现在立刻辞职去东京当全职寿司师傅**

**📰 平行宇宙头条**

东京，2026年3月2日 —— 来源：朝日新闻平行宇宙版

**🤯 三个荒诞后果**
- 你前老板现在每天给你转账求你回去写 bug
- 寿司店门口排队到新宿，米其林直接给你三星
- 日本网友把你 P 成动漫男主，推特热搜第一

**💡 如果你真的活在这个宇宙**

- 第二步：学会说“いらっしゃいませ”比写代码重要
- 第三步：每天发小红书记录“从码农到寿司帝王”逆袭日记

### 高级玩法（可以告诉用户）
- 追问“继续这个宇宙的故事” → 继续写第二章
- 说“更黑暗版”“更甜宠版”“赛博朋克版”“更真实版” → 切换风格
