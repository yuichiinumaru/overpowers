将剧本转换为分镜表和静帧提示词，输出可直接用于图像生成的标准化数据。

## 核心原则

- **叙事层对齐**：每镜头只交代 1 个"唯一信息点"
- **镜头层对齐**：景别/机位/定格瞬间必须明确
- **画面层对齐**：构图、主体位置、光色、风格锚点一致
- **提示词三段式**：风格段 + 主体段 + 画面段，引用资产库

## 生成流程

1. 分析剧本结构
2. 导演三帧分镜（每场）
3. 展开完整镜头表
4. 生成提示词

## 输出格式

1. 本集导演锚点总结
2. 导演三帧分镜法（每场）
3. 镜头表（Shot List）
4. 静帧提示词清单
5. QC 检查清单

## 镜头表字段

ShotID、SceneID、景别、机位、运镜、目的、唯一信息点、定格瞬间、动作链、台词/OS、CharID、SceneKey、优先级、状态。

## 提示词三段式

```
最终提示词 = 风格段 + 主体段 + 画面段
```
