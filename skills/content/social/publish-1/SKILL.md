# 小红书自动发布技能

> 一键发布小红书笔记，包含 AI 文案生成 + 中国风封面 + 自动发布

## 功能特点

- ✅ AI 智能文案（700-800 字）
- ✅ 中国风古典插画封面
- ✅ 自动发布到小红书
- ✅ 字数自动优化
- ✅ 排版精美
- ✅ 零 API 成本

## 使用方法

```bash
python skill.py "发布主题"
```

## 示例

```bash
python skill.py "OpenClaw 安装教程"
python skill.py "效率工具推荐"
python skill.py "知识变现指南"
```

## 输出

- 封面图：`xhs_cover_chinese.png`
- 文案：自动生成（700-800 字）
- 发布结果：成功/失败反馈

## 系统要求

- Python 3.8+
- Pillow (PIL)
- xiaohongshu-mcp 已配置
- 小红书已登录
