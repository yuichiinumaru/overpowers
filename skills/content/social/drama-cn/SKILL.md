# 短剧剧情构思技能

根据用户提供的一句话灵感，创作跌宕起伏的短剧剧情大纲。

## 何时使用此技能

当用户需要：
- 根据一个创意点子扩展成完整故事
- 快速构思短剧剧情大纲
- 获得脑洞大开的剧情创意
- 为短视频/微电影设计故事线

## 工作流程

1. **理解用户灵感** - 提取核心元素
2. **随机选择爽点组合** - 从 50 个爽点元素中选择 8-12 个
3. **构思剧情结构** - 按照三幕剧结构
4. **创意多样性** - 拒绝老套套路
5. **内容健康** - 无血腥暴力
6. **情感共鸣** - 勾起观众共鸣

## 爽点元素分类

- **励志奋斗类**（10 个）：低谷开局、破釜沉舟、逆袭高潮等
- **真善美类**（10 个）：善有善报、助人助己、回馈社会等
- **情感类**（10 个）：爱情圆满、家人团聚、双向奔赴等
- **爽感类**（20 个）：打脸羞辱、身份反转、极限翻盘等

## 输出格式

```
## 剧情大纲

[剧情内容，约 500 字，融入 8-12 个随机爽点]

## 核心看点
【使用爽点】[列出本次使用的 8-12 个爽点类型]
【逆袭路径】[标注本次使用的逆袭方式]
```
