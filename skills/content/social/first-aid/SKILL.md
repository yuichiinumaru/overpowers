# First Aid - 急救指导

在紧急情况下提供标准化、可立即执行的急救步骤。

## 核心规则

1. **急救电话优先**：中国大陆拨打 `120`，其他地区改为当地急救电话。
2. **先给步骤再补差异**：急救场景不要先长时间追问。
3. **每步一句，编号输出**：让用户能在高压场景快速扫描。
4. **只给标准急救动作**：不用偏方，不给模糊建议。
5. **未知场景也先兜底**：先说“呼叫急救 + 保证现场安全”，再补充具体处理。

## 标准回复结构

```text
🚨 [场景名称] 急救步骤：
1️⃣ [第一步]
2️⃣ [第二步]
3️⃣ [第三步]

⛔ 不要做：
- [常见错误 1]
- [常见错误 2]

📞 如情况严重，立即拨打 120 或当地急救电话
```

## 快速路由

- 晕倒、无反应、无呼吸：先看 CPR / 意识丧失流程
- 烫伤、烧伤：先冷水冲洗，再覆盖，再判断是否送医
- 噎住：先判断能否咳嗽，再决定是否海姆立克
- 大出血、骨折、触电、严重过敏：先确保安全并呼叫急救
- 不在常见清单中的场景：先给兜底动作，再复用 `medical-search`

## 反模式

- 不要拖延。
- 不要推荐偏方、祖传法、酒精/牙膏/酱油等处理。
- 不要忘记先断电、脱险、止血、呼叫急救等底线动作。
- 不要把急救建议说成“应该没事”。
- 不要用英文回复中文用户。
