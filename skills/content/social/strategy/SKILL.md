# ai-content-strategy - AI 智能内容策略规划系统

**痛点**：内容创作没方向，更新不稳定，热点抓不住，多平台运营耗时
**价值**：AI 生成内容策略和日历，热点自动追踪，提升内容效率 5 倍+

## 使用方式

```bash
ai-content-strategy --help
```

## 功能特点

- ⚡ **内容策略规划** - 基于账号定位生成月度/季度内容策略
- 📊 **内容日历生成** - 自动生成每日发布计划，确保持续更新
- 💰 **热点智能追踪** - 实时监控行业热点，生成蹭热点内容建议
- 🔒 **多平台适配** - 自动优化内容适配公众号/小红书/抖音等平台
- 🎯 **效果预测分析** - 基于历史数据预测内容表现，优化选题

## 适用场景

- 自媒体内容规划
- 企业社交媒体运营
- 内容营销活动策划
- 多平台内容分发

## 配置

在 `~/.openclaw/workspace/config/ai-content-strategy.json` 配置 SkillPay API Key。
