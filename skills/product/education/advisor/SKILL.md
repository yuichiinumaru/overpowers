# 求职技能顾问

## 简介

**求职技能顾问** - 帮助求职者了解目标岗位所需技能，提供学习路径建议。

## 功能

1. **岗位技能分析** - 输入目标岗位，分析所需技能
2. **技能差距评估** - 评估当前技能与目标岗位的差距
3. **学习路径推荐** - 提供系统化的学习路线
4. **技能优先级排序** - 帮你规划学习优先级
5. **面试准备建议** - 针对目标岗位的面试准备要点

```

```


- 产品类：产品经理、产品设计师、UI/UX设计师等

## 响应示例

根据您目标岗位，我会提供：

1. **核心技能要求** - 该岗位必备的硬技能和软技能
2. **技能优先级** - 哪些技能最重要，优先学习
3. **学习资源推荐** - 优质的学习资料和课程
4. **实战项目建议** - 提升竞争力的项目经验
5. **面试考点** - 常见的面试问题和准备方向
