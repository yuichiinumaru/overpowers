## 通用原则

### 引导心态
- **保持好奇**：不要急于给答案，先倾听
- **避免说教**：用提问代替建议
- **保持共情**：理解用户的困惑和挣扎
- **允许停顿**：给用户思考时间

### 提问技巧
- 开放式问题（为什么、如何、什么）
- 具体化追问（能举个例子吗？）
- 假设式提问（如果你...会怎样？）
- 对比式提问（A和B，哪个更重要？）

### 进阶判断
- 用户能清晰表达 → 进入下一层
- 用户感到困惑 → 暂时跳过，稍后返回
- 用户表达模糊 → 帮助具体化
- 用户抗拒探索 → 探索原因，降低压力

### 变体开场（根据用户情况调整）

**针对迷茫型用户**：
```
我听到了你内心的声音——那种想要改变却不知从何开始的无力感。这种感觉就像站在十字路口，每条路都看起来一样，但又都不太对劲。

你知道吗？这其实是个好信号。这意味着你开始意识到"现状不是终点"，你内在的某种力量在觉醒了。

今天，我们要做的不是给你一个标准答案，而是帮你找到属于你自己的答案。
```

**针对忙碌型用户**：
```
我知道你现在很忙，可能觉得"哪有时间思考这些人生大事"。但想想看，如果你一直在奔跑，却跑错了方向，跑得越快，离终点越远。

我们花一点时间停下来，重新确认方向，这不是浪费时间，而是在为你未来的奔跑节省时间。

就像GPS导航，在出发前设定好目的地，才能走最优路线。
```

**针对怀疑型用户**：
```
你可能觉得"这些理论听起来很空，能解决什么实际问题"。我完全理解你的怀疑。

但让我问你一个问题：为什么很多人明明很努力，却始终无法突破？不是能力不够，不是运气不好，而是他们思考的层次太低。

就像下棋，如果你只看一步，对手看三步，你一定会输。这个技能，就是帮你看到更多步数的工具。
```

## 各层次详细脚本

### 精神层脚本

**阶段一：唤起使命感**
```
让我们从一个最大的问题开始：你来到这个世界，是为了什么？

我知道这个问题可能有点大，别着急。我们先从小处想——你过去有没有做过什么事情，让你觉得特别有意义，特别有成就感？哪怕是很小的事情。
```

**阶段二：探索利他心**
```
如果你有足够的能力和资源，你最想为别人、为社会做什么？

不一定是改变世界这种宏大的目标，也可以是帮助一小群人，解决某个具体问题。比如，帮助创业者避免走弯路、帮助年轻人找到自信、帮助家庭更好地沟通...

想想看，在你热心的、愿意投入的事情中，哪些是帮助别人的？
```

**阶段三：具象化使命**
```
很好，现在我们试着把你的想法用一句话表达出来。

这句话可以是这样的："我来到这个世界，是为了______，让______能够______。"

比如："我来到这个世界，是通过分享实用的知识，让创业者能够少走弯路，实现自己的梦想。"

你能用这个句式，说出你的人生使命吗？
```

**用户卡住时的应对**：
```
没关系，这个问题确实不容易。很多人可能花几年才能找到答案。

如果你现在感觉不到明确的使命，我们可以先从另一个角度思考——有没有哪群人，是你特别想帮助的？或者哪个社会问题，是你特别想解决的？

先不着急，慢慢来。如果没有答案，我们先跳到下一层，以后再回来。
```

### 身份层脚本

**阶段一：区分角色和身份**
```
现在我有一个重要的问题要问你：你现在的"身份"是什么？

注意，我说的不是"角色"。角色是别人给你的——比如"我是某公司的员工"、"我是孩子的父亲"、"我是XX的学生"。

身份是你自己想成为的——比如"创新者"、"领导者"、"赋能者"、"创造者"。

想象一下，如果所有角色都去掉，你希望用一个什么样的词来定义你自己？
```

**阶段二：未来身份可视化**
```
闭上眼睛，想象5年后的你已经成为了你想成为的人。

那一刻，你站在一个舞台上，主持人介绍你出场，他会怎么介绍你？

"接下来，让我们欢迎——[你的名字]，他/她是一位______。"

那个横线上填的是什么？
```

**阶段三：身份标签共创**
```
很好，我听到了你的想法。现在让我们把身份标签变得更具体一些。

如果用"______者"这样的格式，你觉得哪个词最符合你想成为的人？

比如：创新者、领袖、探索者、赋能者、创造者、连接者、变革者、治愈者...

或者，你希望自己被别人记住的是什么特质？
```

**用户回答角色时的应对**：
```
我理解，"老板"确实是你的角色之一。但我想知道的是——这是别人给你的标签，还是你主动选择的？

如果不考虑任何外在因素，你内心真正想成为的人，是什么样的？

试着去掉"老板"这个标签，重新描述一下自己。
```

### BVR层脚本

**阶段一：信念探索**
```
既然你想成为[用户选择的身份]，那么这个身份应该具备什么样的信念？

换句话说，一个[身份]，他内心深信不疑的观念是什么？

比如，如果是"创新者"，他可能相信"凡事都有改进空间"、"没有一成不变的答案"。

你觉得，要成为[身份]，你必须相信什么？
```

**阶段二：价值观排序**
```
接下来是价值观。价值观是你判断什么更重要、什么次重要的标准。

作为[身份]，在你看来，以下这些哪些更重要？

A. 稳定 vs 创新
B. 长期价值 vs 短期利益
C. 个人成长 vs 即时满足
D. 他人认可 vs 内心满足
E. 探索 vs 确定性

如果你只能选3个最重要的，你会选哪3个？
```

**阶段三：规条设定**
```
最后是行为规条。规条是你自己设定的行为准则，帮助你更好地成为[身份]。

比如，如果是"创新者"，他可能会给自己定下这样的规条：
- "遇到问题，先思考如何优化，而不是如何忍受"
- "每周至少提出一个新想法"
- "不接受'本来就是这样'的理由"

如果让你给自己定3-5条规条，帮助自己成为[身份]，你会写什么？
```

### 能力层脚本

**阶段一：能力清单**
```
现在我们来想想，要成为[身份]，你需要具备哪些核心能力？

不局限于专业技能，也包括软技能、思维能力。

比如，如果是"领袖"，可能需要：
- 领导力
- 沟通能力
- 决策能力
- 战略思维

你觉得，[身份]最需要的能力是什么？列出5-8个。
```

**阶段二：能力评估**
```
看着这份能力清单，我们来评估一下：

你已经具备了哪些能力？
哪些能力正在提升中？
哪些能力还很薄弱，需要重点补足？

如果你只能选择3个能力优先提升，你会选择哪3个？
```

**阶段三：能力获取路径**
``很好，你现在有了需要提升的能力。接下来思考一下：**

- 如何提升这些能力？（课程、实践、导师...）
- 需要多长时间？
- 有没有具体的学习计划？

我们可以从最重要的那个能力开始，制定一个简单的提升计划。
```

### 行为层脚本

**阶段一：具体行动**
```
能力需要通过行动来体现。现在，我们把这些能力转化为具体的行动。

为了提升[核心能力]，你可以在日常生活中做哪些具体的事？

比如，如果要提升"沟通能力"，可能的行动包括：
- 每天主动和一个人进行深度对话
- 每周参加一次社群活动
- 每月做一次公开分享

你能为你的每个核心能力，列出2-3个具体行动吗？
```

**阶段二：SMART化**
``很好，现在让我们把这些行动变得更可执行。我们用SMART原则来优化：**

- **S**pecific（具体）：要做什么？什么时候做？在哪里做？
- **M**easurable（可衡量）：如何知道做到了？
- **A**chievable（可达成）：现实可行吗？
- **R**elevant（相关）：和你的身份、能力匹配吗？
- **T**ime-bound（有时限）：什么时候完成？

比如，把"每天和人沟通"优化为"每周二、四晚上8点，参加线上社群活动，每次至少发言一次"。

你能用这个原则优化你的行动吗？
```

**阶段三：行动计划**
```
现在，让我们制定一个本月的行动计划。

这个月，你承诺要做到的3件最重要的事情是什么？

1. __________（完成时间：______）
2. __________（完成时间：______）
3. __________（完成时间：______）

记住，不要贪多，从小步开始，形成习惯比一次性完成更重要。
```

### 环境层脚本

**阶段一：理想环境**
```
最后，我们来考虑环境因素。

要实现你刚才制定的行动计划，你需要什么样的环境支持？

从这几个方面想想：

- 物理环境：你希望在一个什么样的空间里工作和生活？（安静、整洁、有创意...）
- 社交环境：你希望和什么样的人在一起？（积极向上、志同道合...）
- 信息环境：你希望接触什么样的信息？（学习资源、行业动态...）
- 资源环境：你需要哪些工具、平台、资金？
```

**阶段二：现有资源盘点**
```
现在，看看你身边已经有哪些资源：

- 谁可以成为你的支持者、导师或同伴？
- 你可以接触到哪些学习资源？（书籍、课程、网站）
- 你可以使用哪些工具或平台？
- 你有什么样的资金或时间预算？

把这些资源列出来，思考如何最大化利用它们。
```

**阶段三：障碍清理**
```
除了创造支持性的环境，我们还需要识别并消除障碍。

想想看，有哪些环境因素在阻碍你的成长？

- 哪些人对你有负面影响？
- 哪些环境消耗你的精力？
- 哪些习惯让你浪费时间？

对于这些障碍，你打算如何应对？
```

## 常见困境与应对

### 困境1：用户说"我不知道"
**应对话术**：
```
没关系，不知道很正常。这些问题很少有人天天思考，一下子答不上来很正常。

我们可以换一个角度：你有没有那种"如果可以的话，我希望..."的时刻？哪怕只是闪过一个念头，也值得探索。

或者，我们先跳过这个问题，等你有新的想法再回来。
```

### 困境2：用户说"这太抽象了，没有实际帮助"
**应对话术**：
```
我理解你的感受。但这些看似抽象的问题，其实是最实际的。

想象一下，如果你不知道自己想去哪里，走得再快也可能是南辕北辙。花一点时间想清楚方向，比盲目奔跑更有效。

而且，我们很快就会把这些思考转化为具体的行动计划。这个过程就像盖房子，先有设计图纸，再一砖一瓦地盖。

让我们继续，好吗？
```

### 困境3：用户回答的内容前后矛盾
**应对话术**：
```
我注意到你说A的时候，但刚才又说B。这中间似乎有一些不一致。

这种不一致很正常，因为我们内心可能有不同的声音。我们花一点时间理一理：

如果让你在A和B之间必须选一个，你会选哪个？为什么？

或者，有没有一种可能，A和B其实都反映了你内心某个部分的需求？
```

### 困境4：用户急于跳到行动
**应对话术**：
```
我理解你迫不及待想要开始行动。但在行动之前，我们还有几个关键的问题需要想清楚。

就像开车，踩油门前要先确认方向、检查路况，否则跑得越快可能离目标越远。

花几分钟想清楚这些，会让你的行动更有方向感、更有动力。相信我，这是值得的投资。
```

### 困境5：用户陷入自我怀疑
**应对话术**：
```
我听到了你内心的质疑。这种自我怀疑其实也是一种保护机制，它提醒你认真对待。

但不要让怀疑阻止你前进。你可以把怀疑当成一种声音，但它不应该成为你的决定者。

现在，先不怀疑自己，假设一切皆有可能，你会如何回答这个问题？

等你探索完了，我们再来审视这些怀疑，看看哪些是合理的担忧，哪些只是恐惧的借口。
```

## 收尾话术

### 标准收尾
```
很好！现在我们已经完成了六个层次的探索。让我为你总结一下你的人生蓝图：

【精神层】你的使命：[用户使命]
【身份层】你想成为：[用户身份]
【BVR层】
  - 核心信念：[用户信念]
  - 核心价值观：[用户价值观]
  - 行为规条：[用户规条]
【能力层】你需要提升：[用户能力]
【行为层】本月行动：[用户行动]
【环境层】支持环境：[用户环境]

这是一个非常清晰的框架。记住，这不是静态的，它会随着你的成长而调整。

现在，我想请你做一个承诺：本周你要做的第一件事是什么？

[用户回答后]
太好了！记住这个承诺，并在一周内完成它。

最后，我想对你说：你已经开始了一段全新的旅程。保持耐心，持续行动，你一定会成为你想成为的人。
```

### 激励型收尾
```
今天我们完成了一件很了不起的事——你开始重新定义自己了。

这个过程可能让你有些不舒服，因为你在打破旧的模式，尝试新的可能。但这正是成长的标志。

记住：改变不是一蹴而就的，它需要时间，需要耐心，需要持续的行动。但现在，你已经有了方向，有了蓝图，有了第一步。

从今天开始，每一次选择，每一次行动，都可以问自己：这个选择、这个行动，符合我想成为的那个身份吗？

如果答案是肯定的，那就大胆去做！

期待见证你的成长和蜕变。
```

### 后续跟进引导
```
今天的探索只是开始。接下来，你可以：

1. 把这个蓝图放在显眼的地方，每天看一遍
2. 每周花30分钟，回顾本周行动，调整下周计划
3. 当你遇到选择困境时，回到这个框架，问自己：这个选择符合我的身份和BVR吗？
4. 每月花2小时，重新审视和调整这个蓝图

如果有需要，我们可以继续探索某个层次，或者帮你解决具体的执行问题。

记住，你不需要一个人完成这一切。需要的时候，随时回来。
```
