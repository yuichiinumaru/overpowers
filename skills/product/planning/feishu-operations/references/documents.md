## 创建文档

### 新建空白文档
**方法1：从云文档入口**
1. 点击左侧"云文档"
2. 点击右上角"+"
3. 选择"文档"

**方法2：快捷键**
- Windows: Ctrl + Alt + D
- Mac: Cmd + Option + D

**方法3：从云盘**
1. 进入云盘
2. 右键 → 新建 → 文档

### 使用模板创建
1. 云文档 → 点击"+"
2. 选择"模板"
3. 浏览模板分类
4. 选择模板 → 使用

**常用模板分类**
- 项目管理
- 会议记录
- 培训材料
- 个人笔记
- 制度规范

### 创建文件夹
1. 云文档 → 点击"+"
2. 选择"文件夹"
3. 输入文件夹名称

## 文档编辑

### 基础编辑
**输入文本**
- 直接在文档中输入
- 支持多光标编辑（Ctrl/Cmd + 点击）

**删除内容**
- Backspace：删除前一个字符
- Delete：删除后一个字符
- 选中文本后按删除键：删除选中内容

**复制粘贴**
- 复制：Ctrl/Cmd + C
- 剪切：Ctrl/Cmd + X
- 粘贴：Ctrl/Cmd + V
- 粘贴为纯文本：Ctrl/Cmd + Shift + V

### 标题层级
**设置标题**
- 选中文本 → 点击工具栏H1/H2/H3
- 快捷键：
  - H1: Ctrl/Cmd + Alt + 1
  - H2: Ctrl/Cmd + Alt + 2
  - H3: Ctrl/Cmd + Alt + 3

**标题作用**
- H1：文档大标题，一个文档建议只有一个
- H2：章节标题
- H3：小节标题

### 列表
**无序列表**
- 选中文本 → 点击列表图标
- 快捷键：Ctrl/Cmd + Shift + 8
- 子列表：Tab缩进

**有序列表**
- 选中文本 → 点击编号列表图标
- 快捷键：Ctrl/Cmd + Shift + 7

**任务列表（待办）**
- 插入 → 待办事项
- 标记完成：点击复选框

## 格式设置

### 文字格式
**加粗**
- 选中文本 → Ctrl/Cmd + B
- 工具栏：B图标

**斜体**
- 选中文本 → Ctrl/Cmd + I
- 工具栏：I图标

**删除线**
- 选中文本 → Ctrl/Cmd + Shift + X
- 工具栏：S图标

**下划线**
- 选中文本 → Ctrl/Cmd + U

**文字颜色**
- 选中文本 → 点击颜色图标
- 选择颜色

**背景颜色**
- 选中文本 → 点击高亮图标
- 选择颜色

### 段落格式
**对齐方式**
- 左对齐：Ctrl/Cmd + Shift + L
- 居中：Ctrl/Cmd + Shift + E
- 右对齐：Ctrl/Cmd + Shift + R
- 两端对齐：Ctrl/Cmd + Shift + J

**行间距**
- 选中文本 → 右键 → 行间距
- 选择：1.0 / 1.5 / 2.0

**段落间距**
- 选中文本 → 右键 → 段前段后
- 设置间距值

### 插入内容
**插入图片**
- 点击"+" → 图片
- 选择文件或粘贴URL
- 支持拖拽上传

**插入表格**
- 点击"+" → 表格
- 选择行列数
- 或插入 → 表格

**插入代码块**
- 点击"+" → 代码
- 选择编程语言
- 快捷键：Ctrl/Cmd + Alt + C

**插入引用**
- 选中文本 → 点击引用图标
- 快捷键：Ctrl/Cmd + Shift + 9

**插入分割线**
- 插入 → 分割线
- 快捷键：输入---后按空格

**插入链接**
- 选中文本 → 点击链接图标
- 输入URL
- 快捷键：Ctrl/Cmd + K

**插入文件**
- 点击"+" → 文件
- 选择文件
- 生成可下载链接

**插入视频**
- 点击"+" → 视频
- 输入视频URL（支持YouTube、B站等）

**插入公式**
- 插入 → 公式
- 输入LaTeX公式
- 示例：`E=mc^2`

### 页面设置
**页面布局**
- 右侧面板 → 页面设置
- 页边距
- 页面方向（横向/纵向）
- 页面大小（A4/A3/自定义）

**页眉页脚**
- 右侧面板 → 页眉页脚
- 添加页眉
- 添加页脚
- 插入页码

## 协作功能

### 实时协作
**多人同时编辑**
- 文档右上角显示在线人数
- 实时看到他人的光标位置
- 显示正在编辑的内容

**冲突处理**
- 同一段落多人编辑时提示冲突
- 选择保留哪一版本
- 或合并两个版本

### 评论功能
**添加评论**
- 选中文本 → 右键 → 评论
- 输入评论内容 → 发送

**回复评论**
- 点击评论 → 回复
- 输入回复内容 → 发送

**解决评论**
- 评论右上角勾选
- 标记为已解决

### 修订模式
**开启修订**
- 工具栏 → 修订模式
- 或右键 → 修订模式

**查看修订**
- 修订的内容高亮显示
- 点击查看修订详情
- 接受或拒绝修订

### 历史版本
**查看历史版本**
- 文件 → 历史版本
- 选择时间点
- 预览版本内容

**恢复历史版本**
- 历史版本 → 恢复此版本
- 创建新版本或覆盖当前版本

**版本对比**
- 选择两个版本
- 点击对比
- 查看差异

### 文档分享
**生成分享链接**
- 右上角"分享"
- 选择"获取链接"
- 设置权限 → 复制链接

**分享权限**
- 查看权限：仅查看
- 编辑权限：可编辑
- 评论权限：可评论

**分享到群聊**
- 分享 → 发送到群聊
- 选择群聊 → 发送

**邀请协作**
- 分享 → 邀请协作
- 输入邮箱/手机号
- 设置权限 → 发送邀请

## 最佳实践

### 文档结构
1. **标题清晰**：使用H1/H2/H3构建结构
2. **段落分明**：一个段落表达一个观点
3. **列表归纳**：使用列表组织要点
4. **格式统一**：保持格式一致性

### 协作编辑
1. **分工明确**：不同人编辑不同章节
2. **使用评论**：讨论内容而非直接修改
3. **定期保存**：重要修改后及时保存
4. **版本管理**：关键节点创建版本

### 权限管理
1. **最小权限**：按需分配，不过度授权
2. **定期审核**：检查权限，及时回收
3. **使用群组**：批量管理权限

## 常见问题

### Q1: 文档内容丢失怎么办？
A: 查看历史版本，恢复到之前的版本

### Q2: 如何快速格式化文档？
A: 使用模板或右键"清除格式"后重新设置

### Q3: 文档可以离线编辑吗？
A: 可以，PC客户端支持离线编辑，联网后自动同步

### Q4: 如何合并两个文档？
A: 全选一个文档内容 → 复制 → 粘贴到另一个文档

### Q5: 文档容量有限制吗？
A: 单个文档最大100MB，建议拆分大型文档
