## 文档快捷键

### 文本编辑
**光标移动**
- 左移一个字符：←
- 右移一个字符：→
- 上移一行：↑
- 下移一行：↓
- 移到行首：Home / Cmd + ←
- 移到行尾：End / Cmd + →
- 移到文档开头：Ctrl/Cmd + Home
- 移到文档结尾：Ctrl/Cmd + End

**选中文本**
- 选择一个字符：Shift + ←/→
- 选择一行：Shift + ↑/↓
- 选择到行首：Shift + Home
- 选择到行尾：Shift + End
- 全选：Ctrl/Cmd + A

**文本格式**
- 加粗：Ctrl/Cmd + B
- 斜体：Ctrl/Cmd + I
- 下划线：Ctrl/Cmd + U
- 删除线：Ctrl/Cmd + Shift + X

### 标题和列表

- 一级标题：Ctrl/Cmd + Alt + 1
- 二级标题：Ctrl/Cmd + Alt + 2
- 三级标题：Ctrl/Cmd + Alt + 3

**列表**
- 无序列表：Ctrl/Cmd + Shift + 8
- 有序列表：Ctrl/Cmd + Shift + 7
- 待办事项：Ctrl/Cmd + Shift + 9

### 插入内容

## 常见问题

### Q1: 快捷键不生效怎么办？
A: 检查输入法，确保英文输入状态，检查快捷键冲突

### Q2: 如何自定义快捷键？
A: 设置 → 快捷键，修改默认快捷键

### Q3: 如何快速查找某个功能？
A: 使用 Ctrl/Cmd + K 全局搜索，输入功能名称

### Q4: 快捷键太多记不住怎么办？
A: 先记住常用的10个，逐步扩展，使用过程中加深记忆

### Q5: 如何查看所有快捷键？
A: 设置 → 快捷键，查看完整列表
