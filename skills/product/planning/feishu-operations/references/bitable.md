## 创建表格

### 新建空白表格
**方法1：从云文档**
1. 点击左侧"云文档"
2. 点击"+"
3. 选择"多维表格"

**方法2：从云盘**
1. 进入云盘
2. 右键 → 新建 → 多维表格

**方法3：快捷键**
- Windows: Ctrl + Alt + B
- Mac: Cmd + Option + B

### 使用模板创建
1. 云文档 → 点击"+"
2. 选择"模板"
3. 浏览"多维表格"分类
4. 选择模板 → 使用

**常用模板**
- 项目任务跟踪
- 客户信息管理
- 资产清单
- 调查问卷
- 报名表

### 表格结构
**核心概念**
- **表格（Table）**：数据表的集合，一个表格应用包含多个数据表
- **视图（View）**：同一数据的不同展示方式
- **字段（Field）**：数据列，定义数据类型
- **记录（Record）**：数据行，一条完整的数据

## 字段类型

### 基础字段

**文本**
- 用途：存储文本信息
- 示例：姓名、描述、备注
- 设置：最大字符数限制

**数字**
- 用途：存储数值
- 示例：年龄、数量、金额
- 格式：整数、小数、百分比
- 设置：小数位数、千分位

**选项**
- 用途：从预设选项中选择
- 示例：状态（待办/进行中/完成）、优先级（高/中/低）
- 设置：单选或多选、颜色标识

**日期**
- 用途：存储日期时间
- 示例：创建时间、截止日期、生日
- 格式：日期、时间、日期时间
- 设置：日期格式、时区

**人员**
- 用途：关联用户
- 示例：负责人、参与人
- 选择：从通讯录选择
- 可选：单选或多选

**文件**
- 用途：关联文件
- 示例：附件、截图、证明材料
- 上传：从云盘或本地选择
- 类型：图片、文档、其他

**复选框**
- 用途：布尔值（是/否）
- 示例：是否完成、是否确认
- 显示：勾选框或开关

**电话**
- 用途：存储电话号码
- 示例：手机号、座机
- 格式：自动格式化
- 拨号：直接点击拨打

**邮箱**
- 用途：存储邮箱地址
- 示例：工作邮箱、个人邮箱
- 格式：邮箱验证
- 发送：点击直接发送邮件

**网址**
- 用途：存储链接
- 示例：网站地址、文档链接
- 跳转：点击打开链接
- 预览：显示链接预览

### 高级字段

**关联**
- 用途：关联其他表的记录
- 示例：订单关联客户、任务关联项目
- 设置：关联表、关联字段
- 反向关联：自动创建

**查找引用**
- 用途：从关联表引用字段值
- 示例：从客户表引用客户名称
- 依赖：必须先有关联字段

**公式**
- 用途：自动计算结果
- 示例：总价 = 单价 × 数量
- 函数：支持数学、文本、日期、逻辑函数

**自动编号**
- 用途：自动生成编号
- 示例：订单号、申请编号
- 格式：前缀+数字

**创建时间**
- 用途：记录创建时间
- 自动填充：创建记录时自动记录

**创建人**
- 用途：记录创建者
- 自动填充：创建记录时自动记录

**修改时间**
- 用途：记录最后修改时间
- 自动填充：修改记录时自动更新

**修改人**
- 用途：记录最后修改者
- 自动填充：修改记录时自动更新

## 数据录入

### 添加记录
**方法1：点击添加按钮**
- 点击底部"添加记录"按钮
- 填写字段信息

**方法2：快捷键**
- Tab：移动到下一字段
- Enter：完成当前记录，开始新记录

**方法3：复制记录**
- 右键记录 → 复制记录
- 修改复制的记录

### 编辑记录
**编辑单个字段**
- 直接点击字段
- 修改内容
- 自动保存

**批量编辑**
- 选择多条记录（Ctrl/Cmd + 点击）
- 右键 → 批量编辑
- 选择字段 → 输入值

**复制粘贴**
- 复制Excel数据
- 粘贴到多维表格
- 自动识别字段

### 删除记录
**删除单条记录**
- 右键记录 → 删除记录

**删除多条记录**
- 选择多条记录
- 右键 → 删除记录

**恢复删除**
- 历史记录 → 恢复
- 限制：30天内可恢复

## 视图管理

### 创建视图
**表格视图**
- 默认视图
- 类似Excel表格形式
- 适合数据录入和查看

**看板视图**
- 按选项分组
- 卡片式展示
- 适合任务管理

**日历视图**
- 按日期字段展示
- 显示在日历上
- 适合日程安排

**甘特图视图**
- 显示项目时间线
- 依赖关系可视化
- 适合项目管理

**表单视图**
- 填写式界面
- 用于数据收集
- 可生成分享链接

**画廊视图**
- 图片卡片展示
- 适合图片库、产品展示

### 视图设置

**筛选**
- 设置筛选条件
- 支持多条件组合
- 保存筛选方案

**排序**
- 按字段排序
- 支持多字段排序
- 升序/降序

**分组**
- 按字段分组
- 可折叠/展开
- 统计每组数量

**隐藏字段**
- 暂时隐藏不需要的字段
- 保持界面整洁

**冻结列**
- 冻结重要列
- 滚动时保持可见

**调整列宽**
- 拖拽列边界
- 双击自动调整

## 数据处理

### 查找数据
**快速查找**
- Ctrl/Cmd + F
- 输入关键词
- 高亮匹配项

**高级筛选**
- 视图设置 → 筛选
- 添加条件
- 支持逻辑运算（与/或）

**筛选条件类型**
- 等于/不等于
- 包含/不包含

**单字段排序**
- 点击字段标题
- 切换升序/降序

**多字段排序**
- 视图设置 → 排序
- 添加排序字段
- 设置优先级

### 数据统计
**字段统计**
- 数字字段：求和、平均、最大、最小、计数
- 选项字段：统计各选项数量
- 日期字段：最早、最晚

**分组统计**
- 按字段分组
- 显示每组统计
- 支持子分组

### 数据导入导出
**导入数据**
- 文件 → 导入
- 支持格式：Excel、CSV

**导出数据**
- 文件 → 导出

**复制到Excel**
- 选择记录 → 右键
- 复制为Excel格式

## 高级功能

### 公式计算

**数学函数**
```
SUM([字段名])：求和
AVG([字段名])：平均值
MAX([字段名])：最大值
MIN([字段名])：最小值
COUNT([字段名])：计数
```

**文本函数**
```
CONCAT([字段1], [字段2])：连接文本
LEFT([字段], 5)：左侧5个字符
RIGHT([字段], 5)：右侧5个字符
LEN([字段])：字符长度
FIND("文本", [字段])：查找位置
```

**日期函数**
```
TODAY()：今天
NOW()：当前时间
DAYS([日期1], [日期2])：日期差
DATE([年], [月], [日])：创建日期
```

**逻辑函数**
```
IF([条件], [真值], [假值])：条件判断
AND([条件1], [条件2])：逻辑与
OR([条件1], [条件2])：逻辑或
NOT([条件])：逻辑非
```

**示例公式**
```
计算总价：[单价] * [数量]
判断状态：IF([进度]>=100, "完成", "进行中")
计算天数：DAYS([截止日期], TODAY())
```

## 常见问题

### Q1: 表格记录有限制吗？
A: 每个表最多5万条记录，总大小10GB

### Q2: 如何批量修改数据？
A: 选择多条记录 → 右键 → 批量编辑

### Q3: 公式支持嵌套吗？
A: 支持，但建议不要过于复杂，影响性能

### Q4: 可以导入Excel数据吗？
A: 可以，文件 → 导入 → 选择Excel文件

### Q5: 如何保护敏感数据？
A: 设置字段权限，限制查看和编辑
