## 账号问题

### Q1: 忘记密码怎么办？
**解决方案**：
1. 登录页面点击"忘记密码"
2. 输入注册手机号或邮箱
3. 验证身份后重置密码
4. 设置新密码并登录

### Q2: 如何修改手机号？
**操作步骤**：
1. 点击左上角头像
2. 选择"设置"
3. 进入"账号与安全"
4. 点击"手机号"
5. 验证原手机号
6. 输入新手机号
7. 验证新手机号
8. 完成修改

### Q3: 可以同时登录多个设备吗？
**答案**：可以
- 支持PC、网页、移动端同时在线
- 消息实时同步
- 建议每台设备都启用安全锁

### Q4: 如何退出登录？
**操作步骤**：
- PC端：左上角头像 → 退出登录
- 移动端：我的 → 设置 → 退出登录
- 网页版：右上角头像 → 退出登录

### Q5: 账号被锁定怎么办？
**解决方案**：
1. 联系企业IT管理员
2. 或联系飞书客服
3. 验证身份后解锁
4. 检查是否有异常登录

## 消息问题

### Q1: 消息发送失败怎么办？
**排查步骤**：
1. 检查网络连接
2. 重新发送消息
3. 如频繁失败，重启应用
4. 联系客服检查账号状态

### Q2: 如何恢复撤回的消息？
**答案**：无法恢复
- 撤回后消息即删除
- 建议撤回前先复制内容
- 或使用修订模式

### Q3: 群聊消息太多，找不到重要信息？
**解决方案**：
1. 使用消息搜索功能
2. 标记重要消息（右键 → 标记）
3. 使用@提醒重要信息
4. 定期清理无用消息

### Q4: 如何查看历史消息？
**操作方法**：
- 向上滚动聊天记录
- 使用搜索功能查找特定消息
- 按日期筛选

### Q5: 消息通知太频繁？
**解决方案**：
1. 设置 → 通知 → 调整通知频率
2. 对不重要的群聊开启免打扰
3. 设置工作时段，非工作时间不通知

## 文档问题

### Q1: 文档内容丢失怎么办？
**解决方案**：
1. 查看历史版本
2. 恢复到之前的版本
3. 检查是否误删
4. 联系文档所有者

### Q2: 如何快速格式化文档？
**解决方案**：
1. 使用文档模板
2. 使用格式刷复制格式
3. 选中内容 → 右键 → 清除格式
4. 重新设置格式

### Q3: 文档可以离线编辑吗？
**答案**：可以
- PC客户端支持离线编辑
- 联网后自动同步
- 避免冲突，多人协作时在线编辑

### Q4: 如何合并两个文档？
**操作方法**：
1. 打开文档A
2. 全选内容（Ctrl/Cmd + A）
3. 复制（Ctrl/Cmd + C）
4. 打开文档B
5. 粘贴到末尾（Ctrl/Cmd + V）

### Q5: 文档容量有限制吗？
**限制说明**：
- 单个文档最大100MB
- 建议拆分大型文档
- 使用文件夹组织

## 会议问题

### Q1: 视频会议卡顿怎么办？
**解决方案**：
1. 检查网络连接
2. 关闭摄像头
3. 关闭不必要的应用
4. 降低视频质量

### Q2: 如何避免会议时间冲突？
**解决方案**：
1. 查看参会人日历
2. 使用"查找时间"功能
3. 提前预约会议
4. 使用群投票确定时间

### Q3: 会议录制文件保存在哪里？
**查找路径**：
1. 打开云盘
2. 进入"会议录制"文件夹
3. 按日期查找
4. 或在会议文档中查看

### Q4: 无法听到会议声音？
**排查步骤**：
1. 检查扬声器设置
2. 检查麦克风设置
3. 检查系统音量
4. 重新加入会议

### Q5: 飞书妙记支持多语言吗？
**答案**：支持
- 支持中英文
- 自动识别语言
- 混合语言也能识别

## 权限问题

### Q1: 为什么看不到文档？
**可能原因**：
1. 没有查看权限
2. 文档被删除
3. 权限过期
4. 网络问题

**解决方案**：
1. 联系文档所有者获取权限
2. 检查是否被移除协作者
3. 刷新页面或重启应用

### Q2: 如何快速给多人授权？
**解决方案**：
1. 创建群组
2. 给群组授权
3. 将成员加入群组
4. 自动获得权限

### Q3: 权限设置后多久生效？
**答案**：实时生效
- 大部分情况立即生效
- 部分情况可能需要刷新

### Q4: 所有者可以删除文档吗？
**答案**：可以
- 所有者拥有最高权限
- 管理权限也可以删除
- 删除后可30天内恢复

### Q5: 如何查看谁有文档权限？
**操作方法**：
1. 点击右上角"分享"
2. 进入"权限管理"
3. 查看完整权限列表

## 性能问题

### Q1: 飞书运行缓慢怎么办？
**解决方案**：
1. 清理缓存
2. 关闭不必要的标签页
3. 重启应用
4. 更新到最新版本

### Q2: 如何清理缓存？
**操作步骤**：
1. 设置 → 常规
2. 点击"清理缓存"
3. 确认清理
4. 重启应用

### Q3: 消息同步延迟怎么办？
**解决方案**：
1. 检查网络连接
2. 手动刷新（下拉刷新）
3. 退出重新登录
4. 检查服务器状态

### Q4: 文档保存失败？
**排查步骤**：
1. 检查网络连接
2. 检查权限设置
3. 检查文档大小
4. 联系客服

### Q5: 如何提高飞书运行速度？
**优化建议**：
1. 关闭自动下载
2. 减少同时打开的文档数
3. 定期清理缓存
4. 更新到最新版本

## 其他问题

### Q1: 飞书免费吗？
**答案**：
- 个人版：完全免费
- 企业版：有免费额度
- 超出额度需要付费

### Q2: 如何联系客服？
**联系方式**：
1. 设置 → 帮助与反馈
2. 提交问题
3. 或访问帮助中心
4. 或拨打客服电话

### Q3: 如何提交功能建议？
**操作方法**：
1. 设置 → 帮助与反馈
2. 功能建议
3. 描述需求
4. 提交反馈

### Q4: 飞书数据安全吗？
**安全措施**：
- 数据加密传输
- 数据加密存储
- 定期安全审计
- 符合安全标准

### Q5: 如何导出数据？
**导出方式**：
1. 文档：文件 → 导出
2. 表格：文件 → 导出
3. 消息：暂不支持批量导出
4. 联系管理员申请数据导出
