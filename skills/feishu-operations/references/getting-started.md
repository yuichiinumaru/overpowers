## 界面概览

### 飞书客户端布局
飞书提供多端支持：PC客户端、网页版、移动端（iOS/Android）

**PC客户端主要区域**：
1. **左侧导航栏**
   - 消息：单聊、群聊、机器人消息
   - 通讯录：联系人、组织架构
   - 日历：日程、会议
   - 云文档：文档、表格、知识库
   - 飞书妙记：会议记录
   - 云盘：文件管理
   - 工作台：应用入口

2. **中间主内容区**
   - 根据左侧选择显示对应内容
   - 消息列表、文档编辑、日历视图等

3. **右侧面板（可展开）**
   - 消息详情
   - 文档目录
   - 日历详情
   - 通讯录详情

### 飞书网页版
访问 https://www.feishu.cn，登录后界面与客户端类似

### 移动端App
- 底部导航栏：消息、工作台、通讯录、我
- 支持消息、文档、日历、会议等核心功能

## 核心概念

### 账号体系
- **个人账号**：使用手机号或邮箱注册
- **企业账号**：加入企业后获得企业身份
- **开放平台账号**：开发者账号，用于API调用

### 组织架构
- **企业**：最高层级，一个企业包含多个部门
- **部门**：企业的子单位，可包含子部门
- **成员**：企业的个人用户，属于一个或多个部门

### 文件体系
- **云盘**：个人和企业的文件存储空间
- **云文档**：在线协作文档
- **知识库**：组织化的文档集合
- **多维表格**：类似数据库的在线表格

### 权限体系
- **查看权限**：仅可查看内容
- **编辑权限**：可编辑内容
- **管理权限**：可管理权限和设置
- **所有者权限**：完全控制权

## 账号与登录

### 注册账号
1. 访问 https://www.feishu.cn
2. 点击"注册"
3. 输入手机号或邮箱
4. 验证手机号或邮箱
5. 设置密码
6. 完成注册

### 登录方式
1. **手机号/邮箱+密码**
2. **手机验证码登录**
3. **第三方登录**（微信、钉钉等）
4. **SSO登录**（企业用户）

### 加入企业
1. 接收企业邀请邮件或链接
2. 点击链接接受邀请
3. 填写个人信息
4. 完成加入

### 切换身份
如同时加入多个企业，可在个人设置中切换：
- 点击左上角头像 → 设置 → 企业设置 → 切换企业

## 基础设置

### 个人资料设置
1. 点击左上角头像
2. 选择"设置"
3. 可设置：
   - 姓名、英文名
   - 头像
   - 职位、部门
   - 手机号、邮箱
   - 个人简介

### 通知设置
路径：设置 → 通知
- **消息通知**：桌面通知、声音提醒
- **日历提醒**：会议提醒时间设置
- **邮件通知**：重要事项邮件提醒

### 界面设置
路径：设置 → 界面
- 主题：明亮/暗黑
- 语言：中文/英文
- 字体大小
- 侧边栏显示设置

### 隐私设置
路径：设置 → 隐私
- 在线状态显示
- 个人信息可见范围
- 消息已读回执

## 快速上手

### 第一次使用飞书

#### 1. 发送第一条消息
- 点击左侧"消息" → 点击"+"按钮 → 选择联系人 → 输入消息 → 发送

#### 2. 创建第一个文档
- 点击左侧"云文档" → 点击"+" → 选择"文档" → 输入标题 → 开始编辑

#### 3. 创建日历事件
- 点击左侧"日历" → 双击日期 → 输入事件信息 → 保存

#### 4. 加入群聊
- 点击左侧"消息" → 点击"+" → 选择"创建群聊" → 邀请成员 → 创建

### 必知技巧

#### 快捷键（Windows/Mac）
- **发送消息**：Ctrl/Cmd + Enter
- **新建文档**：Ctrl/Cmd + Alt + D
- **新建群聊**：Ctrl/Cmd + Alt + C
- **搜索**：Ctrl/Cmd + K
- **截图**：Ctrl/Cmd + Alt + A

#### 搜索功能
按 Ctrl/Cmd + K 打开全局搜索，可搜索：
- 联系人
- 消息记录
- 文档内容
- 日程
- 文件

#### 文件拖拽
- 直接将文件拖入消息框发送
- 拖入文档编辑区插入
- 拖入云盘上传

#### 右键菜单
在聊天、文档、文件上右键可快速操作：
- 转发消息
- 复制链接
- 下载文件
- 分享文档

## 常见问题

### Q1: 忘记密码怎么办？
A: 在登录页面点击"忘记密码"，通过手机号或邮箱重置

### Q2: 如何修改手机号？
A: 设置 → 账号与安全 → 手机号 → 修改

### Q3: 可以同时登录多个设备吗？
A: 可以，支持PC、网页、移动端同时登录

### Q4: 如何退出登录？
A: 点击左上角头像 → 退出登录

### Q5: 飞书免费吗？
A: 个人版免费，企业版有免费额度，超出后付费
