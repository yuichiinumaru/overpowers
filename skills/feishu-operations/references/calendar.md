## 日历功能

### 日历视图
**日视图**
- 查看当天所有日程
- 时间轴显示
- 可拖拽调整时间

**周视图**
- 查看一周日程
- 显示工作日和周末
- 适合周计划

**月视图**
- 查看整月日程
- 显示重要日期
- 适合月度规划

**列表视图**
- 按时间顺序列出日程
- 显示详情
- 适合日程回顾

### 创建日程
**快速创建**
1. 双击日历日期
2. 输入日程标题
3. 设置时间
4. 保存

**详细创建**
1. 点击"新建日程"
2. 填写详细信息：
   - 标题
   - 时间（开始/结束）
   - 重复（每天/每周/每月/自定义）
   - 地点
   - 描述
   - 提醒
   - 邀请人
   - 会议链接

**从消息创建**
- 右键消息 → 创建日程
- 自动提取时间信息

### 重复日程
**重复类型**
- 每天重复
- 每周重复
- 每月重复
- 每年重复
- 自定义重复

**自定义重复规则**
- 设置重复周期
- 设置结束条件
  - 永不结束
  - 重复N次后结束
  - 结束于某日期

**编辑重复日程**
- 编辑单个事件
- 编辑所有事件
- 编辑此事件及后续事件

### 日程邀请
**邀请参会人**
1. 创建日程 → 添加参与人
2. 输入邮箱/姓名
3. 从通讯录选择
4. 发送邀请

**邀请响应**
- 接受：确认参加
- 拒绝：拒绝参加
- 暂定：待定
- 可选择是否添加到日历

**提醒设置**
- 提前15分钟
- 提前1小时
- 提前1天
- 自定义时间

### 日历共享
**共享日历**
1. 日历设置 → 共享
2. 输入邮箱
3. 设置权限（查看/编辑）
4. 发送邀请

**订阅日历**
- 订阅同事日历
- 查看团队日程
- 避免冲突

### 日历分组
**创建日历**
- 左侧 → 新建日历
- 命名
- 设置颜色

**日历管理**
- 显示/隐藏日历
- 删除日历
- 共享日历

## 快捷键

### 日历快捷键
- 新建日程：Ctrl/Cmd + Alt + T
- 今天：Ctrl/Cmd + T
- 前一天：Ctrl/Cmd + [
- 后一天：Ctrl/Cmd + ]

### 会议快捷键
- 开启/关闭麦克风：M
- 开启/关闭摄像头：V
- 共享屏幕：S
- 静音所有人：Alt + M

## 常见问题

### Q1: 如何避免会议时间冲突？
A: 查看参会人日历，选择空闲时间，使用"查找时间"功能

### Q2: 会议录制文件保存在哪里？
A: 自动保存到云盘，可在会议文档或云盘中找到

### Q3: 如何重复预约会议？
A: 创建日程时设置重复规则，自动生成重复日程

### Q4: 飞书妙记支持多语言吗？
A: 支持中英文，自动识别语言

### Q5: 如何导出会议记录？
A: 飞书妙记 → 导出 → 选择格式（PDF/Word/TXT）
