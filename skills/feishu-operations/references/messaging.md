## 单聊

### 发起单聊
**方法1：从通讯录**
1. 点击左侧"通讯录"
2. 找到联系人
3. 点击姓名发起聊天

**方法2：快速搜索**
1. 按 Ctrl/Cmd + K 打开搜索
2. 输入联系人姓名
3. 选择后进入聊天

**方法3：消息列表**
1. 点击左侧"消息"
2. 点击"+"按钮
3. 选择"发起聊天"
4. 选择联系人

### 消息发送
**文本消息**
- 在输入框直接输入
- 支持表情符号（Win: Win+. / Mac: Ctrl+Cmd+Space）
- 支持换行（Shift+Enter）

**富文本消息**
- **加粗**：选中文本 → 点击B图标
- *斜体*：选中文本 → 点击I图标
- ~~删除线~~：选中文本 → 点击S图标
- 引用：选中文本 → 点击引用图标
- 代码：选中文本 → 点击代码图标
- 链接：选中文本 → 点击链接图标 → 输入URL

**@提醒**
- @某人：输入@ → 选择联系人
- @所有人：输入@所有人（群聊可用，需群主权限）
- @提醒后，对方会收到通知

### 消息编辑
**编辑已发送消息**
- 右键消息 → 编辑 → 修改 → 保存

**撤回消息**
- 右键消息 → 撤回
- 限制：发送后2分钟内可撤回

**转发消息**
- 右键消息 → 转发 → 选择目标聊天

**引用回复**
- 右键消息 → 引用回复 → 输入回复内容

## 群聊

### 创建群聊
**方法1：从联系人**
1. 通讯录 → 选择多个联系人
2. 点击"创建群聊"
3. 设置群名称、群公告

**方法2：从消息列表**
1. 消息 → 点击"+"
2. 选择"创建群聊"
3. 邀请成员 → 创建

**方法3：从现有聊天**
1. 聊天窗口右上角点击"..."
2. 选择"转换为群聊"
3. 邀请成员

### 群聊设置

**群基本信息**
- 群名称：点击群名修改
- 群头像：点击头像上传
- 群公告：设置 → 群公告 → 编辑
- 群描述：设置 → 群描述

**群成员管理**
- 添加成员：群设置 → 成员 → 添加
- 移除成员：群设置 → 成员 → 选择成员 → 移除
- 设置管理员：群设置 → 成员 → 设置管理员
- 转让群主：群设置 → 成员 → 转让群主

**群聊权限**
- 所有人可邀请：群设置 → 权限 → 开启
- 仅管理员可邀请：群设置 → 权限 → 关闭
- 禁止@所有人：群设置 → 权限 → 关闭

### 群消息管理
**消息免打扰**
- 群设置 → 消息通知 → 开启免打扰
- 效果：不接收推送通知

**消息置顶**
- 右键群聊 → 置顶聊天
- 置顶群聊显示在消息列表顶部

**消息归档**
- 右键群聊 → 归档
- 归档后不显示在消息列表

**群公告提醒**
- 设置 → 群公告 → 发布 → 强制提醒
- 所有成员会收到通知

## 消息类型

### 文本消息
- 最常用的消息类型
- 支持富文本格式
- 支持表情和链接

### 图片消息
**发送方式**
- 直接拖拽图片到聊天框
- 点击"+" → 图片 → 选择文件
- 粘贴剪贴板图片（Ctrl/Cmd + V）

**图片编辑**
- 添加标注
- 涂鸦
- 裁剪
- 添加文字

### 文件消息
**发送文件**
- 拖拽文件到聊天框
- 点击"+" → 文件 → 选择文件
- 云盘文件：点击"+" → 云文件 → 选择

**文件大小限制**
- 普通用户：最大10GB
- 企业用户：根据企业套餐

**文件预览**
- 支持在线预览常见格式
- 图片、文档、视频、音频

### 语音消息
**录制语音**
- 点击"+" → 录音 → 按住录音
- 松开发送
- 再次点击取消

**语音转文字**
- 点击语音消息 → 转文字
- 自动转换为文本

### 视频消息
**录制视频**
- 点击"+" → 视频 → 录制
- 最长15分钟

**发送本地视频**
- 点击"+" → 文件 → 选择视频文件

### 链接消息
**发送链接**
- 直接粘贴URL
- 自动生成链接卡片预览

**链接卡片内容**
- 网页标题
- 网页描述
- 网页缩略图

### 位置消息
**发送位置**
- 点击"+" → 位置
- 选择当前定位或搜索地点
- 发送

**位置信息**
- 地点名称
- 地图位置
- 可直接导航

### 表情消息
**发送表情**
- 点击表情图标
- 选择表情包
- 发送

**自定义表情**
- 上传自定义表情包
- 管理表情收藏

## 消息管理

### 消息搜索
**全局搜索**
- Ctrl/Cmd + K 打开搜索
- 输入关键词
- 选择"消息"标签

**聊天内搜索**
- 聊天窗口右上角搜索图标
- 输入关键词
- 查看匹配消息

**高级搜索**
- 支持日期范围
- 支持发件人筛选
- 支持消息类型筛选

### 消息标记
**标记重要消息**
- 右键消息 → 标记
- 消息右上角显示星标

**查看标记消息**
- 聊天窗口 → 右上角星标图标
- 查看所有标记消息

### 消息导出
**导出聊天记录**
- 聊天窗口 → 右上角"..."
- 导出聊天记录
- 选择导出格式（TXT/HTML）

### 消息删除
**删除单条消息**
- 右键消息 → 删除
- 仅删除自己可见的消息

**删除所有消息**
- 聊天窗口 → 右上角"..."
- 清空聊天记录

## 高级功能

### 消息已读回执
**开启已读回执**
- 设置 → 隐私 → 开启"显示已读"

**查看已读状态**
- 单聊：消息下方显示"已读"
- 群聊：查看已读人数

### 消息定时发送
**设置定时发送**
- 输入消息后 → 点击发送图标旁"..."
- 选择"定时发送"
- 设置时间 → 确定

### 消息加密
**设置端到端加密**
- 聊天窗口 → 右上角"..."
- 开启"端到端加密"
- 仅双方可见，服务器无法查看

### 消息翻译
**翻译消息**
- 右键消息 → 翻译
- 选择目标语言

### 消息转发到邮箱
**设置邮件转发**
- 设置 → 消息 → 邮件转发
- 绑定邮箱
- 选择转发的聊天

## 快捷键

### Windows
- 发送消息：Ctrl + Enter
- 换行：Shift + Enter
- 搜索：Ctrl + K
- 截图：Ctrl + Alt + A
- 新建聊天：Ctrl + Alt + C

### Mac
- 发送消息：Cmd + Enter
- 换行：Shift + Enter
- 搜索：Cmd + K
- 截图：Cmd + Ctrl + A
- 新建聊天：Cmd + Option + C

## 常见问题

### Q1: 消息发送失败怎么办？
A: 检查网络连接，重新发送；如频繁失败，联系客服

### Q2: 如何恢复撤回的消息？
A: 撤回后无法恢复，建议复制后再撤回

### Q3: 群聊最多支持多少人？
A: 普通群聊500人，超大群聊支持1000人（企业版）

### Q4: 如何查看历史消息？
A: 向上滚动聊天记录，或使用搜索功能

### Q5: 消息可以保存多久？
A: 永久保存，除非手动删除
