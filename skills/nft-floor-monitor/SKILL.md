# 🖼️ NFT Floor Monitor - 地板价监控

**多平台比价，稀有度分析** - 把握最佳买入时机。

---

## 📋 使用方式

```bash
# 监控系列
nft-floor-monitor BAYC
nft-floor-monitor Azuki
```

## 💰 收费

**¥3/次** - 低价买入，可能赚回 100 倍。

---

## 🔍 分析维度

| 维度 | 说明 |
|------|------|
| 地板价 | 当前最低售价 |
| 24h 成交量 | 交易活跃度 |
| 24h 涨跌 | 价格趋势 |
| 7 天走势 | 历史价格 |
| 交易信号 | 买入/观望/卖出 |

---

## 💡 使用建议

1. 地板价低于 7 天平均可买入
2. 成交量放大时关注
3. 稀有度高的优先
4. 设置价格提醒
