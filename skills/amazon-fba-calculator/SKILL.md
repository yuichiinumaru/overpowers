# amazon-fba-calculator - 亚马逊 FBA 利润精准计算器

**痛点**：FBA 费用复杂难算，隐藏成本多，选品时利润预估不准，导致亏本销售
**价值**：一键计算所有费用，精准预估净利润，避免 90% 的亏本选品

## 使用方式

```bash
amazon-fba-calculator --help
```

## 功能特点

- ⚡ **全费用自动核算** - FBA 配送费、仓储费、佣金、广告费一键计算
- 📊 **ROI 精准分析** - 多维度利润率分析，支持不同定价策略对比
- 💰 **隐藏成本识别** - 退货、弃置、长期仓储等隐性成本自动计入
- 🔒 **竞品利润对比** - 分析竞品定价策略，找到最优价格区间
- 🎯 **选品决策支持** - 基于利润数据给出选品建议和风险预警

## 适用场景

- 亚马逊 FBA 选品利润预估
- 产品定价策略优化
- 跨境电商成本核算
- 店铺盈利分析

## 配置

在 `~/.openclaw/workspace/config/amazon-fba-calculator.json` 配置 SkillPay API Key。
