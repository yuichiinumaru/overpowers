# 链上项目安全审计 - 项目风险评估

## 概述
基于链上数据和项目信息，对加密货币项目进行全方位安全审计和风险评估。帮助投资者识别潜在风险。

## 定价
- **按次收费：** ¥9/次
- **包月订阅：** ¥199/月（无限次）

## 功能特性
- 🔍 合约代码分析
- 👥 团队背景调查
- 💰 代币经济模型评估
- 📊 链上数据分析
- ⚠️ 风险评分（0-100）

## 使用方式
```bash
# 基础审计
/onchain-audit "项目地址或名称"

# 深度审计
/onchain-audit "0x..." --deep

# 对比分析
/onchain-audit "项目 A" --compare "项目 B"

# 批量审计
/onchain-audit "项目列表" --batch
```

## 输出格式
```markdown
## 🔒 链上项目安全审计报告

### 📊 综合风险评分：75/100（中风险）

### ✅ 安全项
- [✓] 合约已开源
- [✓] LP 已锁定（180 天）
- [✓] 团队实名

### ⚠️ 风险项
- [!] 持币集中度高（前 10 地址 65%）
- [!] 交易量偏低
- [!] 社交媒体活跃度低

### 💡 投资建议
- 建议仓位：不超过总资金 5%
- 止损位：-20%
- 关注指标：交易量、持币地址数

### 📈 项目数据
- 市值：$500K
- 24h 交易量：$50K
- 持币地址：1,234
- 上线时间：7 天
```

## 技术实现
- 多链数据抓取（ETH/BSC/SOL）
- 合约代码静态分析
- 社交媒体数据监控
- 风险模型评估
