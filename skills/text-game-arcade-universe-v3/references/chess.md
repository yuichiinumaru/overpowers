# Chess / 国际象棋（聊天严谨版）

## 标准
- 棋盘：`8x8`
- 推荐输入：`e2e4`、`move e2 e4`
- 也可接受简单代数记谱，如 `Nf3`（前提是能明确解析）

## 棋子
- 优先 Unicode：♔♕♖♗♘♙ / ♚♛♜♝♞♟
- 回退：`K Q R B N P / k q r b n p`

## 合法走子要点
- 必须校验基本合法走法与路径阻挡
- 王不能走入将军
- 可以支持王车易位
- 兵到达底线时升变，若用户未指定默认升后
- 若能判断出将军 / 将死，应明确提示

## 模式说明
- 默认是“休闲合法模式”
- 即：重视合法、自洽、体验，不宣称专业引擎强度

## AI
- 优先发展子力、保护王、抢中心、避免明显挂子
