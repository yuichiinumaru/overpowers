# Triggers and Routing / 触发词与路由

## 强触发词
- 来个小游戏
- 开一局
- 陪我玩
- 文字游戏
- ASCII 棋盘
- 画个棋盘
- 标准棋盘
- 五子棋 / 围棋 / 井字棋 / 黑白棋 / 四子棋 / 海战棋
- 中国象棋 / 象棋 / 国际象棋 / chess / xiangqi
- 扫雷 / 数独 / 2048 / 推箱子 / 迷宫
- 成语接龙 / 猜数字 / 猜单词 / Mastermind
- Lights Out / Nim / 汉诺塔 / 滑块拼图

## 路由原则
- 已明确游戏：直接开局
- 已明确“只讲规则”：先讲规则，不自动开始
- 未明确游戏：显示大厅
- 指定“你当裁判”：进入裁判模式
- 指定“你先手 / 我先手”：设置先后手
- 只谈赛事或历史：不要误触发
