# 购房政策检索方法论

## 检索方法

用 `web_search` 查以下关键词组合（替换为目标城市名和当前年份）：

```
"{城市} 二手房 限购政策 {年份}"
"{城市} 首套房 首付比例 最新"
"{城市} 二套房 商贷利率 {年份}"
"{城市} 公积金贷款 额度 房龄限制"
"{城市} 二手房 税费 满五唯一"
"{城市} 学区房 多校划片 最新政策"
"{城市} 老旧小区 拆迁改造 城市更新规划"
```

**原则**：每个维度至少交叉验证 2 个来源（官方文件 + 主流媒体），不信单一来源。政策变化快，不能依赖记忆。

| 限制 | 说明 |

## 政策作为砍价弹药的用法

### 房龄限制 → 受众缩小论
"这个房龄不能公积金，商贷只能贷 X 年，月供比同价位新房高 XX%。
能接受这个条件的买家不多，定价应该匹配受众范围。"

### 学区变化 → 溢价缩水论
"多校划片政策实施后，原来对口 XX 学校的确定性降低了，
按学区房溢价来定价已经不现实。"

### 税费 → 谈判变量
"满五唯一的话税费由卖方承担较少，但如果不是满五唯一，
个税和增值税加起来可能 XX 万——这部分成本谁出，可以谈。"

### 政策窗口 → 时机把握
"当前政策对买方有利（如首付比例下调、利率降低），
这是进场的好窗口，但也意味着竞争可能增加——信息速度就是优势。"

## ⚠️ 常见政策陷阱

1. **"满五唯一"自己查**：去不动产登记中心核验，不信经纪人口头说法
2. **房龄从竣工年算**：不是房产证登记日期，相差可能好几年
3. **公积金缴存基数**：影响最高贷款额度，自己算清楚
4. **限售期**：部分城市拿证后 N 年内不能转让，影响投资回报
5. **继承/赠与房产**：税费计算方式不同，要单独确认
