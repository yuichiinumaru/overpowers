# Fiverr 自動販売スキル

Fiverrでの出品・顧客対応・納品を自動化するスキル。英語ベースで国際的なクライアントに対応。

## Fiverr の特徴

### プラットフォーム概要
```
Fiverr:
├── グローバルマーケット（英語ベース）
├── 「Gig」単位での出品
├── 最低価格 $5〜
├── 手数料 20%
├── Seller Levels: New → Level 1 → Level 2 → Top Rated
└── レビューが極めて重要
```

### ココナラとの違い
| 項目 | Fiverr | ココナラ |
|------|--------|---------|
| 言語 | 英語 | 日本語 |
| 市場 | グローバル | 日本 |
| 価格帯 | $5〜 | ¥500〜 |
| 手数料 | 20% | 22% |
| 競争 | 激しい | 中程度 |

## Gig（出品）作成

### Gig構成要素
```
必須項目:
├── Title: SEO意識、キーワード含む
├── Category: 正確なカテゴリ選択
├── Pricing: 3パッケージ（Basic/Standard/Premium）
├── Description: 詳細な説明、FAQ含む
├── Requirements: 必要な情報リスト
├── Gallery: 画像/動画
└── Tags: 最大5つ
```

### タイトルの書き方
```
✅ 良い例:
"I will create Google Apps Script automation for your spreadsheets"
"I will build a custom Python script for data processing"

❌ 悪い例:
"GAS development" (短すぎる)
"I am an expert developer" (何ができるか不明)
```

### 価格設定（3パッケージ）
```
Programming & Tech の例:

Basic ($50):
├── Simple script (up to 100 lines)
├── 1 revision
├── 3 days delivery
└── Source code

Standard ($100):
├── Medium script (up to 300 lines)
├── 2 revisions
├── 5 days delivery
├── Source code
└── Documentation

Premium ($200):
├── Complex automation (500+ lines)
├── Unlimited revisions
├── 7 days delivery
├── Source code
├── Full documentation
└── 30-day support
```

### 説明文テンプレート
```
⭐ [WHAT YOU'LL GET] ⭐

that saves you hours of manual work.

✅ WHAT I OFFER:
• Custom GAS development
```

## セキュリティ連携（moltbook-security）

### メッセージチェック
```
送信前に確認:
├── 機密情報が含まれていないか
├── Fiverr外への誘導がないか（規約違反）
├── 個人連絡先がないか
└── 不適切な内容がないか
```

### 危険な依頼の検出
```
自動拒否:
├── 著作権侵害への協力
├── スパム/詐欺関連
├── 違法コンテンツ作成
├── 個人情報収集
└── マルウェア作成
```

### 支払いの安全性
```
Fiverr経由:
├── 入金: Fiverrが代行（安全）
├── 出金: PayPal/銀行（設定済み）
├── AIの関与: 売上確認（読み取り）のみ
└── リスク: 低（プラットフォームが保護）
```
