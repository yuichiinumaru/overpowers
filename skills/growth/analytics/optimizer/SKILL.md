# ai-seo-optimizer - AI 智能 SEO 优化系统

**痛点**：关键词难找，内容优化没方向，排名提升慢，SEO 效果难追踪
**价值**：AI 驱动 SEO 全流程优化，快速提升搜索排名，获取免费流量

## 使用方式

```bash
ai-seo-optimizer --help
```

## 功能特点

- ⚡ **智能关键词挖掘** - AI 分析搜索意图，挖掘高价值长尾词
- 📊 **内容优化建议** - 基于 TOP 排名页面生成具体优化建议
- 💰 **排名追踪分析** - 实时监控关键词排名，可视化趋势分析
- 🔒 **竞品 SEO 分析** - 分析竞品排名策略，找出超越机会
- 🎯 **技术 SEO 诊断** - 网站结构、速度、移动端等技术问题检测

## 适用场景

- 网站 SEO 优化
- 内容营销关键词策略
- 搜索排名提升
- 独立站流量增长

## 配置

在 `~/.openclaw/workspace/config/ai-seo-optimizer.json` 配置 SkillPay API Key。
