# Customer Insights Skill

分析客户数据，发现成交机会。

## 功能

- 客户行为分析
- 成交预测
- 客户分级
- 最佳跟进时机
- 个性化营销建议

## 使用方式

```
分析我最近的客户数据
```

```
哪些客户最可能成交？
```

```
给我营销优化建议
```

## 配置

### 客户数据源
```json
{
  "sources": {
    "wechat": "微信聊天记录",
    "email": "邮件往来",
    "crm": "CRM 系统（可选）"
  }
}
```
