# Auto Researcher - Autonomous Research Assistant

## 🎯 Core Features

Autonomously research any topic in depth:
- Multi-source cross-verification.
- CRAAP framework for credibility assessment.
- Automatic reference report generation.
- Knowledge graph construction.
- Continuous monitoring and updates.

## 📚 Research Methodology

### 5-Phase Research Flow

```
1. Define
   - Clarify the question.
   - Identify knowns and unknowns.
   - Set the scope.

2. Search
   - Multi-strategy search.
   - Diverse source gathering.
   - Query optimization.

3. Evaluate
   - CRAAP framework.
   - Data extraction.
   - Document limitations.

4. Synthesize
   - Integrate findings.
   - Resolve contradictions.
   - Identify uncertainties.

5. Verify
   - Cross-checking.
   - Confidence scoring.
```

## 🔍 CRAAP Evaluation Framework

### Currency
- [ ] When was it published/updated?
- [ ] Is the information still current?
- [ ] Are links still valid?
- [ ] Technical topics > 2 years old may be outdated.

### Relevance
- [ ] Does it directly answer the question?
- [ ] Who is the target audience?
- [ ] Is the level of detail appropriate?
- [ ] Is it worth citing?

### Authority
- [ ] What are the author's credentials?
- [ ] Who is the publisher?
- [ ] Contact information provided?
- [ ] Domain type (.gov/.edu/org)?

### Accuracy
- [ ] Is there supporting evidence?
- [ ] Has it been peer-reviewed?
- [ ] Can it be verified by other sources?
- [ ] Are there factual errors?

### Purpose
- [ ] Why does this exist?
- [ ] Informational, commercial, persuasive, or entertainment?
- [ ] Is there obvious bias?
- [ ] Does the author benefit?

### Scoring Standards
```
A (Authoritative): Passes all 5 criteria.
B (Reliable): Passes 4/5 criteria.
C (Useful): Passes 3/5 criteria; use with caution.
D (Weak): Passes ≤ 2/5 criteria.
F (Unreliable): Failure; do not cite.
```
