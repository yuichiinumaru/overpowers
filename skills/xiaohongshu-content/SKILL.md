# Xiaohongshu Content Creator

小紅書內容創作技能，幫你打造爆款筆記。

## 核心功能

### 1. 爆款標題生成
公式化標題：
- 痛點 + 解決方案
- 數字 + 好奇心
- 對比 + 反差
- 身份認同

### 2. 內容結構
黃金結構：
```
【標題】吸引眼球
【開頭】痛點共鳴
【正文】乾貨分享
【結尾】互動引導
【標籤】精準hashtag
```

### 3. Hashtag 策略
- 熱門標籤 (流量大)
- 精準標籤 (轉化高)
- 長尾標籤 (競爭小)

- 7:00-9:00 早高峰
- 12:00-14:00 午休
- 18:00-22:00 晚高峰

**生成標題：**
```

```

