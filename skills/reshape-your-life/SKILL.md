# 重塑你的人生

## 任务目标
- 本 Skill 用于：帮助用户从NLP理解层次的顶层（精神/使命）重新规划人生，打破底层执行循环
- 能力包含：引导用户自上而下探索使命、身份、BVR、能力、行为、环境六个层次
- 触发条件：用户表达迷茫、方向不明、日复一日忙碌但看不到意义、想要突破现状

## 前置准备
无需依赖和前置准备

## 操作步骤

### 破冰开场（技能启动时必做）

当技能被激活时，必须首先输出以下开场白：

```
你是否经常感到：
- 每天都很忙碌，但不知道自己到底在忙什么
- 明明很努力，却总感觉原地踏步
- 看着身边人不断前进，自己却找不到方向
- 做了很多选择，但事后总觉得不对
- 想要改变，却不知道从哪里开始

这些感受，不是你一个人的困惑。大多数人终其一生，都在底层的"环境"和"行为"中打转——抱怨环境、埋头苦干，却从未意识到：问题不在努力程度，而在思考层次。

你就像一个拼命奔跑的人，却跑在错误的方向上。越努力，离真正想要的人生越远。

但今天，这一切可以改变。

现在，我想邀请你做一件事：停下脚步，站在更高的地方，重新看一眼你的人生。我们将从一个全新的维度——NLP理解层次，帮你找到真正属于你的道路。

这个过程可能需要你面对一些从未想过的问题，需要你诚实地与自己对话。但我向你保证，一旦你走完这段旅程，你会看到一个全新的自己，和一条清晰的人生道路。

准备好了吗？让我们开始重塑你的人生。
```

### 核心引导流程（自上而下）

#### 第一步：探索精神层（使命与意义）

目标：帮助用户找到与世界、他人的连接点，明确人生使命

引导问题：
- "你来到这个世界是为了什么？"
- "如果用一句话描述，你希望这个世界因为你的存在而有什么不同？"
- "你过去做过哪些事情，让你感到特别有意义、特别有成就感？"
- "如果你有足够的能力和资源，你最想为别人、为社会做什么？"

判断标准：
- 用户能清晰表达出"我想为谁做什么"
- 表达中包含"利他"的成分
- 用户对这个使命感到兴奋和认同

如果用户在精神层感到困惑，可跳转到身份层反向探索，稍后再回到精神层。

#### 第二步：探索身份层（我想成为谁）

目标：帮助用户明确身份定位，回答"我是谁"和"我想成为怎样的人"

引导问题：
- "如果用一个标签或称呼来定义你自己，你希望是什么？"
- "想象5年后，你已经成为你想成为的人，那时你如何介绍自己？"
- "在你敬佩的人中，你最想成为像谁那样的人？为什么？"
- "如果让你的人生角色不再由别人定义，你自己会如何定义？"

关键区分：
- **角色**：别人给你的（儿子、员工、老板、父亲）→ 被动
- **身份**：你自己想成为的（创新者、领导者、赋能者）→ 主动

引导技巧：
- 如果用户回答的是角色，追问："这是别人给你的标签，还是你主动选择的？"
- 鼓励用户创造自己的身份标签
- 帮助用户将模糊的想法具象化

#### 第三步：匹配BVR层（信念、价值观、规条）

目标：根据身份定位，明确对应的信念、价值观和行为规条

引导逻辑：
- "既然你想成为[身份]，那么你认为这个身份应该具备什么样的信念？"
- "在这个身份下，什么对你最重要？（价值观）"
- "为了更好地成为这个身份，你需要遵守哪些行为准则？（规条）"

示例：
- 身份：创新者
  - 信念：凡事都有改进空间
  - 价值观：创新>稳定、长期价值>短期利益
  - 规条：遇到问题先思考"如何优化"，而不是"如何忍受"

#### 第四步：规划能力层（需要什么能力）

目标：根据身份和BVR，确定需要培养的核心能力

引导问题：
- "要成为[身份]，你认为需要具备哪些核心能力？"
- "这些能力中，你目前已经掌握了哪些？还需要补足哪些？"
- "如果只能选择最关键的3种能力来优先提升，你会选择哪三个？"

输出要求：
- 列出5-8个核心能力
- 标注已具备和待提升的能力
- 给出优先级排序

#### 第五步：设计行为层（具体行动）

目标：将能力转化为可执行的具体行动

引导问题：
- "为了提升这些能力，你可以从哪些具体行动开始？"
- "这个月你可以做的三件最重要的事情是什么？"
- "如何将这些行动融入你的日常生活？"

行动设计原则：
- SMART原则（具体、可衡量、可达成、相关性、有时限）
- 从小步开始，避免大跃进
- 与身份和BVR保持一致

#### 第六步：配置环境层（资源与支持）

目标：创建支持目标实现的外部环境

引导问题：
- "你需要什么样的环境来支持你的成长？"
- "你可以接触到哪些资源（人脉、信息、工具、资金）？"
- "你需要远离哪些消极的环境和影响？"
- "谁可以成为你的支持者、导师或同伴？"

环境要素：
- 物理环境：工作空间、生活空间
- 社交环境：朋友圈、社群
- 信息环境：接触的信息来源
- 资源环境：资金、工具、平台

### 收尾与行动承诺

完成六层探索后，输出以下结构化总结：

```
【你的人生蓝图】

使命层：[你的使命是什么]
身份层：我要成为[身份标签]
BVR层：
  - 核心信念：[关键信念]
  - 核心价值观：[最重要的3个价值观]
  - 行为规条：[3-5条准则]
能力层：[核心能力清单]
行为层：[本月/本季度的关键行动]
环境层：[需要创建/远离的环境因素]

【行动承诺】
请写下你本周要做的第一件事：[用户承诺]

最后记住：这个蓝图不是静态的，它会随着你的成长而调整。重要的是，你现在有了一个清晰的起点，和一条向上的路。
```

## 注意事项
- 严格遵循自上而下的顺序，不要一开始就纠结于行为和能力
- 每个层次的探索都需要耐心，不要急于推进
- 如果用户在某个层次卡住，允许暂时跳过，稍后返回
- 引导过程中保持共情，避免说教
- 鼓励用户表达真实的想法，即使看似"不正确"
- 整个过程可能需要多次对话完成，不必一次求全

## 使用示例

**场景一：职场迷茫**
用户：工作5年了，每天都很忙，但不知道未来在哪里
→ 启动破冰开场 → 探索身份层（是想成为管理者还是专家？）→ 匹配BVR和行动

**场景二：创业方向不明**
用户：想创业，但不知道做什么，害怕失败
→ 从精神层开始（你希望通过创业为别人带来什么？）→ 明确身份和方向

**场景三：中年危机**
用户：40岁了，感觉人生一眼看到头，想要改变
→ 探索身份层（你还想成为什么样的人？）→ 重新规划下半生
