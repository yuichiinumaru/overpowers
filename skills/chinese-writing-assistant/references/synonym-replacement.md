# 常用同义替换库（洗稿专用）

## 动词替换
- 提高/提升/增强 → 优化/赋能/升级/加持/强化/增益
- 降低/减少/缩小 → 压缩/管控/压降/收窄/削减/优化
- 完成/做完/实现 → 落地/交付/达成/跑通/闭环/兑现
- 开始/启动/开展 → 上线/发起/撬动/布局/推进/铺开
- 结束/停止/完成 → 收官/收尾/落地/闭环/完结/交割
- 支持/帮助/协助 → 助力/赋能/保驾/托举/协同/配合
- 显示/表明/证明 → 印证/凸显/彰显/折射/反映/佐证
- 利用/使用/运用 → 复用/撬动/借助/依托/基于/通过
- 认为/觉得/相信 → 坦言/指出/强调/直言/表示/坦言
- 影响/改变/带动 → 撬动/辐射/渗透/传导/赋能/带动

## 名词替换
- 优势/优点/长处 → 亮点/禀赋/壁垒/长板/核心竞争力/差异化优势
- 缺点/不足/问题 → 短板/瓶颈/痛点/卡点/待优化点/提升空间
- 方法/办法/技巧 → 路径/方案/玩法/逻辑/方法论/落地路径
- 结果/效果/成果 → 成效/产出/增益/势能/价值沉淀/落地成果
- 计划/规划/安排 → 布局/蓝图/路径/节奏/排期/落地框架
- 趋势/潮流/方向 → 风口/赛道/主旋律/新态势/发展曲线/行业脉络
- 经验/经历/体会 → 体感/沉淀/认知/方法论/一手心得/实战经验
- 机会/机遇/时机 → 窗口/红利/增量/新赛道/增长极/切入点
- 风险/危险/隐患 → 卡点/不确定性/潜在问题/隐忧/雷区/下行压力
- 能力/本事/水平 → 实力/势能/禀赋/硬实力/核心能力/综合素养

## 形容词替换
- 好的/优秀的/不错的 → 亮眼的/硬核的/突出的/可圈可点的/超预期的/标杆级的
- 差的/不好的/糟糕的 → 不及预期的/待优化的/有提升空间的/偏短板的/滞后的
- 重要的/关键的/核心的 → 重磅的/核心的/战略级的/支柱性的/压舱石级的/关键节点的
- 很多/大量/不少 → 海量/充沛/可观的/不小体量的/量级的/丰富的
- 很少/少量/不多 → 有限的/小幅的/收窄的/量级偏轻的/可控的/小范围的
- 快的/迅速的/快速的 → 高效的/爆发式的/跨越式的/超预期的/提质增速的
- 慢的/缓慢的/迟缓的 → 平稳的/缓步的/循序渐进的/稳扎稳打的/节奏偏缓的
- 新的/全新的/最新的 → 迭代的/升级的/全新维度的/重构级的/颠覆式的/换代级的

## 句式替换
- 因为A，所以B → 基于A的底层逻辑，我们得以实现B / 正是A的加持，才带来了B的结果 / 背靠A的优势，B的落地水到渠成
- 不仅A，还B → 除了基础的A能力，更额外加持了B属性 / 在A的基础上，进一步叠加了B的buff / 既覆盖了A的需求，更满足了B的更高要求
- 如果A，就会B → 一旦触发A的条件，B的效果就会自然显现 / 假设落地A的路径，大概率能收获B的结果 / 若能跑通A的逻辑，B的价值会进一步放大
- 很多人认为A，其实B → 大众认知里普遍觉得A，但实际落地中往往是B / 传统思路会偏向A，但更优解其实是B / 过往的经验是A，而新的趋势已经转向B
