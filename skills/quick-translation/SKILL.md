# Quick Translation Skill

🚀 快速翻译工具，支持 100+ 语言，专业术语识别，批量翻译。

## 功能特性

### 核心功能
- **多语言翻译** - 支持 100+ 语言互译
- **专业术语识别** - 自动识别技术/法律/医疗术语
- **批量翻译** - 一次翻译多个段落或文件
- **翻译历史** - 保存翻译记录，方便回顾
- **发音支持** - 提供原文和译文发音

### 高级功能
- **上下文感知** - 根据上下文选择最佳翻译
- **风格调整** - 正式/非正式/口语化
- **术语库** - 自定义专业术语翻译
- **对比显示** - 原文译文对照查看

## 使用方式

### 基础翻译
```
翻译这段文字到英文：你好，世界
```

### 批量翻译
```
把以下内容全部翻译成日语：
1. 你好
2. 谢谢
3. 再见
```

### 专业翻译
```
用专业术语把这个技术文档翻译成中文
```

### 风格调整
```
用口语化的英语翻译：今天天气真好
```

## 输出格式

### 标准输出
```
🌐 翻译结果

原文：
你好，世界

译文：
Hello, World

语言：中文 → 英文
```

### 专业术语输出
```
🌐 专业翻译

原文：
API endpoint configuration

译文：
API 端点配置

术语说明：
- API: 应用程序接口
- endpoint: 端点
- configuration: 配置
```
