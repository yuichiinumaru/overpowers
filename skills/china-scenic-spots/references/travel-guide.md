# 旅游实用指南

## 🗓️ 按季节推荐

### 春季（3-5月）
**适合：** 江南古镇、名山、赏花
- ⭐ 西湖赏花（3月樱花、4月油菜花）
- ⭐ 皖南徽州（油菜花最美时节）
- ⭐ 扬州瘦西湖（春日烟花三月下扬州）
- ⭐ 黄山（云海最多）
- ⭐ 北京（颐和园、故宫）
- ⭐ 武夷山（茶季+山水）

### 夏季（6-8月）
**适合：** 避暑山岳、高原草原、东北
- ⭐ 长白山（天池冰雪融化）
- ⭐ 九寨沟（流量调整期，需提前预约）
- ⭐ 新疆（那拉提草原、喀纳斯）
- ⭐ 黄果树瀑布（水量最大）
- ⭐ 神农架（避暑胜地）
- ⭐ 哈尔滨中央大街（夏季休闲）

### 秋季（9-10月）
**适合：** 赏叶、全国大部分景区
- ⭐ 九寨沟（秋色最美，色彩最丰富）
- ⭐ 新疆喀纳斯（白桦林金黄）
- ⭐ 北京香山（红叶盛宴）
- ⭐ 黄山（秋高气爽、云海多）
- ⭐ 张家界（秋季云雾缭绕）
- ⭐ 壶口瀑布（水量适中，景色壮观）

### 冬季（11月-次年2月）
**适合：** 南方、海南、东北冰雪
- ⭐ 海南三亚（天涯海角、亚龙湾）
- ⭐ 哈尔滨冰雪大世界（12月开幕）
- ⭐ 长白山（滑雪+雾凇+温泉）
- ⭐ 云南（四季如春，昆明、西双版纳）
- ⭐ 广东（丹霞山、长隆）

## 🚗 交通建议

### 适合自驾的景区
- 壶口瀑布（山西/陕西两侧）
- 张掖七彩丹霞（甘肃）
- 新疆全线（独库公路）
- 川藏线（成都→拉萨）
- 广西（桂林→阳朔）

### 必须跟团或有向导的景区
- 布达拉宫（西藏进藏证）
- 雅鲁藏布大峡谷（边境证）
- 新疆部分地区（导游推荐）

### 高铁直达的热门景区
- 故宫、颐和园（北京）
- 兵马俑（西安北）
- 西湖（杭州）
- 黄山（黄山北站）
- 张家界（张家界站）
