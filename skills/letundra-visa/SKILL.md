# Letundra Visa

Визовая информация с letundra.com.

## Когда использовать

- "Нужна ли виза в Таиланд?"
- "Визовые требования для ОАЭ"
- "Документы для визы в Индию"

## Workflow

1. **Определи slug страны**
   - Таиланд → thailand
   - ОАЭ → uae
   - Турция → turkey
   - и т.д.

2. **Получи данные**
   ```
   {{web_fetch url="https://letundra.com/ru/countries/{slug}/"}}
   ```

3. **Извлеки визовую информацию**
   - Нужна ли виза
   - Безвизовый срок
   - Типы виз
   - Документы

## Output format

```markdown
# 🛂 Виза в [Страна]

**Нужна ли виза:** Да/Нет
**Безвизовый срок:** X дней
```

## Failure handling

- Страна не найдена: "Информация о визах для этой страны недоступна"
