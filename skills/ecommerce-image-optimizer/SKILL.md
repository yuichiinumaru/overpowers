# 电商主图优化 - 点击率提升专家

## 概述
基于电商平台数据，优化商品主图设计，提升点击率和转化率。支持淘宝、京东、拼多多、亚马逊等平台。

## 定价
- **按次收费：** ¥9/次
- **包月订阅：** ¥199/月（无限次）

## 功能特性
- 📊 点击率预测
- 🎨 设计元素分析
- 🏆 竞品主图对比
- 💡 优化建议
- 📈 A/B 测试方案

## 使用方式
```bash
# 基础分析
/image-optimize "商品描述" --platform 淘宝

# 竞品对比
/image-optimize "商品" --compare "竞品 URL"

# A/B 测试
/image-optimize "商品" --ab-test

# 批量优化
/image-optimize "商品列表" --batch
```
