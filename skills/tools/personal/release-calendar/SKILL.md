# Anime Weekly Update Calendar

## Data Sources
Use the `extract_content_from_websites` or `batch_web_search` tool to obtain anime update information from the following websites:
- **Bilibili** (bilibili.com)
- **Bangumi** (bgm.tv)
- **Dongman Zhijia** (dmzj.com)

## Scope of Features
1. **This Week's Updates**: Get anime updated this week
2. **Daily Updates**: Query anime updated on a specific day
3. **Following Reminders**: Update times for specific anime
4. **New Anime Information**: Latest anime news and broadcast times

## Anime Update Patterns (Reference)
Japanese anime are usually updated weekly, with Saturdays and Sundays being the most concentrated update days.

## Seasonal Anime Schedule
- **Winter Anime**: January-March
- **Spring Anime**: April-June
- **Summer Anime**: July-September
- **Autumn Anime**: October-December
