## 💡 Core Features

### 1. Automatic Conversation Recording
**Automatically saves after each conversation:**
- ✅ Conversation topic
- ✅ Key information
- ✅ To-do items
- ✅ Important decisions
- ✅ Emotional state

**Storage Location:** `memory/temp/session_*.md`

### 2. Automatic Daily Summary (20:00)
**Automatically processes every night:**
- ✅ Reads all temporary session files
- ✅ Extracts key information
- ✅ Generates a daily report (`memory/2026-03-10.md`)
- ✅ Deletes temporary files (automatic cleanup)

**Daily Report Content:**
- Tasks completed today
- New knowledge learned
- Important decision records
- To-do items for tomorrow

### 3. Automatic Weekly Refinement (Sunday 20:00)
**Automatically processes every Sunday:**
- ✅ Reads the 7 daily reports from the week
- ✅ Extracts core information
- ✅ Updates long-term memory (`MEMORY.md`)
- ✅ Cleans up expired files (older than 30 days)

**Long-Term Memory Content:**
- User preferences (likes/dislikes)
- Important decisions (why the decision was made)
- Lessons learned (pitfalls encountered)
- Growth trajectory (progress and changes)

### 4. Real-time Saving (Dual Protection)
**Immediately saves at key points:**
- ✅ After skill release
- ✅ When payment information changes
- ✅ After important decisions
- ✅ New discoveries/learning

**Location:** Writes directly to `MEMORY.md`

## 🏗️ System Architecture

```
Conversation Log → Temporary Files → Daily Summary → Weekly Refinement → Long-Term Memory
   ↓           ↓           ↓           ↓           ↓
Auto Save   memory/temp  memory/    MEMORY.md    Permanent Storage
          session_*    YYYY-MM-DD   (Append-only)
```

**Data Security:**
- ✅ Local storage (not uploaded to the cloud)
- ✅ Automatic backups (prevents loss)
- ✅ Privacy protection (sensitive information not recorded)
