# habit-tracker - Habit Tracking Skill

A tool for habit formation tracking and progress statistics.

## Feature List
1. **Habit Setup** - Add new habits, set target frequency
2. **Daily Check-in** - Quickly record daily habit completion status
3. **Progress Statistics** - View consecutive check-in days and completion rate
4. **Reminder Settings** - Set habit formation reminders

## Trigger Keywords
- set habit, add habit
- check in, complete habit
- habit statistics, consecutive days

## Data Storage
- habits.json - Habit list
- checkins.json - Check-in records

## Usage Examples
- "Add a habit to exercise for 30 minutes every day"
- "I've already done my morning run today, check in"
- "Show me my habit completion rate"
