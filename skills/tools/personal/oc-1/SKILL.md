# meeting-notes - Meeting Notes Skill

Intelligent meeting notes and to-do tracking assistant.

## Feature List
1. **Meeting Notes** - Record meeting highlights, decisions, and participants
2. **To-Do Extraction** - Automatically extract to-do items from meetings
3. **History Query** - Search and review past meeting notes
4. **Schedule Association** - Associate meetings with your calendar

## Trigger Keywords
- Record meeting, meeting notes
- Meeting to-dos, meeting highlights
- View meeting, search meeting

## Data Storage
- meetings.json - Meeting notes
- action-items.json - To-do items

## Usage Examples
- "Record this afternoon's project review meeting"
- "View to-dos from last week's meeting"
- "Search for meetings about API design"
