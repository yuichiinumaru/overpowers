# 滴滴打车服务技能

完整的滴滴打车服务流程，支持地址别名、场景偏好、定时叫车，实现"一键打车"体验。

## 使用约束

**触发关键词** → 必须使用此 skill 作为唯一入口：

| 关键词 | 示例 |
|--------|------|
| 打车、叫车 | "我要打车"、"帮我叫个车" |
| 出租车、网约车 | "叫个出租车"、"网约车到哪了" |
| 从A到B | "从家到公司"、"去机场" |
| 上下班场景 | "我要上班"、"下班回家" |
| 车型需求 | "打快车"、"要专车" |
| 查询订单 | "订单状态"、"司机到哪里了" |

**禁止直接调用 MCP 工具**，由 skill 内部统一调度。

## MCP 工具

| 工具 | 用途 | 命令 |
|------|------|------|
| `maps_textsearch` | 地址解析 | `mcporter call "didi-taxi-mcp.maps_textsearch(keywords: \"地址\", city: \"城市\")" --output json` |
| `taxi_estimate` | 价格预估 | `mcporter call "didi-taxi-mcp.taxi_estimate(from_lng: \"xxx\", from_lat: \"xxx\", from_name: \"xxx\", to_lng: \"xxx\", to_lat: \"xxx\", to_name: \"xxx\")" --output json` |
| `taxi_create_order` | 创建订单 | `mcporter call "didi-taxi-mcp.taxi_create_order(product_category: \"201\", estimate_trace_id: \"xxx\", caller_car_phone?: \"xxx\")" --output json` |
| `taxi_query_order` | 查询状态 | `mcporter call "didi-taxi-mcp.taxi_query_order(order_id?: \"xxx\")" --output json` |
| `taxi_get_driver_location` | 司机位置 | `mcporter call "didi-taxi-mcp.taxi_get_driver_location(order_id: \"xxx\")" --output json` |
| `taxi_cancel_order` | 取消订单 | `mcporter call "didi-taxi-mcp.taxi_cancel_order(order_id: \"xxx\", reason?: \"原因\")" --output json` |

## 优先级规则

```
用户 query 参数 > PREFERENCE.md 偏好数据 > 系统默认值
```

满足发单条件（地址完整、车型明确）时**直接发单，无需二次确认**。

### Phase 6: 定时叫车

**检测时间表达：**

| 表达类型 | 示例 | 解析结果 |
|----------|------|----------|
| 具体时间 | "10:20叫车从A到B" | 今天 10:20 发单 |
| 相对时间 | "20分钟后叫车" | 当前时间 + 20分钟 |
| 具体日期 | "明天早上9点去机场" | 明天 09:00 发单 |

**工作流程：**

```
1. 解析时间表达 → 计算触发时间（ISO 8601 格式）
2. 确认基础信息：起点、终点、车型（如有缺失则澄清）
3. 使用 openclaw cron 设置一次性定时任务
4. 澄清用户："⏰ 已为您设置定时任务：..."
```

**OpenClaw Cron 任务设置：**

```bash
# 一次性定时任务（使用 --at）
openclaw cron add \
  --name "didi-taxi:$(date +%s)" \
  --at "2025-03-10T10:20:00.000Z" \
  --session isolated \
  --message "执行定时打车：从家到公司，快车" \
  --announce \
  --channel last
```

**参数说明：**

| 参数 | 用途 |
|------|------|
| `--name` | 任务名称（必需） |
| `--at` | 触发时间（ISO 8601 UTC 格式，或相对时间如 +20m），需要将本地时间转换为 UTC 时间 |
| `--session` | 会话类型（使用 `--message` 时必需指定 `isolated`） |
| `--message` | 触发时执行的 agent prompt（配合 `--session isolated` 使用） |
| `--announce` | 投递执行结果到聊天 |
| `--channel` | 投递渠道（默认: last，投递到最后使用的渠道） |
| `--delete-after-run` | 一次性任务成功后删除 |
| `--wake` | 唤醒模式（now/next-heartbeat，默认: now） |

**定时任务触发时：**

```
🔔 定时任务触发
⏰ 触发时间: 2025-03-10 10:20
📍 执行: 从家到公司，快车

正在为您叫车...
[重新进入 didi-taxi skill，执行 Phase 1-4]
```

**管理定时任务：**

| 操作 | 命令 |
|------|------|
| 查看任务列表 | `openclaw cron list` |
| 查看运行记录 | `openclaw cron runs --id <job_id>` |
| 手动触发任务 | `openclaw cron run <id>` |
| 编辑任务 | `openclaw cron edit <id> [options]` |
| 删除任务 | `openclaw cron rm <id>` |

**澄清规则（强制）：**

| 操作 | 澄清格式 |
|------|----------|
| 设置成功 | "⏰ 已为您设置定时任务：10:20（今天）从家到公司，使用快车" |
| 任务触发 | "🔔 定时任务已触发，正在为您叫车：家 → 公司" |
| 任务取消 | "❌ 已取消定时任务：10:20从家到公司" |
