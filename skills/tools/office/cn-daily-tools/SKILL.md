# 中文日常工具集 CN Daily Tools

开箱即用的中文实用工具，无需任何API密钥。

## 包含工具

### 🌤 天气查询
```
查天气：北京明天
查天气：上海本周
查天气：[任意城市] [时间范围]
```
- 支持全国所有城市
- 实时温度、湿度、风速
- 3天/7天预报
- 体感温度和穿衣建议

### 💱 汇率换算
```
汇率：100美元换人民币
汇率：1000元换日元
汇率：[金额][货币]换[目标货币]
```
- 实时汇率（误差<1%）
- 支持USD/EUR/JPY/KRW/HKD/GBP等主流货币
- 历史趋势简析

### 📰 新闻摘要
```
今日新闻：科技
今日新闻：财经
今日新闻：[领域]
```
- 抓取最新中文新闻
- AI提炼3句话摘要
- 附原文链接

### 📦 快递追踪
```
查快递：顺丰 123456789
查快递：[快递公司] [单号]
```
- 支持顺丰、圆通、中通、韵达、京东等主流快递
- 实时物流状态
- 预计到达时间
