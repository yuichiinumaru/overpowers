# Usage Tracker - AI Agent 技能使用跟踪和付费验证工具

## 🎯 概述

跟踪和管理 AI Agent 技能使用情况，支持免费和付费功能，为"数据聚合大师"提供计费和商业验证基础。

## 📊 功能特性

### 免费功能
- ✅ 使用次数跟踪
- ✅ 基础统计报告
- ✅ 今日用量查询
- ✅ 个人使用记录
- ✅ 系统整体统计

### 付费功能
- 💎 按次收费 (0.001U/次)
- 💰 详细分析报告
- 📈 使用趋势预测
- 📋 数据导出 (CSV, JSON)

## 🛠️ 技术要求

- Node.js >= 14.0.0
- OpenClaw Gateway >= 1.0.0
- SkillPay.me API 集成 (用于付费功能)

## 🚀 快速开始

### 安装技能
```bash
# 通过 clawhub 安装
npx clawhub install usage-tracker

# 本地测试
@openclaw setup YOUR_SKILLPAY_API_KEY
```

### 基础使用 (免费功能)
```bash
# 记录功能使用
@openclaw 跟踪 数据分析

# 生成个人使用报告
@openclaw 报告

# 查看系统统计
@openclaw 统计
```

### 付费功能 (需要先初始化)
```bash
# 初始化支付系统
@openclaw setup YOUR_SKILLPAY_API_KEY

# 触发付费功能使用
@openclaw 计费 详细分析 0.001

# 查看账户余额
@openclaw 查询余额
```

## 💰 商业模式

### 免费增值
- 基础功能完全免费，建立用户群
- 每日 10 次免费调用额度
- 培养长期用户关系

### 按量付费
- 超出免费额度后，每次调用收费 0.001 U
- 支持 USDT 支付，通过 BNB Chain 结算
- 实时计费，透明到链上

### 收费结构
- 平台费: 5% (SkillPay.me 平台)
- 网络费: ~0.001 U (BNB Chain Gas)
- 开发者净收入: 95% - 网络费
