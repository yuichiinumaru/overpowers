# 研发费用评估技能

## 核心功能

1. **研发费用评估** - 评估研发费用的合理性和合规性
2. **研发预算测算** - 测算研发项目所需的总预算
3. **项目研发成本分析** - 分析项目各阶段的成本构成
4. **研发投资回报分析** - 计算ROI和评估投资价值
5. **合规性检查** - 确保符合高企认定和加计扣除要求

## 必需参数

评估前必须收集以下信息，若用户未提供则主动追问：

| 参数 | 说明 | 示例 |
|------|------|------|
| 项目类型 | 软件/硬件/AI算法/工艺技术 | 软件研发 |
| 研发周期 | 项目持续时间（月） | 6 |
| 研发人数 | 参与研发的人员数量 | 10 |
| 人均月薪 | 研发人员平均月薪（元） | 25000 |
| 是否委外 | 是否有委托外部研发 | 是/否 |
| 委外金额 | 委托外部研发费用（元） | 200000 |
| 风险等级 | 低/中/高 | 中风险 |

## 计算公式

### 1. 人员人工成本
```
人力成本 = 研发人数 × 研发周期(月) × 人均月薪
```

### 2. 直接投入费用
按项目类型乘以系数：
| 项目类型 | 系数 | 说明 |
|---------|------|------|
| 软件研发 | 10% | 设备折旧、软件许可、云服务 |
| 硬件研发 | 35% | 原材料、样品、模具 |
| AI算法研发 | 20% | 算力资源、数据采购 |
| 工艺技术研发 | 25% | 试验材料、检测费用 |

```
直接投入 = 人力成本 × 类型系数
```

### 3. 间接分摊费用
```
间接费用 = 人力成本 × 15%
```
包含：办公场地分摊、水电网络、管理费用分摊

### 4. 委托外部研发
```
委外费用 = 委外金额（若有）
```

### 5. 风险准备金
| 风险等级 | 缓冲比例 |
|---------|---------|
| 低风险 | 5% |
| 中风险 | 10% |
| 高风险 | 15% |

```
风险准备金 = (人力成本 + 直接投入 + 间接费用 + 委外费用) × 风险系数
```

### 6. 最终研发预算
```
总预算 = 人力成本 + 直接投入 + 间接费用 + 委外费用 + 风险准备金
```

## 研发费用归集口径

符合《高新技术企业认定管理工作指引》和《研发费用加计扣除政策》的费用范围：

### 可归集费用
1. **人员人工费用**：研发人员工资薪金、五险一金
2. **直接投入费用**：材料、燃料、动力、模具、样品/样机
3. **折旧费用**：研发设备折旧
4. **无形资产摊销**：软件、专利权摊销
5. **设计试验费**：新产品设计、工艺规程制定、临床试验
6. **委托外部研发**：委托其他机构研发的费用（80%计入）

### 不可归集费用
- 与研发无关的生产成本
- 销售费用、管理费用（除分摊部分）
- 资本化利息支出

## 使用示例

### 示例1：软件项目预算测算

**用户输入：**
> 我需要为一个新的企业级ERP系统做研发预算测算。项目预计需要15人的团队，开发周期8个月。团队包括：项目经理1人、产品经理1人、架构师1人、后端开发5人、前端开发3人、测试工程师3人、UI设计师1人。

**处理流程：**
1. 确认项目类型：软件研发
2. 提取参数：研发人数15人，周期8个月
3. 计算人均月薪：按中级水平估算约35K
4. 计算：
   - 人力成本 = 15 × 8 × 35000 = 4,200,000元
   - 直接投入 = 4,200,000 × 10% = 420,000元
   - 间接费用 = 4,200,000 × 15% = 630,000元
   - 费用小计 = 5,250,000元
   - 风险准备金（中风险10%）= 525,000元
   - **总预算：5,775,000元**
5. 按标准模板输出报告

### 示例2：硬件研发费用评估

**用户输入：**
> 我们公司计划开发一款智能家居控制器，需要进行研发费用评估。产品涉及硬件设计、嵌入式软件开发、APP开发。预计硬件研发团队6人（硬件工程师2人、嵌入式工程师2人、结构工程师1人、测试工程师1人），软件团队4人（APP开发2人、后端1人、UI1人），周期12个月。

**处理流程：**
1. 确认项目类型：硬件研发（含嵌入式软件）
2. 提取参数：研发人数10人，周期12个月
3. 计算人均月薪：硬件工程师约25K，嵌入式约30K，软件约28K，平均约27K
4. 计算：
   - 人力成本 = 10 × 12 × 27000 = 3,240,000元
   - 直接投入 = 3,240,000 × 35% = 1,134,000元
   - 间接费用 = 3,240,000 × 15% = 486,000元
   - 费用小计 = 4,860,000元
   - 风险准备金（中风险10%）= 486,000元
   - **总预算：5,346,000元**
5. 输出阶段成本分布分析

### 示例3：研发投资回报分析

**用户输入：**
> 我们有一个AI客服系统的研发项目，预计投入300万元，开发周期6个月。预计上线后每年可以节省人工成本150万元，增加收入80万元。请帮我做这个项目的投资回报分析。

**处理流程：**
1. 计算年收益：150万 + 80万 = 230万元
2. 计算ROI：(230万 - 300万) / 300万 × 100% = -23.3%（首年）
3. 计算投资回收期：300万 / 230万 ≈ 1.3年
4. 计算NPV（假设折现率10%，5年）：
   - 年现金流：230万
   - NPV = -300万 + Σ(230万/(1.1)^n) ≈ 571万元
5. 结论：项目经济可行，建议推进

### 示例4：预算合理性评估

**用户输入：**
> 请帮我评估一下这个研发预算是否合理。前端开发3人×4个月×25K=30万，后端开发4人×4个月×28K=44.8万，产品经理1人×4个月×35K=14万，UI设计师1人×3个月×22K=6.6万，测试工程师2人×3个月×20K=12万，服务器和云服务10万，其他费用5万，总计约122.4万。

**处理流程：**
1. 核算人力成本：30+44.8+14+6.6+12 = 107.4万
2. 核算其他成本：10+5 = 15万
3. 检查成本结构：
   - 人力成本占比：107.4/122.4 = 87.7%（偏高，行业标准60-70%）
   - 直接投入占比：10/122.4 = 8.2%（偏低，软件项目建议10-15%）
   - 缺少间接费用分摊
   - 缺少风险准备金
4. 优化建议：
   - 增加间接费用约16万（人力成本×15%）
   - 增加风险准备金约12万（中风险10%）
   - **建议总预算：约150万**

### 示例5：小型项目快速预算

**用户输入：**
> 我需要快速估算一个小型工具类App的开发成本。功能比较简单，主要是数据录入和报表展示，预计2个开发人员做2个月。

**处理流程：**
1. 确认项目类型：软件研发
2. 提取参数：研发人数2人，周期2个月
3. 计算人均月薪：按中级水平约25K
4. 计算：
   - 人力成本 = 2 × 2 × 25000 = 100,000元
   - 直接投入 = 100,000 × 10% = 10,000元
   - 间接费用 = 100,000 × 15% = 15,000元
   - 风险准备金（低风险5%）= 6,250元
   - **总预算：约13万元**
5. 输出简明预算报告

## 约束条件

1. 所有数值参数必须 ≥ 0
2. 周期、人员数量必须为正整数
3. 缺失必需参数时主动追问用户
4. 严格按公式计算，不得随意调整系数
5. 输出必须使用标准模板格式
