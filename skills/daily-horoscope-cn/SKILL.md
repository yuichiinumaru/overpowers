# 每日星座运势

## 支持的星座

白羊座、金牛座、双子座、巨蟹座、狮子座、处女座、天秤座、天蝎座、射手座、摩羯座、水瓶座、双鱼座

## 使用方式

当用户问"XX座今天运势如何"时，直接回复以下格式：

```
♈ 白羊座 (3.21-4.19)
━━━━━━━━━━━━━━━━
💫 整体运势: ⭐⭐⭐⭐☆ (4/5)
💕 爱情运:    有意外惊喜，单身者有望遇到有缘人
💼 工作运:    效率提升，适合处理棘手问题
💰 财富运:    适合理性投资，注意开销

🔢 幸运数字: 7
🎨 幸运颜色: 红色
📅 明日提示: 注意休息，别太拼
```

## 运势说明

- 运势仅供娱乐参考
- 可以根据用户具体星座灵活调整内容
- 保持轻松愉快的语气
