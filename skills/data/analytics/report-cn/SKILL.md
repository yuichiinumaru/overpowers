# AI 财务报表生成器 📊

> 自动化生成财务报表，让财务工作更高效

## ✨ 核心功能

### 📈 自动生成报表
- **三大报表**：资产负债表、利润表、现金流量表
- **一键生成**：输入数据，自动生成标准报表
- **自动化处理**：智能计算、自动平衡
- **合规格式**：符合会计准则要求

### 🔍 智能财务分析
- **财务比率分析**：流动比率、速动比率、资产负债率等
- **趋势分析**：同比、环比变化分析
- **结构分析**：收入结构、成本结构分析
- **异常检测**：智能发现异常数据

### 💰 税务筹划
- **税负分析**：增值税、所得税税负分析
- **筹划建议**：合法节税方案建议
- **风险提示**：税务风险预警
- **政策解读**：最新税收政策解读

### 📤 多格式导出
- **PDF 格式**：正式报表，适合提交
- **Excel 格式**：可编辑，适合分析
- **Word 格式**：带分析的完整报告
- **图片格式**：适合演示展示

## 📊 支持的报表类型

1. **资产负债表** - 反映企业财务状况
2. **利润表** - 反映企业经营成果
3. **现金流量表** - 反映企业现金流动情况
4. **所有者权益变动表** - 反映所有者权益变化
5. **附注** - 报表补充说明
