# 保险术语解释

帮助理解保险合同中的常见术语。

---

## 一、基础术语

### 1.1 保障相关

| 术语 | 解释 |
|------|------|
| **保单** | 保险合同，具有法律效力 |
| **投保人** | 花钱买保险的人 |
| **被保险人** | 受保障的人 |
| **受益人** | 领赔偿金的人 |
| **保额** | 保险公司赔付的上限 |
| **保费** | 每年交给保险公司的钱 |
| **保障期限** | 保险有效的时间 |

### 1.2 缴费相关

| 术语 | 解释 |
|------|------|
| **缴费期限** | 需要交多少年 |
| **年缴/季缴/月缴** | 缴费频率 |
| **犹豫期** | 签收后可反悔退保的期限（通常10-15天） |
| **宽限期** | 逾期未缴费仍保障的期限（通常60天） |

---

## 二、医疗险术语

| 术语 | 解释 |
|------|------|
| **免赔额** | 自己需要承担的部分，超过才报销 |
| **报销比例** | 保险公司承担的比例（通常80%-100%） |
| **社保内/外** | 医保目录内/外的费用 |
| **等待期** | 投保后需等待才能理赔（通常30-90天） |
| **保证续保** | 无论健康状况变化都可续保 |
| **垫付** | 保险公司帮忙垫付医疗费 |

---

## 三、重疾险术语

| 术语 | 解释 |
|------|------|
| **确诊即赔** | 符合合同定义就一次性给钱 |
| **轻症/重症** | 疾病的严重程度 |
| **多次赔付** | 同一种病或不同病可以赔多次 |
| **分组/不分组** | 多次赔付是否限制疾病分组 |
| **身故责任** | 去世后是否赔钱 |

---

## 四、寿险术语

| 术语 | 解释 |
|------|------|
| **定期寿险** | 保障一段时间（如到60岁） |
| **终身寿险** | 保障一辈子 |
| **全残** | 达到合同定义的全残标准 |
| **减保** | 减少保额 |

---

## 五、费用相关

| 术语 | 解释 |
|------|------|
| **现金价值** | 退保能拿回的钱 |
| **消费型** | 保费不返还，保障到期作废 |
| **储蓄型** | 保障+储蓄，没理赔也能拿回钱 |
| **返还型** | 到期未理赔返还保费 |

---

## 六、注意事项

1. **等待期内出险** — 通常不赔，只退保费
2. **既往症** — 投保前已有的疾病，通常不赔
3. **免责条款** — 哪些情况一定不赔
4. **不如实告知** — 可能导致拒赔

---

## 七、常见误区

| 误区 | 真相 |
|------|------|
| "买保险贵的就是好的" | 适合的才是好的 |
| "有医保不需要商保" | 医保只能报销部分 |
| "的重疾险确诊就赔" | 必须符合合同定义 |
| "多买几份医疗险可以重复赔" | 报销型不能重复赔 |
| "保险买了就可以赔" | 需过了等待期且符合理赔条件 |
