# AI 数据分析助手 📊

> 让数据开口说话，一键生成专业分析报告

## ✨ 核心功能

### 📈 智能图表生成
- **自动推荐图表**：根据数据类型智能推荐最佳图表
- **20+ 图表类型**：柱状图、折线图、饼图、热力图、散点图、雷达图、漏斗图等
- **一键美化**：自动配色、布局优化
- **交互式图表**：支持缩放、筛选、钻取

### 🔮 预测分析
- **时间序列预测**：预测未来趋势
- **回归分析**：发现数据关系
- **分类预测**：智能分类数据
- **预测准确度**：显示预测置信区间

### 🚨 异常检测
- **智能识别**：自动发现异常数据点
- **异常原因**：AI 分析异常原因
- **异常预警**：设置阈值自动预警
- **历史对比**：对比历史同期数据

### 📝 自动生成报告
- **一键生成**：自动生成完整分析报告
- **多格式导出**：PDF、Word、PPT、Excel
- **报告模板**：多种专业模板可选
- **定制报告**：按需定制报告内容

## 📊 支持的数据格式

- Excel (.xlsx, .xls)
- CSV (.csv)
- JSON (.json)
- 数据库直连（MySQL, PostgreSQL）
