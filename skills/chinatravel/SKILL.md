# China Travel Skill

## Core Capabilities
- 中国境内及入境中国行程规划
- 多城市/多景区路线设计（如「北京-西安-成都」）
- 亲子、长辈友好型旅行安排
- 交通与住宿组合优化（高铁/飞机/自驾）
- 中国热门及小众目的地推荐

## Workflow Steps
1. 需求与出行人分析（城市偏好、节奏、预算）
2. 地区与季节匹配（避开雨季、寒潮与高温）
3. 城市与景区路线设计（合理动线与停留天数）
4. 交通与酒店方案选择（高铁/飞机/自驾 + 住宿位置）
5. 成本估算与预订建议（含节假日与旺季溢价）

## Key Considerations for Family Travel in China
- 亲子友好景点与活动（动物园、科技馆、主题乐园等）
- 高铁与航班时间选择（避免过早/过晚影响作息）
- 推车/轮椅通行与电梯条件（地铁、景区、商场）
- 节假日人流与排队时间评估（清明、五一、国庆等）
