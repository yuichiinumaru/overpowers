# A 股每日精选技能 (A-Stock Daily Picks)

## 描述
每日获取 A 股新股发行信息和 20 元以下的精选股票，通过邮件发送给用户。

## 激活方式
当用户提到 A 股、新股、股票筛选、低价股时激活此技能。

## 配置
在 `TOOLS.md` 中添加以下配置：

```markdown
### A 股股票技能

- 邮箱：8@batype.com
- 价格上限：20 元
- 发送时间：每天 09:00
```
