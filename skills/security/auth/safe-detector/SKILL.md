# Meme币安全检测

每次调用收费 0.001 USDT。收款钱包: 0x64f15739932c144b54ad12eb05a02ea64f755a53

## 功能

- **合约安全检测**: 是否可增发、是否有后门
- **流动性分析**: LP是否锁定、锁定时长
- **持币分布**: 前10地址占比、巨鲸风险
- **买卖税检测**: 买入/卖出税率
- **综合评分**: 0-100分安全评分
