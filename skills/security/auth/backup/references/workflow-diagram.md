```

│  • 内容: config/ + memory/ + scripts/                                   │
│  • 触发: 立即(pendingChanges) 或 12小时定期                              │
└─────────────────────────────────────────────────────────────────────────┘
                                    ▲

┌─────────────────────────────────────────────────────────────────────────┐

│ ───────────────────────────────────                                     │
│  • 文件: MEMORY.md                                                      │
│  • 内容: 关键决策、待办事项、用户档案、配置信息                           │

│  • 作用: 跨会话保持核心信息                                              │

                                    ▲

│  • 作用: 快速回顾会话内容                                                │
└─────────────────────────────────────────────────────────────────────────┘
                                    ▲

│ ───────────────────────────────                                         │
```

## 差异标记系统

```
当前会话分析
      │
      ▼
┌─────────────────┐
│ 对比上次存档     │
│ (memory/index   │
│  或上次 .md)    │
└────────┬────────┘
         │
    ┌────┼────┐
    ▼    ▼    ▼
 [NEW] [UPD] [KEEP]
 新增   更新   保持

标记规则:
• [NEW] - 本次新增的关键决策/待办/配置
• [UPD] - 修改过的内容（附带 previousValue）
• [KEEP] - 与上次相同，无变化

示例:
- [NEW] **新增决策**: 采用四层数据架构
- [UPD] **更新决策**: 备份间隔改为12小时（原：1小时）
- [KEEP] **保持决策**: Token 超限阈值 70k/100k
```
