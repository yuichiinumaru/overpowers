# 餐饮创业选址与开店评估 Skill

基于勇哥餐饮创业视频整理的完整自测评估系统，帮助创业者科学评估选址和开店可行性。

## 快速开始

### 1. 选址评估（最常用）

向用户提供选址评估问卷，收集以下信息：

**基础必过项**（全部必须达标，否则直接放弃）：
- 产权清晰、能办执照、有烟道、水电正常、消防合格、房东可靠、无拆迁风险

**核心评估项**：
- 日均人流量（社区800+/商圈2000+）
- 客群匹配度（匹配/一般/不匹配）
- 租金占营收比例（≤15%为优，≤20%可接受）
- 合同年限（≥3年）
- 回本周期（≤12个月）

收集完成后，调用评估脚本：
```bash
# 保存用户输入为JSON文件
cat > /tmp/eval_input.json << 'EOF'
{
  "location": {
    "property_clear": true,
    "license": true,
    "ventilation": true,
    "utilities": true,
    "fire_safety": true,
    "landlord": true,
    "demolition_risk": false,
    "daily_flow": 2500,
    "customer_match": "match",
    "rent_15_percent": true,
    "contract_3y": true,
    ...
  }
}
EOF

# 运行评估
python3 scripts/evaluate.py /tmp/eval_input.json
```

### 2. 开店可行性评估

评估3个核心问题：
1. 回本周期是否≤12个月？
2. 租金是否≤营收20%？
3. 是否能亲自盯店3个月以上？

**决策规则**：
- 3个"是" → 可以开，成功率高
- 2个"是" → 谨慎开，必须优化
- ≤1个"是" → 别开，90%亏

### 3. 回本周期计算

```
月纯利润 = 月营业额 - 月房租 - 月人工 - 月原材料 - 月水电
回本周期(月) = 总投资 ÷ 月纯利润
```

**勇哥标准**：
- <8个月：可以开
- 8-12个月：谨慎开
- >12个月：不建议开

## 评估维度详解

### 选址评估6大维度

| 维度 | 权重 | 满分 | 核心指标 |
|:---|:---:|:---:|:---|
| 基础必过项 | 必备 | 100 | 7项全部为"是" |
| 人流客群 | 极高 | 55 | 日均人流+客群匹配 |
| 门头动线 | 高 | 40 | 门头醒目+顺人流方向 |
| 竞争业态 | 中 | 33 | 餐饮聚集+竞争可控 |
| 成本回本 | 极高 | 51 | 租金≤15%+合同3年+能回本 |
| 风险避雷 | 高 | 45 | 无修路风险+非死角 |

**评级标准**：
- A级（280+分）：优质旺铺，果断租
- B级（230-279分）：可谈条件租
- C级（180-229分）：谨慎考虑
- D级（<180分）：直接放弃

### 创业自测6大维度

1. **资金自测**：6个月备用金、不借钱、覆盖所有启动成本
2. **选址自测**：流量达标、客群匹配、租金≤20%
3. **产品自测**：简单可复制、毛利≥55%、不依赖大厨
4. **回本模型**：计算月纯利润和回本周期
5. **运营能力**：每天12小时、盯店3个月、能处理投诉
6. **风险自测**：能扛3个月亏损、合同稳定、接受淡旺季

## 避坑指南

### 5种必死开店模型（绝对避免）

| 模型 | 特征 | 勇哥建议 |
|:---|:---|:---|
| 快招加盟坑 | 0加盟费广告、100+SKU、强制装修 | 避开所有快招品牌 |
| 网红概念店 | 装修几十万、颜值>味道、客单价高 | 装修不能吃 |
| 大厨依赖型 | 老板甩手、菜品复杂、大厨谈条件 | 必须标准化 |
| 大而全 | 200㎡+、100+道菜、5-8人 | 新手选小店 |
| 盲目跟风 | 别人卖啥你卖啥、同质化竞争 | 找差异化切口 |

### 4种常胜店模型（推荐）

| 模型 | 配置 | 赚钱逻辑 |
|:---|:---|:---|
| 夫妻店 | 1-2人、10-30㎡、无员工 | 人工成本为0 |
| 蜜雪型 | 20-40㎡、高频刚需、平价 | 薄利多销 |
| 爆品店 | 1-3个爆品、小面积 | 聚焦=暴利 |
| 早餐店 | 10-20㎡、早高峰4小时 | 现金流最好 |

**勇哥口诀**：夫妻店稳，蜜雪爆，单品强，早餐赚。

## 使用示例

### 示例1：选址评估

用户："我想在商业街开个面馆，日流量大概3000人，租金8000/月，预估月营收4万，帮我评估一下"

评估过程：
1. 确认基础必过项（产权、执照、烟道等）
2. 人流量3000 = 商圈标准 ✓
3. 租金占比：8000/40000 = 20% ✓
4. 计算回本周期
5. 给出评级和建议

### 示例2：开店决策

用户："我有20万资金，想加盟一个奶茶品牌，总部说年入百万，能不能做？"

评估过程：
1. 识别为"快招加盟坑"
2. 提示奶茶属于"中年破产四件套"
3. 评估资金（20万是否够6个月备用金）
4. 建议从摆摊测试开始

### 示例3：回本计算

用户："总投资15万，月营收3万，房租5000，人工8000，原材料40%，水电2000，多久回本？"

计算：
- 月原材料：30000 × 40% = 12000
- 月纯利润：30000 - 5000 - 8000 - 12000 - 2000 = 3000
- 回本周期：150000 ÷ 3000 = 50个月

结论：回本周期50个月 > 12个月，不建议开。
