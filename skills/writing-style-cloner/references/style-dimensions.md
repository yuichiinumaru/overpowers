# 写作风格分析维度详解

> 📖 **文件定位**：本文件为**风格分析方法论**，用于分析新作者的写作风格时参考。
> 
> ⚠️ **注意**：此文件不直接用于改写，改写时请参考 `SKILL.md` 中已固化的 Antonia 风格档案。
> 
> **使用场景**：
> - 需要分析新的写作样本时
> - 需要更新/扩展现有风格档案时
> - 需要理解风格分析的完整维度时

---

## 维度1：句式结构

### 分析要点

| 特征 | 如何识别 | 记录方式 |
|------|----------|----------|
| **长短句比例** | 统计句子平均字数 | 短句主导(<15字)/中等(15-30字)/长句主导(>30字) |
| **句式偏好** | 陈述句/疑问句/感叹句比例 | 记录各类型占比 |
| **复杂度** | 单句/复句/多重复句 | 简单直接/层层递进/复杂嵌套 |
| **段落长度** | 每段平均句子数 | 短段(1-3句)/中段(4-6句)/长段(7+句) |

### 示例对照

**短句主导风格**：
> 我辞职了。三十五岁，裸辞。没有下家，没有存款，只有一腔孤勇。

**长句主导风格**：
> 在我三十五岁那年的某个平凡的周二下午，我做出了一个在外人看来极其冲动但我自己却酝酿已久的决定——递交辞呈，离开那个我工作了八年的公司。

## 维度2：用词特征

### 分析要点

| 特征 | 如何识别 | 记录方式 |
|------|----------|----------|
| **口语化程度** | 是否使用"的话""其实""反正"等 | 高度口语/适度口语/书面正式 |
| **专业术语** | 行业词汇使用频率 | 无术语/适度使用/术语密集 |
| **高频词汇** | 统计重复出现的词 | 列出Top10高频词 |
| **形容词风格** | 华丽/朴素/精准 | 记录典型形容词 |
| **动词力度** | 强动词/弱动词偏好 | 如"冲击"vs"影响" |

### 高频词类型

| 类型 | 示例 |
|------|------|
| 情感词 | 真的、特别、非常、超级 |
| 逻辑词 | 所以、因为、但是、其实 |
| 口头禅 | 说真的、你懂的、讲真 |
| 态度词 | 我觉得、在我看来、坦白说 |

## 维度3：修辞手法

### 分析要点

| 手法 | 识别方式 | 频率记录 |
|------|----------|----------|
| **比喻** | "像...""如同...""是..." | 高频/适中/少用/不用 |
| **排比** | 连续三个或以上相似结构 | 高频/适中/少用/不用 |
| **反问** | 不需回答的疑问句 | 高频/适中/少用/不用 |
| **设问** | 自问自答 | 高频/适中/少用/不用 |
| **引用** | 名言、诗句、数据 | 高频/适中/少用/不用 |
| **对比** | 正反对照、前后对比 | 高频/适中/少用/不用 |

### 示例对照

**高修辞风格**：
> 人生就像一场马拉松，不在于你跑得多快，而在于你能跑多远。那些看似遥不可及的终点，那些曾经让你望而却步的距离，那些深夜里独自流下的汗水——终将汇成一条通往梦想的河流。

**低修辞风格**：
> 人生是长跑，速度不重要，耐力才重要。坚持下去就会到达终点。

## 维度4：叙事视角

### 分析要点

| 特征 | 选项 | 说明 |
|------|------|------|
| **人称** | 第一人称/第三人称/混合 | "我"vs"他/她"vs交替使用 |
| **读者称呼** | 你/你们/各位/读者/无称呼 | 与读者的对话感 |
| **距离感** | 亲密/适中/疏离 | 像朋友聊天/像老师讲课/像专家陈述 |
| **卷入感** | 高/中/低 | 是否频繁邀请读者参与思考 |

### 示例对照

**亲密距离**：
> 你有没有过这种感觉？就是明明很累，却怎么也睡不着...

**专业距离**：
> 研究表明，现代人普遍面临睡眠质量下降的问题，这一现象被称为...

## 维度5：情感基调

### 分析要点

| 维度 | 光谱 | 判断依据 |
|------|------|----------|
| **理性-感性** | 理性←→感性 | 逻辑论证vs情感共鸣 |
| **正式-轻松** | 正式←→轻松 | 措辞严谨vs随意幽默 |
| **克制-热烈** | 克制←→热烈 | 平淡叙述vs激情表达 |
| **乐观-悲观** | 乐观←→悲观 | 正面积极vs忧虑批判 |
| **确定-探索** | 确定←→探索 | 给答案vs提问题 |

### 基调描述模板

```
[作者名]的写作风格偏向[理性/感性]，语气[正式/轻松]，
情感表达[克制/热烈]，整体呈现[乐观/中性/悲观]的态度，
倾向于[给出明确观点/引导读者思考]。
```

## 维度6：结构模式

### 分析要点

| 结构元素 | 分析内容 |
|----------|----------|
| **开头模式** | 故事开头/问题开头/金句开头/数据开头/场景开头 |
| **正文结构** | 总分总/并列/递进/对比/时间线 |
| **过渡方式** | 自然过渡/明确标记/问题引导 |
| **结尾风格** | 总结升华/金句收尾/开放思考/行动号召/呼应开头 |

### 常见开头模式

| 模式 | 示例 |
|------|------|
| 故事型 | "去年冬天，我在北京的一个咖啡馆里..." |
| 问题型 | "你有没有想过，为什么..." |
| 金句型 | "有人说，人生最大的遗憾不是失败，而是..." |
| 数据型 | "据统计，90%的人都曾经历过..." |
| 场景型 | "凌晨三点，城市安静下来，只有..." |

## 维度7：个人标记

### 分析要点

这是最能体现个人风格的维度，需要仔细收集：

| 类型 | 说明 | 示例 |
|------|------|------|
| **口头禅** | 反复出现的表达 | "说真的""讲道理""你品，你细品" |
| **标志性句式** | 个人特色的句型 | "与其...不如...""不是...而是..." |
| **独特用语** | 自创或偏好的词 | "搞事情""整活""破防" |
| **情绪词** | 表达态度的习惯用词 | "绝了""离谱""太真实了" |
| **结尾签名** | 固定的结尾方式 | "以上。""就酱。""共勉。" |

### 收集模板

```

结尾习惯：
___
```
