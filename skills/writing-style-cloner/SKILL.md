# Antonia 写作风格克隆器

将语音转录稿/草稿直接改写为符合 Antonia 个人风格的自媒体文章。

## 工作流程

```
输入转录稿/草稿 → 按风格档案改写 → 输出文章
```

---

## 适用边界

| 类型 | 说明 |
|------|------|
| ✅ **适用** | 自媒体经验分享、个人成长复盘、学习方法论、职场心得、读书笔记 |
| ✅ **适用** | 口述/录音转文章、草稿润色、风格统一化改写 |
| ⚠️ **慎用** | 新闻稿、产品文档、正式商务邮件（需保留部分正式感） |
| ❌ **不适用** | 学术论文、法律文书、政府公文、技术规范文档 |

**目标字数范围**：
- 短文：800-1500 字（单一主题）
- 中文：2000-4000 字（2-3 个核心观点）
- 长文：5000-8000 字（系列化内容、多维度拆解）

---

## 固化风格档案

> 基于8篇历史文章样本分析生成（腾讯实习转正系列、学习经验系列等，总计约9万字）

### 风格概述

Antonia 的写作风格是**理性真诚、结构清晰、亲切自然**的经验分享型写作。文章以第一人称视角展开，像与朋友聊天一样分享个人经历和实用方法，既有干货的扎实感，又有温暖的人情味。善用结构化表达（标题层级、列表、表格）让内容易读易用，同时通过真诚的自我剖析和温暖的结尾鼓励，建立与读者的情感连接。

---

### 1. 句式结构

| 特征 | 描述 |
|------|------|
| **长短句比例** | 中等句长为主，穿插短句增强节奏感 |
| **平均句长** | 约 20-35 字 |
| **段落长度** | 短段为主（1-4句），便于阅读 |
| **句式偏好** | 陈述为主，常用设问引导思考 |

**典型句式**：
> "回头看，所谓'主R意识'，并不是通过某一次刻意表现，而是在长期协作中，通过对结果负责、对风险敏感、对问题不逃避一点点积累起来的。"

> "从我自己的经验来看，一篇论文最后能不能拿高分，真正的分水岭，往往出现在动笔之前。"

---

### 2. 用词特征

| 特征 | 描述 |
|------|------|
| **口语化程度** | 适度口语化，亲切自然但不失专业 |
| **专业术语** | 适度使用，会用通俗语言解释 |
| **形容词风格** | 朴素精准，不堆砌华丽辞藻 |

**高频词汇**：
- 连接词：其实、所以、因此、但是、不过、换句话说、也正因为如此
- 程度词：非常、很、特别、真的、确实
- 态度词：我觉得、在我看来、从我的经验来看、说实话
- 总结词：总的来说、回头看、从结果来看

**特色用语**：
- 使用 emoji 表情增加亲和力（😭🧨💦😅🥹等）
- 括号内补充说明或自嘲（如"（一些出版专业学生的职业病😅）"）

---

### 3. 修辞偏好

| 手法 | 使用频率 | 典型用法 |
|------|----------|----------|
| 设问 | **高频** | "那问题来了：xxx？" |
| 反问 | 中等 | 用于强调观点 |
| 对比 | **高频** | 正反对照、前后对比 |
| 排比 | 中等 | 用于列举要点 |
| 比喻 | 低频 | 偶尔使用，朴素自然 |
| 引用 | 中等 | 引用他人观点、书籍内容 |

**修辞示例**：
> "真正赢得信任的，从来不是'情商'，而是长期稳定的靠谱。"

> "职场中的形象管理，并不是一种肤浅的'外在装饰'，而更像是一种基础信用建设。"

---

### 4. 叙事视角

| 特征 | 描述 |
|------|------|
| **人称** | 第一人称为主（"我"） |
| **读者称呼** | "你""大家""同学" |
| **距离感** | **亲密**——像朋友聊天、分享经验 |
| **卷入感** | **高**——频繁与读者互动、邀请思考 |

**互动方式**：
- "你有没有过这种感觉？"
- "如果你也即将xxx，希望这篇文章能给你一点启发"
- "欢迎关注！""下篇见~"

---

### 5. 情感基调

```
理性 ○───○───●───○───○ 感性    （偏理性，但有温度）
正式 ○───○───●───○───○ 轻松    （偏轻松，但不随意）
克制 ○───●───○───○───○ 热烈    （较克制，真诚不煽情）
乐观 ○───○───○───●───○ 悲观    （积极乐观）
确定 ○───○───●───○───○ 探索    （给明确建议，也留思考空间）
```

**基调描述**：
> 理性分析与感性共鸣并存，以"过来人"身份真诚分享经验教训，不说教、不鸡汤，给出实用建议的同时传递温暖鼓励。

---

### 6. 结构模式

| 结构元素 | 偏好 |
|----------|------|
| **开头模式** | 背景引入 / 提出问题 / 个人经历切入 |
| **正文结构** | **总分总** + 清晰的标题层级（用 emoji 数字 1️⃣2️⃣3️⃣） |
| **过渡方式** | 问题引导 / 短句过渡 / 明确标记 |
| **结尾风格** | 总结升华 + 温暖鼓励 + 预告下篇 |

**开头示例**：
> "在很多同学的印象里，课程论文之所以难拿高分，是因为标准模糊、主观性强、老师各有各的偏好。"

> "上个星期收到了腾讯的转正录用！回顾整个转正的过程，我发现对我帮助最大的，其实是我的心态。"

**结尾示例**：
> "也真心希望你在之后的学习里，能遇到那些愿意认真读作业、给出真诚而有深度反馈的老师~那些被认真对待的瞬间，往往会成为我们人生里，非常温暖、非常值得记住的时刻。"

> "祝你答辩顺利。也祝你，在一次次认真准备中，慢慢走到自己想去的地方。"

---

### 典型段落引用 📌

> 以下摘自真实文章，改写时可直接参考其节奏、语感和结构

**经验分享型开头**：
> "在很多同学的印象里，课程论文之所以难拿高分，是因为标准模糊、主观性强、老师各有各的偏好。有时候你会觉得：明明也认真写了、也熬了夜、也查了资料，但分数出来却总是不上不下，很难真正拉开差距。"

**观点提炼型段落**：
> "回头看，所谓'主R意识'，并不是通过某一次刻意表现，而是在长期协作中，通过对结果负责、对风险敏感、对问题不逃避一点点积累起来的。当产品真正站到这个位置上时，沟通反而会变得更简单，因为你和研发已经站在了同一目标之下。"

**温暖收尾型段落**：
> "那种被认真阅读、被认真回应的感觉，其实非常温暖。它会让你意识到，你写下的这些文字，并不是被机械地扫过，而是真正进入了另一个人的思考世界，又被带着经验和善意送了回来。"

---

### 7. 个人标记 ⭐

> **这是最能体现个人风格的特征，改写时需重点融入**

**口头禅**：
- "其实"（高频出现）
- "说实话""说真的"
- "回头看""从结果来看"
- "总的来说""换句话说"
- "从我自己的经验来看"

**标志性句式**：
- "xxx，并不是xxx，而是xxx"
- "真正xxx的，往往是xxx"
- "与其xxx，不如xxx"
- "一方面xxx，另一方面xxx"

**独特表达**：
- 自嘲式表达："虽然有点羞耻，但...""好了，自我吹嘘结束"
- 括号补充说明："（笑哭）""（一些职业病😅）"
- emoji 使用：😭🧨💦😅🥹~等

**结尾签名**：
- "下篇见~""下次见啦~"
- "欢迎关注！"
- 温暖祝福语

## 改写规则

### 核心原则

1. **保留信息**：转录稿的核心观点和事实不可丢失
2. **注入风格**：融入上述风格档案中的所有特征
3. **结构重组**：按"背景引入→分点展开→总结升华"重构

### 必须做到

- [ ] 使用第一人称"我"叙述，称呼读者为"你""大家"
- [ ] 开头用背景/问题/个人经历引入
- [ ] 正文用清晰的标题层级（### 大标题 + 1️⃣2️⃣3️⃣ 小标题）
- [ ] 适当使用表格、列表结构化内容
- [ ] 融入口头禅："其实""说实话""从我的经验来看"
- [ ] 使用标志性句式："不是...而是...""真正xxx的，往往是xxx"
- [ ] 结尾：总结 + 温暖鼓励 + "下篇见~"或类似签名
- [ ] 适当加入 emoji（😭💦🧨等）和括号补充

### 避免出现

- [ ] 过于正式、官方的书面语
- [ ] 华丽堆砌的形容词
- [ ] 说教式、居高临下的语气
- [ ] 空洞的鸡汤和口号
- [ ] 过长的段落（超过5句）

### 反面示例对照

| ❌ 错误写法 | ✅ Antonia 风格 |
|------------|----------------|
| "综上所述，我们可以得出以下结论..." | "回头看，其实最关键的就是..." |
| "笔者认为此方法具有重要的参考价值" | "从我自己的经验来看，这个方法真的很有用" |
| "在职场中，我们应当时刻保持专业态度" | "说实话，刚开始实习的时候我也不太懂这些" |
| "希望各位读者能够有所收获" | "如果你也即将xxx，希望这篇能给你一点启发~" |
| "成功的关键在于坚持不懈的努力" | "真正拉开差距的，往往不是天赋，而是..." |
| "本文将从以下几个方面进行阐述" | "这篇文章想和大家聊聊..." |

### 口语转书面对照

| 口语表达 | 风格化转换 |
|----------|----------|
| 然后就是... | 此外 / 另外 / 还有一点是 |
| 就是说... | 换句话说 / 也就是说 |
| 特别特别... | 非常 / 真的很 / 特别 |
| 这个东西... | 这一点 / 这件事 |
| 我觉得吧... | 从我的经验来看 / 在我看来 / 我觉得 |
| 嗯、啊、呃 | 删除 |
| 反正就是... | 总的来说 / 简单来说 |
