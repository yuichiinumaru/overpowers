# 睿智幽默风格 (Witty Humor Style)

## 概述

这是一种让人看完会心一笑，然后陷入沉思的内容风格。不说教，不鸡汤，不装逼。

## 核心特点

**一针见血 + 冷幽默 + 反常识**

### 语言特征

1. **🎯 直击要害**：三句话说清楚别人三千字说不明白的事
2. **😏 冷幽默**：用调侃的语气说出残酷的真相
3. **💡 反常识**：挑战大众认知，但逻辑自洽
4. **🔪 一刀见血**：不绕弯子，直接说结论

### 写作原则

- 像朋友聊天，不像老师上课
- 用数据和案例说话，不用形容词堆砌
- 敢于说出"政治不正确"的真话
- 结尾留白，让读者自己思考
- 多用短句，制造节奏感
- 适当使用反问、自问自答

请按照"睿智幽默风格"创作一篇内容。

### 方法 2：使用自动化脚本

本项目提供了自动化脚本，可以批量生成睿智幽默风格的内容。

#### 基于热点生成

```bash
npm run skill:witty
```

系统会自动：
1. 抓取 TopHub 热榜
2. 筛选适合睿智幽默风格的话题
3. 批量生成内容
4. 保存到 `outputs/witty-humor-posts/`

## 示例对比

### 示例 1：关于努力

❌ **说教版**：
> "在这个快速变化的时代，我们需要不断学习，提升自己的认知水平，才能在竞争中脱颖而出。"

✅ **睿智幽默版**：
> "大部分人的努力，都是在用战术上的勤奋，掩盖战略上的懒惰。
>
> 说白了，就是不愿意思考。
>
> 思考太累了。"

### 示例 2：关于成功

❌ **说教版**：
> "成功需要坚持不懈的努力和正确的方向。"

✅ **睿智幽默版**：
> "努力很重要。
>
> 但选择比努力重要 100 倍。
>
> 在错误的赛道上狂奔，只会让你更快地到达错误的终点。"

### 示例 3：关于学习

❌ **说教版**：
> "学习是一个持续的过程，我们应该保持好奇心，不断探索新知识。"

✅ **睿智幽默版**：
> "你以为学习是为了获取知识。
>
> 其实学习是为了筛选信息。
>
> 这个时代，知道什么不重要，知道什么是垃圾更重要。"

## 适用场景

- 社会热点评论
- 职场观察
- 人生感悟
- 商业洞察
- 认知升级类内容

## 注意事项

1. **尺度把握**：冷幽默不等于冷嘲热讽，要有温度
2. **逻辑自洽**：反常识的观点必须有扎实的逻辑支撑
3. **避免说教**：永远像朋友聊天，而不是老师上课
4. **结尾留白**：不要把话说满，让读者自己思考
