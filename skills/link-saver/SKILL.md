# link-saver - 链接收藏夹

保存和整理有用的网页链接。

## 功能
1. **保存链接** - 收藏网页链接并添加描述
2. **分类管理** - 按类别整理链接
3. **快速搜索** - 搜索已保存的链接

## 触发关键词
- 保存链接
- 收藏链接
- 添加书签
- 查看链接
- 搜索链接

## 数据存储
- links.json - 链接数据

## 使用示例
- "保存 https://example.com 我的收藏"
- "查看所有链接"
- "搜索 Python 教程"
