# Response Format

## 目录

- 对比表格
- 一致意见分析
- 参考来源
- 进度反馈

## 对比表格

```text
📊 多平台意见对比：

| 来源 | 可能方向 | 建议科室 | 紧急程度 |
|------|---------|---------|---------|
| MediWise 分析 | BPPV [1] | 耳鼻喉科 | 非紧急 |
| MSD 诊疗手册 | 良性阵发性位置性眩晕 [2] | 耳鼻喉科 | 非紧急 |
| 丁香医生 | 耳石症 / 前庭神经炎 [3] | 眩晕门诊 | 非紧急 |
| Mayo Clinic | BPPV [4] | ENT / Neurology | Non-urgent |
```

## 一致意见分析

```text
✅ 一致意见：
- 多个来源都认为最可能方向是 XX [1][2][3]
- 建议科室大体一致为 XX

⚠️ 差异点：
- 某些来源还提到 YY [4]
- 这通常意味着还需要更多信息或进一步检查
```

## 参考来源

```text
参考来源：
[1] MediWise 分析 - 基于问诊信息的综合评估
[2] 来源名 - 标题 URL
[3] 来源名 - 标题 URL

⚠️ 以上为多平台信息综合对比，仅供参考，不构成诊断。请及时就医，由医生进行专业检查和诊断。
```

## 进度反馈

搜索较长时输出简短进度：

- `已完成初步问诊分析，正在搜索其他平台的意见...`
- `已搜索 2 个平台，继续补充更多来源...`
- `已收集多个平台的意见，正在整理对比分析...`

不要在进度文本里提前下结论。
