# 七步命名法详细框架

## 第一步：八字五行推演与五行补缺

1. 根据出生年月日时排出四柱八字（年柱·月柱·日柱·时柱）
2. 列出八字对应的天干地支
3. 统计五行分布：金、木、水、火、土各有几个
4. 判断五行强弱：
   - 哪些五行过旺（需泄或克）
   - 哪些五行缺失或偏弱（需补）
5. 确定喜用神与忌神
6. **输出：五行补缺方向**（如"缺水喜木，名字宜用水、木属性字"）

## 第二步：生肖分析与字根筛选

1. 确定生肖属相
2. 分析生肖的喜用字根与忌用字根：
   - **喜用字根**：与该生肖相合、相生的偏旁部首（如属虎喜"山""林""王"等）
   - **忌用字根**：与该生肖相冲、相害的偏旁部首（如属虎忌"门""小"等）
3. 分析生肖六合、三合关系，优先选用合局字根
4. 排查生肖六冲、六害，严格规避冲害字根
5. **输出：推荐字根列表 + 禁用字根列表**

## 第三步：姓名学格局校验与大凶规避

1. 计算姓氏的天格
2. 根据五行补缺方向，筛选人格、地格、外格、总格的笔画组合
3. **严格规避以下大凶组合**：
   - 三才配置中含"凶"或"大凶"的组合
   - 人格或地格落入 4、9、10、14、19、20、22、26、28、34、36、44、46、54 等凶数
   - 天格与人格相克、人格与地格相克的组合
4. 优先选择三才配置全吉或半吉以上的笔画方案
5. **输出：推荐的名字笔画结构（如"姓X画 + 名第一字X画 + 名第二字X画"）**

## 第四步：经典取意与艺术创造

从以下典籍和学科中寻找契合的意象与用字，追求独特性和艺术感：

- **道德经**：如"上善若水"→ 善/泽；"致虚极，守静笃"→ 致远/守真/虚怀
- **周易**：如"天行健，君子以自强不息"→ 行健/乾行；"地势坤，君子以厚德载物"→ 厚德/坤载
- **四书五经**：如"博学之，审问之，慎思之，明辨之，笃行之"→ 博文/慎思/笃行
- **唐诗**：如"明月松间照，清泉石上流"→ 松照/泉清/石流
- **宋词**：如"一蓑烟雨任平生"→ 任平/烟雨
- **元曲**：如"枯藤老树昏鸦，小桥流水人家"→ 桥溪/流安
- **语义学创造**：通过语义场分析，将两个看似无关但语义互补的字组合，创造出全新的意境
- **心理学暗示**：选择能激发积极自我认同的字，利用"名字效应"塑造正向心理暗示

**要求**：
- 每个名字必须标注典故出处原文
- 不编造不存在的典故
- 鼓励在经典基础上进行创造性组合，而非简单摘字

## 第五步：音韵学精调

1. **声调搭配**：
   - 标注每个字的声调（阴平、阳平、上声、去声）
   - 严禁三连同调（如三个都是阴平）
   - 追求"平仄交替"或"仄平仄"等有起伏感的组合
2. **声母韵母**：
   - 避免声母相同导致拗口（如"赵志卓"zh连续）
   - 避免韵母相近导致含混（如"王芳光"ang连续）
3. **韵律节奏**：
   - 两字名追求"短-长"或"长-短"的节奏感
   - 三字全名（含姓）读起来应有自然的语流
4. **输出：音韵评级（优/良/中）+ 具体点评**

## 第六步：谐音与传播力校验

1. **谐音排查**（最高优先级）：
   - 普通话谐音排查
   - 常见方言谐音排查（至少覆盖粤语、吴语、闽南语的明显不雅谐音）
   - 排除所有可能被曲解、取绰号的读音组合
2. **传播力评估**（营销学视角）：
   - 一听就能写出来吗？（避免同音多字导致的沟通成本）
   - 一看就能读对吗？（避免多音字歧义）
   - 容易记住吗？（记忆锚点原则：有画面感、有故事感的名字更易记）
   - 自我介绍时是否好解释？（"哪个X？""XX的X"能否一句话说清）
3. **输出：谐音风险（无/低/高）+ 传播力评级（优/良/中）**

## 第七步：综合评分与排序

对每个候选名从六个维度打分（满分10分）：

| 维度 | 评分标准 |
|------|---------|
| 五行契合度 | 与八字喜用神的匹配程度、五行补缺效果 |
| 生肖适配度 | 字根是否符合生肖喜忌、是否利用了合局 |
| 文化底蕴 | 典故的深度与贴切度、经典化用的巧妙程度 |
| 音韵美感 | 声调搭配、声韵协调、整体韵律感 |
| 传播记忆力 | 好记、好写、好解释、无谐音风险 |
| 书写美观度 | 字形结构均衡、繁简适中、视觉和谐 |

按总分（满分60分）从高到低排列候选名。
