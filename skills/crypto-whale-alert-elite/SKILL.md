# crypto-whale-alert-elite - 巨鲸警报精英版

## 技能描述
精英版加密货币巨鲸追踪系统，实时监控大额转账和交易所流动，提供市场趋势预警。支持多链监控、智能分类、历史对比。

## 功能特性
- 🐋 实时大额转账监控（>$100K）
- 📊 交易所流入/流出分析
- 🏦 机构钱包地址标记
- 📈 累积/派发模式识别
- 🔔 自定义金额阈值警报
- 🌐 多链支持（ETH/BTC/SOL 等）

## 使用方式
```bash
# 监控大额转账
/crypto-whale-alert-elite --watch --min-amount 1000000 --chain ethereum

# 追踪特定地址
/crypto-whale-alert-elite --track <钱包地址> --alias <名称>

# 交易所流动分析
/crypto-whale-alert-elite --exchange-flow --exchange binance

# 历史对比
/crypto-whale-alert-elite --compare --token BTC --days 30
```
