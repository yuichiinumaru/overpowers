# Medium Writer

Medium Partner Program向け記事執筆・公開。収益化可能な記事作成。

## 収益ポテンシャル

- **Partner Program**: $200-2,000/月
- **フォロワー1万人時**: $500-3,000/月
- **バイラル記事**: $1,000-10,000/記事

## Medium Partner Program

### 収益モデル
```
読者がMemberの場合:
├── 読了時間に応じて報酬
├── クラップ数は影響小
└── 初回読者ボーナスあり
```

### 参加条件
```
- 100人以上のフォロワー
- 直近1年に1記事以上公開
- Stripeアカウント連携
```
