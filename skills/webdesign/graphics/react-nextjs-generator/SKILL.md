# React Next.js Project Generator

## Description
Generates a complete React + Next.js project based on requirement documents and UI design mockups, utilizing Ant Design, Tailwind CSS, and Zustand as the technology stack.

## Usage
When the user uploads requirement documents and UI design mockups, execute the following steps:

1. Analyze requirement document content
2. Parse UI design mockup elements
3. Generate project structure
4. Create page components
5. Configure state management
6. Set up styling system

## Tech Stack
- React
- Next.js
- Ant Design (antd)
- Tailwind CSS
- Zustand (State Management)
