# Baidu Maps jsapi-ui-kit

A lightweight Baidu Maps UI component library, providing capabilities for place autocomplete, place search, and detail display.

## When to Use

- You want to quickly integrate standardized map UI components without developing from scratch.
- You need to add an autocomplete function to a search input box in your map application.
- You need to add a place search function to your map application.
- You need to display POI detail information (name, address, phone, etc.).
