# WeChat Comprehensive Introduction

This skill helps you gain a comprehensive understanding of WeChat.

## WeChat Overview

### Basic Information
- **Launch Date**: January 21, 2011
- **Developer**: Tencent
- **Founder**: Allen Zhang
- **Headquarters**: Shenzhen, China
- **User Scale**: Over 1.3 billion monthly active users

### Development History

| Year | Milestone |

| 2014 | WeChat Official Accounts Platform launched |
| 2015 | WeChat City Services launched |
| 2017 | Mini Programs released |
| 2020 | Channels launched |
| 2023 | WeChat HarmonyOS version released |
