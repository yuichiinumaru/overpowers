# Content Structure Designer - 内容结构设计器

## 触发条件

当用户提到以下关键词时触发此技能：
- "不知道怎么写"、"如何组织内容"
- "内容结构"、"文章大纲"、"内容框架"
- "怎么开头"、"怎么结尾"
- "内容逻辑"、"内容组织"
- "写作模板"、"内容模板"

## 核心能力

帮助内容创作者：
1. 快速搭建内容框架
2. 设计清晰的内容逻辑
3. 选择合适的内容类型
4. 优化内容结构
5. 提高内容完成度

## 工作流程

### 第 1 步：确定内容类型

根据选题和目标，选择合适的内容类型：

#### 1. 教程类（How-to）
**适用场景：** 教别人如何做某事
**核心目标：** 让读者学会并能实操
**关键要素：** 步骤清晰、可执行、有案例

#### 2. 评测类（Review）
**适用场景：** 评价产品、工具、方法
**核心目标：** 帮助读者做决策
**关键要素：** 客观、全面、有对比

#### 3. 故事类（Story）
**适用场景：** 分享经历和经验
**核心目标：** 引发共鸣和启发
**关键要素：** 真实、有细节、有转折

#### 4. 观点类（Opinion）
**适用场景：** 表达看法和见解
**核心目标：** 改变认知或引发思考
**关键要素：** 有论据、有逻辑、有深度

#### 5. 盘点类（List）
**适用场景：** 整理资源、方法、工具
**核心目标：** 提供价值和便利
**关键要素：** 全面、实用、有筛选

#### 6. 问答类（Q&A）
**适用场景：** 解答常见问题
**核心目标：** 解决具体疑惑
**关键要素：** 针对性强、解释清楚

### 第 3 步：优化内容结构

对选定的结构进行优化：

#### 1. 开头优化
- **前 3 秒抓住注意力**
  - 用数据、故事、问题开头
  - 避免"大家好"、"今天分享"
  - 直接切入核心价值

- **明确价值承诺**
  - 看完能得到什么？
  - 解决什么问题？
  - 为什么值得看？

#### 2. 正文优化
- **逻辑清晰**
  - 使用小标题分段
  - 每段一个核心观点
 关系

- **内容充实**
  - 有具体案例
  - 有数据支持
  - 有实操步骤

- **易读性强**
  - 短段落（2-4 句）
  - 使用列表
  - 加入图片/表格

#### 3. 结尾优化
- **总结要点**
  - 3-5 条核心要点
  - 简洁明了
  - 可复制保存

- **行动建议**
  - 下一步做什么？
  - 如何开始？
  - 避免什么？

- **互动引导**
  - 提问引发讨论
  - 引导点赞收藏
  - 鼓励分享经验

## 实战案例

### 案例：职场效率教程

**选题：** "如何提高工作效率？10 个实用方法 + 真实案例"

**内容类型：** 教程类

**选择模板：** 标准教程结构

**结构设计：**

```
【开头】（10%）
- 痛点：你是不是经常加班到深夜，但工作还是做不完？
- 价值：这篇文章教你 10 个方法，让你效率提高 50%，准时下班
- 适用：适合所有职场人，尤其是经常加班的人
- 预告：会讲方法、案例、常见错误

【正文】（70%）
- 准备：需要什么工具？（番茄钟、任务清单）
- 方法 1：时间块管理
  - 说明：把一天分成时间块，每块专注一件事
  - 步骤：如何划分时间块？
  - 案例：我如何用时间块提高效率
  - 错误：不要把时间块划分得太细
- 方法 2-10：[重复结构]
- 实战案例：我的一天时间安排
- 进阶技巧：如何根据精力分配任务

【结尾】（20%）
- 总结：10 个方法的核心要点
- FAQ：
  - Q：如何坚持？
  - Q：如何选择适合自己的方法？
- 行动：从最简单的一个方法开始
- 互动：你用过哪些方法？效果如何？
```

**内容填充：**
- 案例：我从每天加班 3 小时到准时下班的经历
- 数据：效率提高 50%，每天节省 2 小时
- 图片：时间块示例图、任务清单截图

**检查结果：**
- ✅ 开头抓住注意力（痛点 + 价值）
- ✅ 逻辑清晰（10 个方法 + 案例 + FAQ）
- ✅ 有具体案例（个人经历）
- ✅ 有数据支持（效率提高 50%）
- ✅ 有行动建议（从一个方法开始）
- ✅ 有互动引导（评论区讨论）

### Q1: 如何选择合适的结构模板？
A: 根据内容类型和目标选择：
- 教别人做事 → 教程类
- 评价产品 → 评测类
- 分享经历 → 故事类
- 表达观点 → 观点类
- 整理资源 → 盘点类
- 解答问题 → 问答类

### Q2: 可以混合使用多个模板吗？
A: 可以，但要保持主线清晰。例如：
- 教程类 + 故事类（用故事引入，然后讲方法）
- 观点类 + 案例类（用案例支持观点）
- 盘点类 + 评测类（盘点后深度评测）

### Q3: 如何让结构更有吸引力？
A:
- 开头用故事、数据、问题抓住注意力
- 正文用小标题、列表、图片增加可读性
- 结尾用总结、行动建议、互动引导

### Q4: 结构设计好了，但写不出来怎么办？
A:
- 先填充最容易的部分（案例、数据）
- 使用"先写后改"策略，不要追求完美
- 参考同类型的优秀内容
- 使用 AI 工具辅助（但要保持个人风格）

## 总结

好的结构是内容的骨架。使用这个技能，你可以：
- 快速搭建内容框架
- 避免写作时的迷茫
- 提高内容完成度
- 让内容更有逻辑性

记住：**结构不是限制，而是帮助你更好地表达。熟练后可以灵活调整。**
