# 动漫每周更新日历

## 数据来源
使用 `extract_content_from_websites` 或 `batch_web_search` 工具从以下网站获取动漫更新信息：
- **哔哩哔哩** (bilibili.com)
- **Bangumi** (bgm.tv)
- **动漫之家** (dmzj.com)

## 功能范围
1. **本周更新列表**：获取本周更新的动漫
2. **每日更新**：按日期查询当天更新的动漫
3. **追番提醒**：特定动漫的更新时间
4. **新番资讯**：最新动漫资讯和播出时间

## 动漫更新规律（参考）
日本动漫通常按星期更新，周六和周日更新最为集中。

## 四季新番时间
- **冬季番**: 1月-3月
- **春季番**: 4月-6月
- **夏季番**: 7月-9月
- **秋季番**: 10月-12月
