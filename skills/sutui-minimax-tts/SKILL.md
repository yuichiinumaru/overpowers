# MiniMax 多人对话语音合成

根据用户需求生成多人对话剧本，智能匹配角色音色，合成完整音频和分段音频，最终生成精美的 HTML 展示页面。

## 工作流

1. **分析用户需求** - 获取对话主题、角色数量、对话风格
2. **生成对话剧本** - 创作自然流畅的多人对话
3. **智能匹配音色** - 调用 MCP 工具获取可用音色并匹配
4. **文本预处理** - 添加语气词增强情感表达
5. **分段语音合成** - 为每段对话调用语音合成
6. **生成完整长音频** - 合并所有分段音频
7. **生成 HTML 展示页面** - 使用模板生成展示页面

## 输出文件

```
dialogue_output/
├── dialogue.html          # 完整展示页面
├── dialogue.json          # 对话数据（含音频 URL）
├── merged.mp3             # 合并后的完整长音频
└── segments/              # 分段音频（可选）
```

## 注意事项

- 音色一致性：同一角色在整个对话中使用相同音色
- 情绪连贯性：相邻对话的情绪过渡要自然
- 文本长度：单段对话建议 20-100 字
- 语气词适度：每段对话 1-2 个语气词为宜
