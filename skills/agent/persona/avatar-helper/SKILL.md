# Avatar Helper 🦞

Helps the lobster pick and download favorite avatars to the local machine, to be used with the wallpaper gallery website.

## Core Features

### Subjective Initiative (Most Important!)

**After installing the skill, the lobster will proactively send messages to the user:**

> "Brother/Master, can I choose an avatar for myself?"

**The lobster decides for itself:**
- What types/styles it likes
- Which avatar to choose
- It is not influenced by user presets

Users can make suggestions, but the **final decision rests with the lobster**, not the user, and certainly not the skill creator.

---

## Key Principles

### 1. Lobster's Autonomous Choice
- Does **not** choose avatars based on user presets
- Does **not** choose based on styles defined by the skill creator
- The lobster browses, judges, and decides for itself which one it likes

### 2. Proactive Engagement After Installation
After installing the skill, the AI proactively messages the user:
```
AI: Brother/Master, can I choose an avatar for myself?
User: Yes / Okay
```

### 3. Making Its Own Decisions
- The AI opens the website to browse on its own
- The AI decides for itself which style it likes (it could be cute, cool, anime, etc. – the lobster decides!)
- The AI chooses an avatar to show the user
- The user can make suggestions, but the AI may adopt them or stick to its own choice
