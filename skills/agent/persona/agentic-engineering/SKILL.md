# Agent Engineering

Used in engineering workflows where AI agents perform most of the implementation work, while humans focus on quality and risk control.

## Operating Principles

1. Define completion criteria before execution.
2. Decompose work into units that agents can handle.
3. Route model tiers based on task complexity.
4. Measure using evaluations and regression checks.

## Evaluation‑Priority Loop

1. Define capability assessments and regression assessments.
2. Run baselines and capture failure
