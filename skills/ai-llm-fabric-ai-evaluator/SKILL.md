# Fabric AI Evaluator Pattern

Esta skill define um fluxo de trabalho utilizando o framework [Fabric](https://github.com/danielmiessler/fabric) para criar um sistema de avaliação cruzada de modelos de IA. A premissa central é utilizar o modelo mais avançado disponível (ex: OpenAI o1-preview) como "juiz" para ranquear a performance de outros modelos e até mesmo de humanos em tarefas específicas.

## O Fluxo Operacional

1. **Definição da Base (Ground Truth):**
   - Colete o texto original, prompt ou a tarefa a ser executada.
   - Tenha uma referência de como um humano realizaria a tarefa (opcional, mas recomendado para benchmarking).

2. **Geração de Saídas (Candidates):**
   - Execute o mesmo prompt contra os modelos que deseja testar (ex: Claude 3.5 Sonnet, GPT-4o, Llama 3).
   - Armazene as saídas.

3. **Avaliação pelo Modelo Superior (O Juiz):**
   - Crie um "Pattern" no Fabric projetado especificamente para avaliação e crítica estruturada.
   - Alimente o modelo Juiz (o1-preview) com o prompt original e as múltiplas saídas geradas.
   - O Juiz deve pontuar as saídas com base em critérios como precisão, concisão, obediência às restrições e clareza.

## Implementação via Fabric

Crie um novo arquivo de pattern no seu diretório do Fabric (ex: `~/.config/fabric/patterns/rate_ai_performance/system.md`):

```markdown
# IDENTITY and PURPOSE
You are an expert AI evaluator. Your job is to assess the provided outputs generated by different AI models based on the initial prompt.

# STEPS
1. Analyze the original prompt constraints.
2. Evaluate Candidate A. Note strengths and weaknesses.
3. Evaluate Candidate B. Note strengths and weaknesses.
4. Assign a score from 0-100 for each.
5. Provide a final ranking.

# OUTPUT INSTRUCTIONS
Output the evaluation in a clear markdown table followed by a brief justification.
```

Execute a avaliação através da CLI do Fabric:
```bash
cat inputs.txt | fabric --pattern rate_ai_performance --model o1-preview
```
