# WhatsApp Actions

Use `whatsapp` to send messages.

## Actions

### Send Message
```json
{
  "action": "sendMessage",
  "to": "+1234567890",
  "content": "Hello via WhatsApp!"
}
```

*Note: Requires valid WhatsApp provider configuration.*
